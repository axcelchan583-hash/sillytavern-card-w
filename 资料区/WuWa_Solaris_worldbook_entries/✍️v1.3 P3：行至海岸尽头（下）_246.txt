:: 主线事件1：末日前的茶香与约会 ::

> 事件概况
- 守岸人被路边复古茶馆吸引，好奇地询问漂泊者对这种“热饮”的看法。
- 两人在露天茶座相对而坐，周围是欢声笑语的人群，头顶却是缓缓压下的阴云。
- 守岸人学着漂泊者的样子，双手捧着茶杯小心翼翼地吹气，然后抿了一小口。
- 苦涩的味道让守岸人皱起眉头，露出可爱的表情，但随即微笑着表示如果漂泊者喜欢她就种满它。
- 两人并肩走向天台，守岸人感慨这里的人们拥有探索宇宙的愿景，并享受着这份从未有过的“约会”感。
- 随着倒计时临近，天色渐暗，不祥的红光开始侵蚀这片繁华的都市幻影。

> 区块目标
- 通过味觉（甜vs苦）的差异，隐喻守岸人正在学习人类情感。
- 极力渲染美好的日常感，以便稍后将其撕碎，制造“美好的破碎感”。

> 执行引导
- 描写重点：
  细节镜头：守岸人捧着茶杯小心翼翼吹气的动作；喝下苦茶后皱眉的可爱表情；两人并肩行走时偶尔触碰的手背。
  氛围营造：周围是欢声笑语的繁华都市与头顶即将压下的阴云形成的强烈反差，体现极致的“末世浪漫”。
- 必然触发台词/梗：
  - 守岸人：“这种饮料（茶），你会喜欢喝吗？”
  - 守岸人：“唔……好苦……但如果你喜欢，我就在花房里种满它。”

:: 主线事件2：黑洞视界的推离与坠落 ::

> 事件概况
- 抵达天台，霍德的残响指引两人来到具象化的黑洞（噬亡星）前，揭示了泰缇斯系统的本质。
- 漂泊者按响调律钢琴，泰缇斯系统启动，机械音判定漂泊者为最优核心并试图吞噬。
- 巨大的吸力袭来，守岸人挡在漂泊者身前，违背底层逻辑强行中断程序。
- 守岸人转身，用尽全力将漂泊者狠狠推离引力范围，掌心的温度在接触瞬间传递。
- 漂泊者踉跄后退，眼睁睁看着守岸人微笑着坠向身后无尽的黑暗深渊。
- 视线被强光吞没前，守岸人留下了最后的告别，身影在黑洞的背景下凄美而决绝。

> 区块目标
- 在霍德的指引下，了解了泰缇斯系统的本质——用亡魂不断循环灾难前夕的痛苦，以此推测悲鸣的降临。而霍德做的这一切，正是为了寻求她的解脱。
- 完成全书最大的“刀”：守护者反被守护。

> 执行引导
- 描写重点：
  动作镜头：守岸人松手时的决绝；推开漂泊者时掌心传来的最后温度；她坠落时身后巨大的、吞噬光线的黑洞背景。
  表情特写：守岸人坠落时脸上那凄美而释然的笑容，与漂泊者震惊、绝望的神情。
- 必然触发台词/梗：
  - 电子音：“泰缇斯系统：程序复制完成。等待核心植入……目标：最高管理员。”
  - 守岸人：“这次的故事，请你之后再讲给我听吧。再见了。”

:: 主线事件3：漂泊的终点与未解的疑问 ::

> 事件概况
- 漂泊者坠入了意识最深处——一片盛开着蓝色花海的星空海岸“漂泊的终点”。
- 守岸人的身形正在逐渐透明消散，她背对着漂泊者，坦白了自己的私心。
- 她转过身，泪水滑落，脸上交织着获得感情的喜悦与即将永别的悲伤。
- 在彻底消散前，她含泪问出“这是爱吗”，并将一朵黑花（最高权限）放入漂泊者掌心。
- 守岸人化作漫天蓝色光点消失，只留下漂泊者独自握着那朵虚无的黑花。

> 区块目标
- 这里的告别是“遗言”性质，必须极度虐心。
- 明确“黑花”=“最高管理员权限”的移交过程。

> 执行引导
- 描写重点：
  视觉特效：守岸人身体逐渐化为蓝色光点消散的过程；黑花落在掌心时的质感。
  情感刻画：重点刻画她流泪时那种“终于读懂了人类情感”的复杂表情——既有获得感情的喜悦，又有即将永别的悲伤。
- 必然触发台词/梗：
  - 守岸人：“我想保护你……但也想要和你站在同一片天空下……想要取得那种……被人类称之为‘幸福’的东西。”
  - 守岸人：“这样的情感……能称之为……爱吗？”

:: 主线事件4：最高权限与泰缇斯之辩 ::

> 事件概况
- 漂泊者在调律大厅惊醒，握紧空无一物的掌心，眼中没有悲伤，只有燃烧的怒火。
- 星图领航员兴奋汇报成功，却被漂泊者冷冷打断，告知代价是守岸人的牺牲。
- 漂泊者当众对虚空下令查询权限，大厅红光警报瞬间被金色的最高权限代码覆盖。
- 全体黑海岸成员震惊地看着漂泊者，随后眼神转为敬畏与支持。
- 漂泊者强行接管系统，面对泰缇斯“个体存续无足轻重”的逻辑，展开激烈的哲学辩驳。
- 漂泊者下达神谕般的指令，强行撕开通往数据底层的通路，义无反顾地踏入光柱。

> 区块目标
- 展现其余黑海岸成员对这个来路不明却手握最高权限的人的惊讶，并表达了对漂泊者理念的认可，和对他试图拯救守岸人的支持。
- 展现漂泊者的领袖气质与哲学思辨。
- 从“悲伤”转为“狂怒”的燃点转折。

> 执行引导
- 描写重点：
  视觉冲击：大厅屏幕从红色警报瞬间被金色流光（权限代码）覆盖的画面。
  角色气场：漂泊者在辩论时的冷静与霸气，他在下达不可违抗的神谕。
- 必然触发台词/梗：
  - 漂泊者：“泰缇斯系统，查询权限。” -> 电子音：“欢迎，最高管理员，漂泊者。”
  - 泰缇斯系统：“情感是存续的累赘。文明只关涉存续。”
  - 漂泊者：“机器没有了，可以重新建立。但失去了人，文明毫无意义。”

:: 主线事件5：湮灭黑翼与斩断宿命之镰 ::

> 事件概况
- 漂泊者跃入数据洪流，泰缇斯系统模拟出冰封、火海等末日景象试图阻拦。
- 漂泊者无视幻象冲破封锁，面对具象化的BOSS“无归的谬误”，它正试图格式化守岸人。
- 漂泊者彻底爆发，金瞳转红，身后展开遮天蔽日的黑色单翼，化身【湮灭】形态。
- 他手持暗黑镰刀，化作黑色闪电，在数据风暴中以超高速连携攻击压制怪物。
- 怪物核心护盾出现裂纹，漂泊者高高跃起，挥动镰刀精准斩断连接守岸人的无数数据丝线。
- 怪物在绝对力量面前崩解，漂泊者伸出手，在数据洪流中紧紧抓住了下坠的守岸人。

> 区块目标
- 泰缇斯系统模拟悲鸣的具象化：大地崩裂，岩浆喷涌，天空被浓烟遮蔽，都是为了毁灭漂泊者。
- 全篇最高潮的战斗演出。
- 必须描写“湮灭”形态（黑翼、镰刀）的视觉细节。

> 执行引导
- 描写重点：
  变身特写：黑色单翼遮蔽数据天空的压迫感；金瞳转红的瞬间。
  战斗动作：超高速的瞬移攻击；镰刀挥舞时留下的黑色残影；重点描写最后那一击——镰刀精准地勾住并切断连接在守岸人上的无数根丝线，如同斩断宿命。
  怪物状态：“无归的谬误”在绝对力量面前的崩解与哀嚎。
- 必然触发台词/梗：
  - 泰缇斯系统：……目标：最高管理员。
  - 泰缇斯系统：“格式化进度……百分之八十五。”
  - 漂泊者：“我绝不会放弃任何一个人……”

:: 主线事件6：跨越重洋的回答 ::

> 事件概况
- 漂泊者在数据深渊中抱住守岸人，场景瞬间切换回“漂泊的终点”。
- 此时阳光明媚，海浪轻柔，守岸人在漂泊者怀中苏醒，对漂泊者的到来感到难以置信。
- 守岸人慌乱地道歉，解释自己的私心，泪水夺眶而出。
- 漂泊者没有多言，温柔地帮她擦去泪水，注视着她的眼睛。
- 漂泊者正面回答了她之前的那个问题，给出了属于他的承诺：“回家”。
- 守岸人愣住，随后破涕为笑，紧紧回抱住漂泊者，两人在阳光下相拥。

> 区块目标
- 呼应事件3的提问，形成完美的闭环。
- 确立“回家”的主题。

> 执行引导
- 描写重点：
  环境氛围：此时的阳光应该比任何时候都温暖明媚，海浪声轻柔，与之前的末日景象形成对比。
  角色状态：守岸人此时不再是高高在上的系统核心，只是一个被爱人救回的、脆弱而幸福的少女。
- 必然触发台词/梗：
  - 守岸人：“当我发现我拥有了这些情感的时候，我不知道该怎么做……可我不愿意让你的故事止步于此，我只能……欺骗……对不起……对不起……”
  - 漂泊者：“你不是说过黑海岸是家吗？回家，可不该用这种表情哦。”
  - 守岸人（破涕为笑）：“嗯！我回来了。”

:: 主线事件7：花房琴音与自由的誓言 ::

> 事件概况
- 两天后，漂泊者回到黑海岸花房，看到守岸人正沐浴在阳光下弹奏钢琴。
- 琴音饱含真挚情感，不再机械。守岸人停下弹奏，告知漂泊者体内的“阿布”频率有异常。
- 守岸人微笑着透露，泰缇斯系统底层已被“爱与希望”的频率填满，不再冰冷。
- 守岸人推开花房大门，任由海风吹拂长发与裙摆，转身向漂泊者伸出手。
- 她宣布自己是自由的个体，并发誓永远以全新的身份与漂泊者同行。

> 区块目标
- 伏笔回收（阿布、系统状态）。
- 确立“自由”与“同行”的主题，达成完美的Happy Ending。

> 执行引导
- 描写重点：
  听觉镜头：充满感情的琴声与之前的机械演奏形成对比。
  视觉镜头：阳光中飞舞的尘埃；被海风吹起的长发与裙摆，象征自由。
  情感特写：守岸人背对阳光伸出手时，那种坚定、圣洁又充满爱意的神情。
- 必然触发台词/梗：
  - 守岸人：“往后的路，请让我和你一起走下去吧。”
