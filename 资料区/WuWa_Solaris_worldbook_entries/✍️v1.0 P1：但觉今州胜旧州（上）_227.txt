- 目标1：完成从苏醒到初次战斗的引导，确立漂泊者“失忆”与“肉身吸纳声骸”的特殊体质。

- 主角状态：从昏迷中苏醒，身体有溺水后的沉重感，记忆全失，右手背上有金色的声痕印记。
- 伴随NPC：初期无，苏醒后秧秧（温柔冷静）、炽霞（元气活泼）在侧。
- 指引行为：神秘女神的推手将意识从虚空推向现实，耳畔传来模糊的呼唤声。

:: 主线事件1：群星垂眸与云陵苏醒 ::

> 事件概况
- 意识悬浮于浩瀚星海之中，一位身姿朦胧的神秘神明（女神）注视着漂泊者。
- 神明轻轻推动漂泊者，两人的手掌在虚空中短暂相触，随后漂泊者如流星般穿过“天空海”，坠入蔚蓝星球。
- 画面转至云陵谷，漂泊者在草地上猛然睁眼，发现自己正枕在少女（秧秧）柔软的大腿上。
- 炽霞正准备进行“初级巡尉必备急救”的心肺复苏，甚至已经摆好了按压姿势，因漂泊者的苏醒而急忙收手。
- 秧秧跪坐在漂泊者身侧，温柔地检查漂泊者的瞳孔与脉搏，确认体征平稳。

> 区块目标
- 目标1：通过神明的“垂眸”与“触碰”，暗示漂泊者非凡的来历。
- 目标2：通过“女神推手”与“少女膝枕”的对比，营造从神性到人性的落地感。
- 目标3：确立秧秧“温柔守护者”与炽霞“元气冤家”的初始形象。

> 执行引导
- 描写重点：
    视觉镜头：漂泊者睁眼，首先看到的是秧秧垂下的发丝和担忧的眼神（俯视视角），随后炽霞的大脸突然闯入视野，破坏了旖旎的氛围。
	互动细节：漂泊者起身时，秧秧伸手搀扶的动作，以及漂泊者离开膝枕后残留的触感。
- 必然触发台词/梗：
  - 神秘女子：“（无言的注视与推手动作）”
  - 秧秧：“呼吸……没有异常。心跳……应该很快……才对。”
  - 炽霞：“初级巡尉必备急救技能第一课——心肺复苏！但现实实践这可能还是第一次……”

:: 主线事件2：初战残象与岁主幻视 ::

> 事件概况
- 三人结伴同行，途中炽霞指出漂泊者拥有声痕与葫芦，必定是共鸣者。
- 遭遇巨浪级残象袭击，秧秧下意识地冲到漂泊者身前，张开手臂试图护住漂泊者。
- 漂泊者凭借身体本能拔刀战斗，动作利落地斩杀残象，令炽霞惊叹。
- 经过巨大的龙形雕像（岁主·角）时，漂泊者脑海中闪过一段第一人称记忆：曾与这尊巨龙面对面近距离对视。

> 区块目标
- 目标1：展现秧秧外柔内刚的性格，以及对陌生人（漂泊者）下意识的保护欲。
- 目标2：埋下“岁主旧识”的伏笔。

> 执行引导
- 描写重点：
  秧秧挡在身前的背影，以及发梢拂过漂泊者脸颊的微风；漂泊者注视雕像时脑中突兀闪回的威严龙瞳。
- 必然触发台词/梗：
  - 炽霞：“虽然漂泊者什么都想不起来，但有件事我可以十成确定！你是一位共鸣者。”
  - 秧秧：“与真实的岁主对面而立？按漂泊者描述这么近的距离，除非是岁主的共鸣者……”

:: 主线事件3：无冠者讨伐与肉身吞噬 ::

> 事件概况
- 秧秧与炽霞注意到天空海很活跃，可能快要产生新无音区了，建议加速前进。
- 在悬崖上，发现白芷在云陵谷悬崖下新生成的无音区中央发呆，三人开启滑翔伞翼，从高处跃下，落地瞬间白芷冷静汇报状况。
- 怒涛级残象“无冠者”凝聚成型，四人合力激战，漂泊者给予最后一击，留下可以吸收的残响。
- 伙伴教授漂泊者使用自己腰间的葫芦终端吸收声骸，但葫芦等级太低只能作罢。
- 漂泊者右手突然传来剧烈的饥饿感与牵引力，脑海中响起“饿……好饿……”的声音，这个声音只有漂泊者本人能听见，而且漂泊者肚子并不饿（为体内的“阿布”埋下伏笔）。
- 漂泊者正疑惑着声音的源头，随后在众目睽睽之下，没有使用终端，漂泊者直接用右手触碰并将无冠者的巨大残响强行吸入体内。
- 众人震惊之际，全域广播强制接入，今州令尹今汐发出邀请。

> 区块目标
- 目标1：极具视觉冲击力地展现“肉身吸纳声骸”的异能，营造“怪物般的强大”氛围。
- 目标2：通过令尹的广播，将行动目标引向今州城。

> 执行引导
- 描写重点：
  右手背上声痕的灼热感，能量被强行扯入身体时的冲击，以及炽霞目瞪口呆的表情。
- 必然触发台词/梗：
  - 秧秧：“古籍中有写：‘斯天人降世，乾坤指掌，纳骸于身……’”
  - 炽霞：“这……这是什么力量！你……难道一直在扮猪吃老虎！生吞残象？”
  - 今汐：“这位客人，在您苏醒之后，您一定感到十分迷茫……请您于今州城内边庭一见。”

:: 主线事件4：边庭会面与信物交付 ::

> 事件概况
- 经过长途跋涉到达今州边庭，秧秧负责办理手续，漂泊者独自进入会客室。
- 令尹近卫散华接待，她用异样的眼神紧盯着漂泊者，随后解释她眼中看到的只有“频率”。
- 散华表示漂泊者的频率与令尹一样纯净真实，暗示他的强大能力，随后转交四件信物：紫色树叶、过期糖丸、小型日晷、莽吉柿。
- 告知漂泊者有三日期限，可自行调查信物线索。

> 区块目标
- 目标1：通过散华特殊的视角（频率感知），侧面印证漂泊者的特殊性。
- 目标2：确立解谜任务（四件信物）。

> 执行引导
- 描写重点：
  散华那仿佛能穿透肉体的冰冷注视，以及她提到“频率”时语气中的狂热与敬意。
- 必然触发台词/梗：
  - 散华：“我的眼睛可以看见生物本身的频率。或者说，我只能看到‘频率’。”
  - 散华：“您是您存在于世间，本来的样貌。”

:: 主线事件5：研究院体检与糖丸之谜 ::

> 事件概况
- 三人前往华胥研究院，遇到暴躁的莫特斐，他协助拆解日晷发现是机关盒，需要“时间”和“方位”开启。
- 白芷亲自为漂泊者进行身体检查，她的手指仔细地触碰、按压漂泊者吸收声骸的右手，眉头紧锁。
- 白芷得出结论：漂泊者体内可能存在另一个空间或生物。
- 化验结果显示“糖丸”是二十年前的过期疫苗，对应干支纪年中的“未”年（未时）。

> 区块目标
- 目标1：通过白芷的触诊，增加肢体接触描写，同时引出“阿布”存在的科学解释。
- 目标2：解开第一个谜题线索（糖丸=未时）。

> 执行引导
- 描写重点：
  白芷冰凉的手指在漂泊者温热的手臂肌肤上游走检查的触感；莫特斐对日晷精巧结构的赞叹。
- 必然触发台词/梗：
  - 白芷：“更像是你的身体里存在着另一个空间或是……生物。”
  - 秧秧：“二十多年前的疫苗……难道糖丸指向的线索，与二十多年前的那场今州儿童疫病有关？”
