[章节标题]
剧情版本v2.3 Part 4: 赞妮伴星-夜行的焰光（下）

[章节核心目标]
- 目标1：揭示赞妮的“焰光的夜行者”身份，并了解其行动的深层动机。
- 目标2：通过与线人柯林的接触，找到并潜入新“黑街”据点，揭露其运作的残酷真相。
- 目标3：见证赞妮的“焰光”形态，并合力击败新“塔洛斯”，彻底终结黑街的威胁。
- 目标4：深化与赞妮的关系，通过标志性的“摸头”互动与信物交换，和贯穿全文的“及格万岁”，确立超越普通同伴的羁绊。

[章节前置状态]
- 地点/环境：拉古那市中心，清晨。阳光明媚，空气清新。
- 主角状态：漂泊者正在进行日常活动，突然收到了来自赞妮的通讯。
- 伴随NPC：无。
- 指引行为：赞妮的通讯传来，她的声音虽然平稳但背景音暴露了她的焦躁。她提到了一个本应消失的名字“塔洛斯”，并以“有事相求”和“加班福利”为由，约你在市中心的冰激凌摊见面。

[章节事件列表]
1. 主线事件：冰淇淋的坦白
2. 主线事件：下水道的求救信
3. 主线事件：绝望的避难所与觉悟之火
4. 主线事件：伪神的陨落
5. 主线事件：最后的录音
6. 主线事件：焰光下的约定

[章节终止条件]
漂泊者收下了赞妮赠予的武器，赞妮认真地做出“不会拖你后腿”的承诺，本次合作行动正式结束。

:: 主线事件1：冰淇淋的坦白 ::

> 事件概况
- 漂泊者抵达冰激凌摊，看到赞妮把每一个口味都点了一份，正对着一座小山般的冰激凌发愁。
- 赞妮将此作为“加班福利”，并坦言需要补充糖分才有勇气进行接下来的“坦白”。
- 她说明了家族员工近期频繁被骚扰的困境，并将唯一的突破口指向了只信任漂泊者的那个小女孩。
- 漂泊者直接点破了赞妮就是“焰光的夜行者”，赞妮先是一愣，狡辩信奉“及格万岁”的她不会做这种事。
- 漂泊者列举了昨晚的种种疑点作为证据。
- 赞妮感到羞耻，随即无奈捂脸承认，解释“焰光的夜行者”只是坊间传闻的中二外号，并理直气壮地宣称，自己当初剿灭黑街的最重要理由是“源头治理”，为了避免加班。

> 区块目标
- 目标1：通过对话揭示赞妮的秘密身份，完成角色形象的关键反转。
- 目标2：展现赞妮在强大之外“怕麻烦”、“爱吃甜食”、“会害羞”的日常一面，使其形象更丰满。
- 目标3：确立下一步的行动计划——由漂泊者担任诱饵，寻找小女孩。

> 执行引导
- 描写重点：
	视觉镜头：堆成小山的五彩冰激凌，赞妮把脸埋进手心、耳根泛红的窘迫模样。
	互动细节：赞妮递给漂泊者勺子，分享冰激凌的动作。

- 必然触发台词/梗：
  - 赞妮：“如果不补充点糖分，我就没有勇气面对接下来要坦白的事实了。”
  - 赞妮：“行侠仗义……那种说法太羞耻了。”
  - 赞妮：“这可是最重要的理由！这叫‘源头治理’！”
  - 赞妮：“既然新的‘塔洛斯’冒出来了，那就再清理一次好了。就当是售后服务。”

:: 主线事件2：下水道的求救信 ::

> 事件概况
- 入夜后，漂泊者与赞妮分头行动，成功引出新一批混混。
- 混混在被击败的边缘，抓住了玛格烈特餐厅的声骸“焙焙”作为人质进行威胁。
- 赞妮被这种下三滥的手段激怒，声音变冷，瞬间出现在混混面前，一记侧踢将其踹飞并嵌进墙里。
- 昨晚的少女柯林现身，承认在找他们，并解释了新“塔洛斯”用“终端”将人变成怪物的暴行。
- 柯林因“终端是家族技术”的传闻而不敢信任身穿制服的赞妮。
- 赞妮为了获取信任，当即通过通讯器命令自己的部下撤离，只留下自己和漂泊者。
- 柯林被赞妮的行为打动，带领二人找到中间人，并通过“物理交流”问出了新入口在下水道。
- 在入口处，赞妮递给漂泊者一个面具。漂泊者踢到一支录音笔，播放后得知是一名调查黑街的记者留下的，其中提到了一个叫“布兰登”的人。

> 区块目标
- 目标1：展现赞妮在面对卑劣行径时的愤怒与果决，强化其“正义执行者”的形象。
- 目标2：通过柯林的叙述，引出新“塔洛斯”和“终端”的核心设定，并制造“家族是否为幕后黑手”的悬念。
- 目标3：通过赞妮主动遣散援兵的行为，展现她的担当与对漂泊者的信赖，赢得柯林的信任。
- 目标4：引入“记者”和“布兰登”的暗线，为故事增添悲剧色彩和悬疑感。

> 执行引导
- 描写重点：
	视觉镜头：混混被踹飞嵌进墙壁的“大”字型。柯林因恐惧和犹豫而瑟缩的姿态。
	互动细节：赞妮将救下的声骸“焙焙”还给玛格烈特的温柔动作。

- 必然触发台词/梗：
  - 赞妮：“这种下三滥的手段，只会让人更加火大。”
  - 柯林：“大家都说……那是家族提供给她的技术。所以我……我不敢相信穿着制服的人。”
  - 赞妮：“现在，这里只有我和漂泊者了。能给我们一点信任吗？”
  - 赞妮：“戴上这个。我也得维护一下‘焰光的夜行者’的神秘感。”

:: 主线事件3：绝望的避难所与觉悟之火 ::

> 事件概况
- 进入下水道深处，柯林带领二人来到一处避难所，却发现一群人倒在地上，奄奄一息，他们是被塔洛斯的手下打伤后遗弃在这里的。
- 一位名为“胡伯”的老者认出了柯林，并道出真相：柯林早已通过“埃弗拉德金库”的招工考核，本可离开，却被塔洛斯势力强行逼回。
- 赞妮上前摘下面具，正式自我介绍，并搬出（虚构的）“员工手册第八预案”，宣布有义务帮助准员工柯林解决入职困难。
- 其他人认出赞妮正是考核时“一根手指掀翻贡多拉”的考官，并称赞“桂冠”漂泊者比“焰光的夜行者”还厉害。
- 看着眼前惨状，赞妮的决心发生质变。她将自己的终端交给柯林呼叫后援，并对漂泊者说，自己要打破“及格万岁”的原则，必须“全力以赴”。
- 在前往核心区的路上，积水挡路。赞妮毫不犹豫地用拳头砸开排水口的盖子，展现出急切的心情。
- 漂泊者调侃她所谓的“第八预案”，赞妮坦然承认是瞎掰的，并笑着说漂泊者也在做着吃力不讨好的事，两人相视一笑，默契加深。

> 区块目标
- 目标1：揭示柯林与赞妮所在组织的直接联系，为赞妮的介入提供更坚实和个人化的动机。
- 目标2：通过受害者的惨状，极大地提升事件的严肃性和紧迫感，推动赞妮的情感转变。
- 目标3：完成赞妮从“敷衍了事”到“全力以赴”的关键心理转变，使其后续爆发更具说服力。
- 目标4：通过战斗间隙的轻松互动，加深漂泊者与赞妮之间的战友情谊和彼此认同。

> 执行引导
- 描写重点：
	视觉镜头：倒在地上的老人们绝望的神情，赞妮摘下面具后严肃的脸，她眼神中燃起真正怒火的瞬间，以及最后与漂泊者相视而笑的轻松片刻。
	互动细节：赞妮将终端递给柯林的信赖动作，用拳头砸开机关的果决。

- 必然触发台词/梗：
  - 胡伯：“要不是那个塔洛斯突然来了，你早就离开这里了……”
  - 赞妮：“我是赞妮，来自埃弗拉德金库。根据员工手册里的第八预案……”
  - 赞妮：“朋友，看来我要食言了。原本只打算做到及格的程度，但现在必须全力以赴了。”
  - 赞妮：“（瞎掰的）那是当然了。……彼此彼此。”

:: 主线事件4：伪神的陨落 ::

> 事件概况
- 两人抵达黑街的核心区域，新塔洛斯正高高在上地坐在一个杂物堆砌的“王座”上。
- 面对挑衅，塔洛斯按下终端，她的身体开始异化，变成一个手持长剑、身披残破甲胄的骑士型残象，散发着狂暴的威压。
- 她从高处跃下，挥舞长剑带着风压斩向赞妮。赞妮召唤出自己的变形大剑，与之展开激战。剑刃碰撞的火花四溅，最终赞妮凭借更胜一筹的技巧与力量，一记重劈击溃了塔洛斯的残象形态，将其打回人形。
- 被打回原形的塔洛斯愤怒地大喊：“你就是靠你那把武器！”
- 听到这句指责，赞妮收起了自己的大剑，体内迸发出炽热的白光，头发随之变为燃烧般的白色，衣服也随之崩裂露出胸前丰硕白皙的乳沟。
- 塔洛斯看到这个形态，瞳孔剧烈收缩，声音因恐惧而变形：“那燃烧的白发……焰光的长发……你是那个‘焰光的夜行者’？！”
- 战斗进入了拳拳到肉的阶段。酣畅淋漓的交锋中，塔洛斯的蛮力逐渐不支，战斗迅速演变成了赞妮对她的单方面压制。赞妮的每一次攻击都精准而沉重。
- 在彻底压制对方后，赞妮的身影瞬间消失在原地。下一秒，一只如同铁钳般的手死死扣住了塔洛斯的咽喉，单手将她举至半空。
- 赞妮凝视着因窒息而挣扎的塔洛斯，用冰冷刺骨的声音宣告：“如果你对秩序已经没有了敬畏，那就恐惧我吧。”
- 话音刚落，赞妮奋力一拳砸向她的脸，将塔洛斯重重砸在墙上，使其彻底昏死过去。

> 区块目标
- 目标1：详细描绘赞妮“焰光”形态的战斗方式，展现其压倒性的力量与平日的反差。
- 目标2：通过战斗中的台词，塑造赞妮对“秩序”的理解和对“恃强凌弱者”的绝对压制。
- 目标3：彻底击败反派，解决“黑街”的直接危机。

> 执行引导
- 描写重点：
	视觉镜头：剑刃碰撞的火花，赞妮收起大剑的决绝动作，她头发变白、周身散发白光的变身瞬间。拳拳到肉的打击感，以及单手将敌人举起的恐怖力量感。
	互动细节：赞妮冰冷的眼神中倒映着塔洛斯涨红的脸，以及她宣告时一字一顿的压迫感。

- 必然触发台词/梗：
  - 塔洛斯：“你就是靠你那把武器！”
  - 塔洛斯：“焰光的长发……你是那个‘焰光的夜行者’？！”
  - 赞妮：“在谈论秩序之前，先学会别给人添麻烦吧。”
  - 赞妮：“看着我，懦夫。”

:: 主线事件5：最后的录音 ::

> 事件概况
- 战斗结束，赞妮虽然还保持着焰光形态，但气场收敛，变回疲惫的家族员工，向漂泊者提议翘班喝咖啡。
- 在离开的路上，漂泊者在阴影中发现了第四份日志，旁边有大片干涸的血迹和钝器砸出的凹坑。
- 播放录音，传来记者极度虚弱的声音，他描述自己手已断，意识不清，但仍在坚信朋友布兰登已经逃脱。
- 录音的最后，记者温柔地低语，像是自我欺骗，说要出去后给布兰登一拳。录音在此结束。
- 现场的痕迹无声地诉说着记者最后的结局，他和布兰登都未能逃出。

> 区块目标
- 目标1：揭示记者的最终命运，为“黑街”的残酷画上一个悲剧性的句号。
- 目标2：在激烈的战斗后，通过这段悲伤的录音，营造出一种沉重、引人深思的氛围。
- 目标3：强化赞妮“守住剩下的日常”的决心。

> 执行引导
- 描写重点：
	视觉镜头：地面上干涸发黑的喷溅状血迹，录音笔上沾满的污泥。黑暗与光明的交界处，两人沉默的身影。
	互动细节：漂泊者与赞妮在听完录音后长久的沉默。

- 必然触发台词/梗：
  - 记者：“英白拉多在上……让这个受诅咒的鬼地方……在暗无天日的下水道里慢慢腐烂吧……”
  - 记者：“布兰登，等我在外面找到你，我一定会狠狠给你一拳的……你给我等着吧……”
  - 赞妮：“……走吧。我们能做的，就是守住剩下的日常。”

:: 主线事件6：焰光下的约定 ::

> 事件概况
- 回到地面，赞妮在阳光下停下脚步，真诚地感谢漂泊者的存在与付出。
- 她极其自然地伸出手，像安抚后辈一样，轻轻摸了摸漂泊者的头。
- 动作完成后，赞妮猛然意识到不妥，瞬间收回手，脸上闪过尴尬并道歉。
- 漂泊者看着有些慌乱的她，向前一步，带着“记仇”的玩味语气说：“我要摸回来。”
- 在赞妮愣住的瞬间，漂泊者伸手覆盖在她的头顶，指尖轻柔地触碰到了她标志性的双角。
- 赞妮的身体僵硬了一瞬，微微睁大眼睛，本能想躲但最终没有动，任由漂泊者的手停留在头上。
- 片刻后，她不自然地别过头，耳尖泛红，为了掩饰尴尬，迅速掏出一把指虎递给漂泊者。
- 她将这件曾经的武器作为“封口费”或“契约”送出，漂泊者调侃她说的“及格万岁”，而赞妮则回应不能这样了，因为这样会给漂泊者拖后腿。


> 区块目标
- 目标1：通过“摸头”与“反向摸头”的亲密互动，营造暧昧的氛围，将两人的关系推向新的高度。
- 目标2：展现赞妮在强大和干练之外，作为普通女孩会害羞、会慌乱的一面。
- 目标3：通过赠送武器这一行为，确立两人之间无需言说的默契与“契约”。

> 执行引导
- 描写重点：
	视觉镜头：赞妮在阳光下真诚的眼神，她摸头后尴尬收手的瞬间，漂泊者指尖触碰她双角的特写，她泛红的耳尖和不自然的侧脸。
	互动细节：赞妮身体从僵硬到放松的细微变化。漂泊者在说出“我要摸回来”时，语气中应带有的那份平静而又略带戏谑的感觉。两人之间因尴尬而凝滞，又因互动而变得温暖的空气。

- 必然触发台词/梗：
  - 赞妮：“谢谢，辛苦了。”
  - 赞妮：“啊，抱歉……习惯……”
  - 漂泊者：“我要摸回来。”
  - 赞妮：“……咳。这你可没提前说啊。”
  - 赞妮：“你可以把它当封口费，也可以当一份契约。”
