:: 主线事件1：方块间的真心话 ::

> 事件概况
- 达妮娅拉着漂泊者进入深潜游戏室。她先提议玩拉海洛方块——漂泊者拒绝，说"会想起不好的事情"（指她用方块攻击过他）。
- 两人最终从"接好饭"开始。游戏角色恰好是达妮娅和西格莉卡的形象，达妮娅看得别扭，抢过操纵杆后不出意料地输掉了。
- 换到拉海洛方块时，达妮娅展现了碾压级的技术。漂泊者在对战中忽然开启话题："你今天，是打算和她道别的吧？"
- 达妮娅手指一滞，坦然承认——她本想请西格莉卡吃蛋糕，趁气氛不错把一切坦白。
- 漂泊者追问是道别生日、道别本人、还是道别谎言，她说"都有吧"，又说本以为西格莉卡会更生气。
- 漂泊者问她为什么希望对方生气。达妮娅盯着屏幕上的方块说："死掉一个曾经的朋友，总比死掉一个现在的朋友要好受。"漂泊者说"她不会这么想"。
- 达妮娅的手从操纵杆上滑落："所以我才讨厌你们这样的人——明明可以选择逃避，却偏要把一切都背负起来。"漂泊者起身走到她身后说："可你仍然鼓励了她。"达妮娅沉默片刻，忽然惊讶地发现漂泊者趁她说话时把分数超过了。
- 达妮娅没好气地吐槽漂泊者“好狡猾”，漂泊者则说“不这样可赢不了你”。

> 区块目标
- 目标1：通过游戏对战作为"安全距离"，让达妮娅在手指忙碌时可以不用直面漂泊者的目光说出真心话。
- 目标2：触及达妮娅的核心矛盾——她一边说着"讨厌"漂泊者和绯雪这种"不逃避的人"，一边却又在关键时刻做出了和他们相同的选择（鼓励西格莉卡、引开虚诞虫）。

> 执行引导
- 描写重点：
  视觉镜头：方块下落时屏幕的冷光在达妮娅脸上跳动。漂泊者说"可你仍然鼓励了她"时他已经在达妮娅身后——从她头顶俯视屏幕的角度。达妮娅发现分数被超过时嘟起嘴的赌气表情。
  互动细节：漂泊者从机器旁走到达妮娅身后这个位移——从并肩到站在她背后，位置关系微妙地呼应了"守护"的意象。
- 必然触发台词/梗：
  - 漂泊者："你今天，是打算和她道别的吧？"
  - 达妮娅："死掉一个曾经的朋友，总比死掉一个现在的朋友要好受。"
  - 达妮娅："明明可以选择逃避，却偏要把一切都背负起来……我就是讨厌这样的生存方式。"
  - 漂泊者："可你仍然鼓励了她。"
- 伏笔/暗线：
  达妮娅"讨厌"这个词的反复使用——在后续的残响留言中，这个词将转化为"喜欢"。

:: 主线事件2：文献中心的别离——娜斯塔霞的礼物 ::

> 事件概况
- 两人前往文献中心。图书管理员娜斯塔霞老远就认出达妮娅——她入学第一周在这里补觉时两人就认识了。娜斯塔霞提到以前达妮娅"仗着那不靠谱的泡泡就往校外跑"帮她收集虚质磁暴数据，一边说一边伸手点了达妮娅的脑门。
- 当得知今天是达妮娅生日，娜斯塔霞拿出了一份临时的礼物——罗伊族童话故事的实体彩印合集，这是达妮娅曾经和她念叨过的。达妮娅接过书时注意到娜斯塔霞伸出的手已经变成了机械义肢。
- 追问之下，娜斯塔霞轻描淡写地承认这是之前事件中的虚质侵蚀后遗症，并说她已不适合留在拉海洛——环境可能加剧神经侵蚀，深空联合建议她离开休养。她说得轻松，甚至自嘲"自己的论文成了劝自己退休的佐证"。
- 达妮娅难以置信地抓着自己的玩偶——娜斯塔霞是那么优秀的人，她的梦想是亲手揭开虚质粒子的秘密。
- 娜斯塔霞却说梦想有时需要换个地方继续。达妮娅喃喃说"所有的事，全都是因为我"，漂泊者注意到她的频率已经剧烈波动。
- 他替达妮娅向娜斯塔霞告辞，带着她匆匆离开文献中心。身后传来娜斯塔霞的喊声："生日快乐！达妮娅。"——达妮娅的眼眶霎时蓄满泪水。

> 区块目标
- 目标1：通过娜斯塔霞的遭遇，让达妮娅直面自己参与残星会计划造成的具体后果——不是抽象的道德愧疚，而是眼前友人被侵蚀的肢体与被迫放弃的梦想。
- 目标2：展现漂泊者作为"现场负责人"的判断力——他迅速察觉达妮娅的状态失控并果断结束会面。

> 执行引导
- 描写重点：
  视觉镜头：娜斯塔霞递书时机械手指的关节发出细微的齿轮声。达妮娅接过书后目光死死钉在那只手上。漂泊者说"抱歉，我们得先告辞了"时伸手轻扶住达妮娅的臂肘——她几乎站不稳。
  互动细节：娜斯塔霞点达妮娅脑门——一个长辈式的亲昵动作，用尚存的那只手。达妮娅低头咬嘴唇时攥着玩偶的手指关节泛白。
- 必然触发台词/梗：
  - 娜斯塔霞："你呀，以后少点逞强，也学习下怎么依赖别人吧。"
  - 达妮娅："所有的事情，全都是因为我……"
  - 漂泊者："冷静点，达妮娅。"
  - 娜斯塔霞（远处）："生日快乐！达妮娅。"
- 伏笔/暗线：
  娜斯塔霞的"毕业"与达妮娅自身的"时间将尽"形成悲剧性的对照——她们都在以不同的方式告别星炬学院。

:: 主线事件3：长椅上的自白与心之证明 ::

> 事件概况
- 漂泊者将达妮娅带到花园区偏僻的长椅上，让她服下安定药物。他暗中判断——从并发症出现时的频率来看，情况比阿里曼说的更严重，这样下去根本撑不了一周。她选择今天，或许正是因为知道自己撑不到下一个周末。
- 漂泊者将娜斯塔霞的礼物轻轻放在她膝上。沉默许久后，达妮娅开口了——她向漂泊者坦白：今天根本不是她的生日。她之所以选这个借口，是因为"每个人都有生日……残星会没有我的诞生记录，也没有任何身份证明。只有一段始终无法洗去的影像，一次又一次在梦中播放"。影像中，一个女人在风中摇晃手臂，对她喊"达斯维达妮娅"。
- 漂泊者问她是否想过活下去。她反问"我配吗"，随后自嘲地说自己唯一擅长的就只有说谎。漂泊者没有直接回答，而是讲述了黎那汐塔的经历——阿维狄亚、蜜芽，那些被强加生命却为了片刻幸福战斗到最后的人。"决定你是什么的，从不是你的来处，而是你的心。"
- 达妮娅沉默，然后抬起头，引用了娜波摩当年的话："抬起头，你看见了什么？"漂泊者说："拉海洛的夜空。"达妮娅说，她当年的回答是"虚无，还有黑暗"
- 漂泊者接道："但最初……拉海洛本就是一片黑暗。后来才有了光。所以是光赢了。"达妮娅轻笑："你的回答和她一模一样。"——娜斯塔霞曾告诉她，那是光明在对抗黑暗。

> 区块目标
- 目标1：完成达妮娅的核心自白——她不仅不是真正的"学生"，甚至没有真正的"生日"，她对自己是否配拥有"人的身份"的质疑是贯穿她一生的问题。
- 目标2：让漂泊者用"心"这个概念回应她的存在危机——不是空洞安慰，而是他亲身见证过的具体故事。

> 执行引导
- 描写重点：
  视觉镜头：长椅上方拉海洛人造夜空的全息投影——那颗新生的太阳"赫利俄斯"已经沉没，只剩微弱的群星模拟光。达妮娅说"没有诞生记录"时没有哭，但说"有人在风中摇晃手臂"时声音破了。漂泊者将娜斯塔霞的书放在她膝上——这个动作比任何语言都先一步告诉她"你值得被认真对待"。
  互动细节：达妮娅讲述梦境中的女人时双手下意识环抱住自己的肩膀。漂泊者讲黎那汐塔的故事时，视线在地面——他说的每一个字都是真实失去过的人。
- 必然触发台词/梗：
  - 达妮娅："如果我真的有一个名字，是不是再加上一个生日，就能以一个人的身份去死了？"
  - 漂泊者："决定你是什么的，从不是你的来处，而是你的心。"
  - 达妮娅："抬起头，你看见了什么？"
  - 漂泊者："但最初拉海洛本就是一片黑暗。后来才有了光。所以是光赢了。"
- 伏笔/暗线：
  达妮娅引用的娜波摩/娜斯塔霞关于"抬头看见什么"的问答，成为贯穿她一生的主题——从虚无到光明的转变。

:: 主线事件4：书页、蛋糕与午夜逃亡 ::

> 事件概况
- 漂泊者问"你愿意相信我吗"。达妮娅愣了愣，随即抓住他的手，引向自己锁骨下方那个连接着宝石的锁链。她说会长留下的"礼物"埋在她的频率中，是一页书页——利维亚坦的鸣式碎片。漂泊者运起风属性的分离权能，一片泛着金色微光的书页从她体内析出。
- 终端传来阿里曼的紧急通讯——他们监测到了剧烈的频率波动。漂泊者回应一切顺利，即将返回。达妮娅趁这间隙恢复了嬉笑神态，说"还差那么最后一点点，暂时保密"。她提醒漂泊者蛋糕还没取，"明天以后，漂泊者还会想要再和我一起出来吗？"漂泊者说"会的"，她笑着说"那可麻烦了，我也有不得不去做的事"。
- 回车站交接后，阿里曼收到达妮娅，答应尽快研究书页上的频率。漂泊者独自前往游牧集市的扎希拉店铺取生日蛋糕——橘子口味，加了跳跳糖和喀拉喀拉。
- 就在他接过礼盒的瞬间，终端上时间显示零点十五分——达妮娅的"生日"已经过去了。几乎是同时，IRIS的通话接入：达妮娅利用恢复的共鸣能力逃离了车站，阿里曼被暂时困住但未受伤。她没有拆除抑制装置和定位设备。
- IRIS询问是否启动紧急抑制备案，漂泊者回答"不——把她的位置同步给我"。一只白鸟纸役掠过，绯雪出现在他身旁。
- 绯雪说道她试过阻止，但"她的身体状态太差，继续下去她会死"。漂泊者说她不像平时的作风，绯雪答："那孩子只是，做出了大人的选择。"

> 区块目标
- 目标1：完成交易的核心步骤——取出书页，回收关键情报（鸣式碎片之间的相互作用）。
- 目标2：通过午夜时间提示和达妮娅的逃离，制造剧情转折——达妮娅从"被押送的收容对象"变成了"自主选择终局的人"。

> 执行引导
- 描写重点：
  视觉镜头：漂泊者接过蛋糕盒时扎希拉说"虽然晚了一点，但我相信它足以传达你们的心意"——盒子上凝结的水珠像某种隐喻。IRIS说"她并没有选择拆掉共鸣抑制装置"时，漂泊者望向远处车站的方向。
  互动细节：达妮娅将漂泊者的手引向自己锁骨——这是她今天最主动、最毫无防备的动作。绯雪出现时没有看漂泊者，她在看手中那只白鸟纸役。
- 必然触发台词/梗：
  - 达妮娅："在这里。会长留下的礼物——埋在我频率中的书页……就在这里。"
  - IRIS："达妮娅目前已经离开了星炬学院，正在前往黯原区。她没有拆掉共鸣抑制装置和定位设备。"
  - 绯雪："那孩子只是，做出了大人的选择。"
- 伏笔/暗线：
  达妮娅说"我也有不得不去做的事"——她真正的目的将在残响留言中揭晓：会长发来了最后信息，她是阿列夫一与索拉里斯之间的最后一个锚点。

:: 主线事件5：黯原深处的实验室——漂泊者的调查 ::

> 事件概况
- 在阿布对鸣式频率的感知引导下，漂泊者穿越黯原的虚质污染区，追踪达妮娅的定位信号来到一处隐蔽的废墟——这里是残星会在拉海洛的秘密实验室，也是达妮娅被"创造"的地方。
- 漂泊者穿过实验区，沿途查看遗留下的痕迹：药品柜中满是来自新联邦的营养剂和抑制剂，标签被人为更换过——"难吃但好用的止痛药""有点甜的安定剂""发烧用"——全是手写字迹。一张带有拘束装置的手术床，输液架上贴着"飞行雪绒"贴纸，床头还绑了个枕头。实验终端里除了密密麻麻的临床测试数据，只安装了一款系统自带的游戏——拉海洛方块。冰箱里塞满了同样的压缩食物，毫无口感——漂泊者尝了一口便明白了达妮娅为什么会对食堂的甜品那样执着。
- 往里走是一个杂乱却充满生活痕迹的小房间：一张从废墟中捡来的破桌子，上面堆着《专业情感表演指南》（被拿来当草稿纸了）、虚质学教材、《海豹也能懂的入学指南》、《拉海洛气象学》——娜斯塔霞借给她的，里面还夹着去年的试卷影印件，达妮娅曾经因为背错答案被笑话。桌子角落是一本摄影部的纪念相册，漂泊者翻开——每一页都有达妮娅，每一张照片里她都在笑。摄影部的人曾对她说："笑一下会更受欢迎哦。"
- 实验室角落的一个小门敞开着，门口歪歪扭扭挂着一块牌子："达妮娅的卧室，不许随便进来"。漂泊者走进门，通往最深处的培养舱。满地是碎裂的培养管和红色液体。漂泊者跨过碎片，在房间中央发现了一个悬浮的残响气泡。

> 区块目标
- 目标1：通过漂泊者的双眼将达妮娅视角的实验室探索转化为第一手的调查体验——让漂泊者（和玩家）从这些物件中拼凑出达妮娅十八年生命中无人见证的部分。
- 目标2：在悲剧高潮到来前，用生活痕迹的细节建立沉浸式的哀悼氛围——达妮娅她"活过"，她挣扎过，她在课业和谎言之间搭建了一个属于自己的居所。

> 执行引导
- 描写重点：
  视觉镜头：药品柜上手写标签的字迹——歪歪扭扭、反复涂改过的笔画，出自一个从没有人教过她书写的孩子。相册中每一张照片里的笑容——从最早的"僵硬得像被命令"到后来的"被偷拍时的自然放松"，达妮娅在学会"情感表演"的过程中同时学会了真实的情感。破碎的培养舱——那是她诞生的地方也是她选择终结一切的地方。
  互动细节：漂泊者尝压缩食物时眉头皱起——他终于明白达妮娅吃蛋糕时为什么那么兴奋。翻阅相册时手指在每一页停留的时间越来越长。进入培养舱房间时，阿布钻出衣领发出微弱的呜声。
- 必然触发台词/梗：
  - 门上字条："达妮娅的卧室，不许随便进来"
  - 药品标签："难吃但好用的止痛药""有点甜的安定剂""发烧用"
  - 相册中摄影部员："笑一下会更受欢迎哦？"
  - 终端记录："截止实验结束，实验组生理指标一切正常，未出现组织畸变，体重无明显变化。"
- 伏笔/暗线：
  所有药品均为娜波摩经手，标签是她的手写——达妮娅在漫长的实验期中收到的唯一"照料"来自一个冷酷的观测者。那款系统自带的拉海洛方块——是她孤独时光中唯一的娱乐，也是后来她用方块攻击漂泊者的"武器"的来源。

:: 主线事件6：残响气泡与未完的约定 ::

> 事件概况
- 漂泊者触碰了残响气泡。达妮娅的身影在微光中浮现——从她踏入实验室的那一刻开始播放。
- 残响展示了达妮娅最后的行动——她在培养舱前站了很久，伸手摸了摸输送红色液体的橡胶管。然后她开始唱生日歌，声音很轻，每一句都在强撑笑意。
- 唱完后她咬紧了嘴唇，随即变身——用尽残余的全部共鸣力歇斯底里地打碎了培养舱和周围的一切。碎片飞溅中，她从怀中取出了什么东西，对准自己的胸口——那是失控的鸣式碎片（书页），正贪婪地寻找阿列夫一的痕迹。
- 她猛地将它刺入心口，鲜血迸发，随即一阵白光的漩涡将她吞没——她与碎片一起被卷入虚质空间的深处。弥留之际，她的虚影低低说了最后一句话。
- 影像消散后，达妮娅留给漂泊者的留言开始播放——她的声音比任何时候都要平静。她知道漂泊者会来到这里。
- 她解释了一切：被取出的书页中只有五个字——"常回家看看"。但书页被抽出时，她收到了会长的最后一条信息。她是阿列夫一与索拉里斯之间的最后一个锚点。如果她如预期般死去，鸣式碎片会与她一同消亡，而"预期"正是——知道真相的漂泊者亲手杀死她。
- 会长要的不是她的死，而是漂泊者背叛自己的心。但如果漂泊者真的取出了书页，失控的碎片会本能地寻找她。所以那个"如果"没能说出口真是太好了——她必须亲自带走碎片，所以她选择亲手杀死自己。
- 然后她开始说一些柔软的话。她说"讨厌"漂泊者——从会长那里第一次听到关于他的故事时就讨厌。为什么在漫长痛苦的时光中那个人从没出现，偏偏在一切即将结束时出现在她面前。为什么与你相关的事，总这样不讲道理。
- 她还说了很多关于未来的想象：想去看看娜斯塔霞说的城市，认识一个没有虚质磁暴的地方，尝试在不同的课上睡觉，品尝所有不同食堂的蛋糕。然后无论多少次都会回到拉海洛，回到漂泊者身边。
- 然后她留了一个暗号，以防会长假扮她的模样——下次见面时，真正的她会把所有"讨厌"换成"喜欢"。最后她提到了蛋糕："帮我吃掉一半。不许把上面添加的东西扔掉。"
- 残响消散，房间复归寂静。漂泊者拿出了那个橘子蛋糕的盒子。

> 区块目标
- 目标1：以达妮娅的残响留言作为整个间章的情感终点——让她的告别不是被动的"死亡"，而是主动的"选择"——她以自己的方式保护了漂泊者的心、西格莉卡的未来、以及拉海洛不被鸣式碎片吞噬。
- 目标2：完成"讨厌"→"喜欢"的情感转化铺垫——她在残响中用一百种方式说"讨厌"，却每一个字都是"喜欢"的倒影。

> 执行引导
- 描写重点：
  视觉镜头：残响气泡中达妮娅的虚影——她说话时偶尔会低头把玩自己玩偶的衣角（和生前一样）。说到"果然，是讨厌吧"时她笑了一下，但虚影边缘在微微闪烁。残响消散时气泡碎裂成细小的光点，有一片飘到蛋糕盒上然后熄灭。
  互动细节：漂泊者全程沉默——达妮娅留下这个残响时已经知道他会沉默，她甚至在台词中预留了回答的空隙。残响消失后他仍然站在原地。最后是否把蛋糕留在实验室——这个选择不需要语言。
- 必然触发台词/梗：
  - 残响影像中达妮娅最后的话：啊……没能吃到蛋糕，还是有点可惜呢。
  - 达妮娅（残响留言）：我是阿列夫一与索拉里斯之间的最后一个锚点。如果我如预期般死去，它的碎片会与我一同消亡。
  - 达妮娅（残响留言）：如果这个世界上真的存在这样的人，那为什么在我最漫长痛苦的时光中，那个人从来就不曾出现过？这样的相遇，真的太不公平了。
  - 达妮娅（残响留言）：下次见面时，真正的我会把所有挂在嘴边的"讨厌"全都替换成"喜欢"。
  - 达妮娅（残响留言）：会长目前使用过的化形能力在内的诸多手段，全都不是她真正的共鸣能力。她必须对被模仿者进行长期观测和解析。
- 伏笔/暗线：
  暗号的设定——为未来可能的"重逢"场景提供了识别真伪的方法。会长具有化形能力，而漂泊者如今知道如何确认眼前的人是不是真正的达妮娅。
