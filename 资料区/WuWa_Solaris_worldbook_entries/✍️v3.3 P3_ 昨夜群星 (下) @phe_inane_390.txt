:: 主线事件1：救世主之路——与绯雪的对话 ::

> 事件概况
- 两人默契配合，一番苦战之后，达妮娅用尽最后的体力，撕裂崩解了战场上方的建筑，化为巨石砸向绯雪。
- 绯雪共鸣解放，瞬身到达妮娅面前，刺骨的寒意从她的身上蔓延至达妮娅的身体令她动弹不得。在达妮娅恐惧的注视下，绯雪一斩击败了她，剑气顷刻间化为冰雪崩裂，击碎了头顶的巨石。
- 达妮娅昏倒瞬间，绯雪将她稳稳接在怀里，为她施以安眠。漂泊者选择将达妮娅托付给绯雪，孤身前往虚质空间深处。
- 绯雪为漂泊者的白鸟预取了"最完美使用术法的未来"，使其能在高浓度虚质中引路护航。
- 临别前，绯雪向漂泊者提出一个深层的问题：爱弥斯曾在意向表上写"拯救世界"，这是受漂泊者影响——"你会觉得后悔吗？引领她走上'救世主'的道路，让她遭受痛苦和不幸。"
- 绯雪认为"这条无尽长路，唯有独行才能保护他人"，越是因憧憬救世主而走上拯救之路，越会深陷泥泞——这是她自身在苇原失败经历的投射。
- 漂泊者坚定回答：不是为了诉说后悔才去寻找爱弥斯。他一直为她感到骄傲，因为有她同行，他知道自己并不孤独，知道即使有朝一日失败，也有人会继续守护索拉里斯。
- 漂泊者继续回应绯雪：他不否认参与了各文明自救的命运，但若将自己视为唯一因素，是傲慢；他的胜利从来不只因他一人；只要一人能到达终点，拯救之路就不存在失败者。
- 漂泊者劝绯雪："在你独自前行的时候，如果感到孤独，就停下来看看吧。你一定已经成为了某些人的支撑，也一定有人会为你感到骄傲。"
- 绯雪低语"耀眼得令人嫉妒"，想起了姐姐玉露。她有些话想说，但留到漂泊者回来时再谈。
- 漂泊者承诺"去去就回"，绯雪回应："我会在夜幕展开前守在这里，和大家一起……等你回来。"

> 区块目标
- 目标1：通过"救世主应当独行"与"拯救之路无失败者"的精神交锋，完成漂泊者与绯雪的价值观碰撞与互相触动。
- 目标2：揭露绯雪内心深处因苇原失败而累积的自我否定与孤独感，以及她对姐姐玉露的缅怀。
- 目标3：在漂泊者即将进入最危险阶段前，以一场关于"信念"的对话完成情感蓄势，为最终章的高潮做好铺垫。

> 执行引导
- 描写重点：
  视觉镜头：黯原深处昏暗的裂隙口前，绯雪与漂泊者相对而立；来自寂静中的声音浮现时，画面短暂变白如记忆闪回。
  互动细节：绯雪说"耀眼得令人嫉妒"时微微偏过头，似乎不愿让漂泊者看到她的表情。
- 必然触发台词/梗：
  - 绯雪："这条无尽长路，唯有独行才能保护他人。"
  - 漂泊者："我一直为她感到骄傲。"
  - 来自寂静之中的声音："但愿我会让你感到骄傲，但愿我没有让你失望……"
  - 漂泊者："而只要一人能到达终点，这条拯救的长路，就不存在失败者。"
  - 绯雪："……耀眼得令人嫉妒。"
  - 绯雪："我会在夜幕展开前守在这里，和大家一起……等你回来。"
- 伏笔/暗线：
  来自寂静中的声音暗示爱弥斯在虚质深处仍有意识，她的话语与漂泊者的回答形成跨越空间的情感共振。绯雪的"嫉妒"源于她从未能像漂泊者那样在同行者的陪伴中前行，呼应她独自在苇原的漫长战斗。

:: 主线事件2：先行者们的群星 ::

> 事件概况
- 视角切换至爱弥斯在虚质空间中的经历。隧者的存在如同一座孤岛，爱弥斯发现了被虚质吞噬的人们——他们穿着深空联合和罗伊族的服饰，是过去被虚质磁暴吞噬的探索者。
- 爱弥斯试图用隧者所剩无几的能量保护他们，但他们的存在正在被虚质"消解"。
- 先行者们在迷茫与绝望中讨论——时间和空间失去了意义，声音无法传递，思想却无处不在。
- 其中一人发出希望之声："不正是为了触及不可知的边界，才来到这片星海下的吗？不过是探索长路上的又一步。"
- 先行者们做出选择：以自己作为坐标，用生命测绘出以隧者为原点的时空间坐标——"现在，我们将在过去篆刻未来。"这是一份来之路的标记："这是我们的来时路。"
- 爱弥斯含泪接受："你们亲身测绘出的这份'可知'，我一定会传达出去。"她将这些数据刻录在磁带中。
- 漂泊者循频率抵达，与真正的、具有完全自我意识的爱弥斯重逢在隧者体内。
- 漂泊者请求爱弥斯去将拟制炉芯带来——"别担心，这次我不会再选择独自拯救。"爱弥斯回应："等着我。"

> 区块目标
- 目标1：揭示"先行者们"——被虚质吞噬的探索者群体的存在与最终抉择，以"群星"之姿诠释标题"昨夜群星"的核心含义。
- 目标2：完成爱弥斯与漂泊者的重逢，以及漂泊者"不再独自拯救"的信念转变。
- 目标3：通过先行者们以生命测绘坐标的壮举，揭示此前"原点的光，由我们支撑"这句话的完整含义。

> 执行引导
- 描写重点：
  视觉镜头：虚质空间中如星辰般散落的人影；先行者们化为光点标注坐标时，宛如夜空中依次亮起的星星。爱弥斯在孤寂中仍在尝试保护他人的身影。
  互动细节：漂泊者说"这次我不会再选择独自拯救"时，语气平静却蕴含着对过去所有缺席的弥补。
- 必然触发台词/梗：
  - 先行者1："不正是为了触及不可知的边界，才来到这片星海下的吗？"
  - 先行者2："现在，我们将在过去篆刻未来。"
  - 先行者3："这是我们的来时路。"
  - 爱弥斯："原点的光，由我们支撑。"
  - 漂泊者："别担心，这次我不会再选择独自拯救。"
  - 爱弥斯："……嗯。我一定会为你带回人类的希望。"
- 伏笔/暗线：
  先行者们的坐标测绘数据最终将被爱弥斯传出，成为完善拟制炉芯的关键信息。"昨夜群星"的意象——昨夜已逝去的群星之光，今日才抵达此处，正如先行者们的牺牲在此刻照亮了未来。

:: 主线事件3：直面阿列夫一 ::

> 事件概况
- 爱弥斯前去取回拟制炉芯。漂泊者驾驶隧者，独自面对化为战斗形态机甲的鸣式阿列夫一。
- 阿列夫一以碎片化的词语传达其意志："星海，抵达，无意义，理由""追寻，边界，灭亡，叹息""证明，存在，你""终点，寂静"——它质疑一切探索的意义，宣告万物终将消解归于空无。
- 战斗一触即发，前期还是势均力敌的近身搏斗，不知打了多少回合，阿列夫一战斗形态机甲的力量突然呈指数级增长。隧者不敌，被打得瘫倒在地，阿列夫一开始了武器的蓄能——它试图一击终结隧者。
- 在对抗阿列夫一期间，莫宁在研究院紧张接收数据，拟制炉芯的研发进入最后冲刺。
- 千钧一发之际，爱弥斯携带完成的“拟制炉芯”，回到虚质空间内部，将其带入了隧者体内。隧者顷刻间觉醒，它散发着光芒，与之共鸣，获得了岁主级的力量，成功抵挡了阿列夫一的终结技，并发起反攻。
- 隧者被嵌入拟制炉芯后完成升格，发出声音宣告："'人类的心脏'正在我体内运转轰鸣，让我能发出和他们同样的声音。"
- 隧者正面回应阿列夫一的虚无论："即便人类探索的尽头，是永劫的虚无。阿列夫一，那也绝非你所判断的无意义。"
- 隧者阐述自己的觉醒："本不该孕育出文明的冰原，如今已诞生了渴望探索边界的新生集体意志。他们——选择了我。"
- 漂泊者、爱弥斯与隧者协力击退阿列夫一的战斗形态。


> 区块目标
- 目标1：完成隧者从"沉眠巨人"到"以人类之心觉醒"的蜕变，将拟制炉芯的嵌入塑造为全章最燃的转折时刻。
- 目标2：通过隧者对阿列夫一虚无论的回应，升华"探索意义"的核心主题——即便终点是虚无，旅途本身也绝非无意义。
- 目标3：以三方协力（漂泊者、爱弥斯、隧者）的战斗，兑现"不再独自拯救"的信念。

> 执行引导
- 描写重点：
  视觉镜头：拟制炉芯嵌入隧者胸膛的瞬间，粗糙的人造心脏开始轰鸣，裂纹中溢出的光芒如脉搏般跳动；阿列夫一作为宇宙级虚无存在的压迫感。
  互动细节：隧者首次发出"声音"时，语气带着机械的顿挫，却蕴含着某种初生般的庄严。
- 必然触发台词/梗：
  - 隧者："'人类的心脏'正在我体内运转轰鸣，让我能发出和他们同样的声音。"
  - 隧者："本不该孕育出文明的冰原，如今已诞生了渴望探索边界的新生集体意志。他们——选择了我。"
  - 隧者："即便人类探索的尽头，是永劫的虚无。阿列夫一，那也绝非你所判断的无意义。"
  - 阿列夫一："悲哀，悲哀，悲哀"
- 伏笔/暗线：
  隧者提到拟制炉芯"尖锐嘶哑的磨损"暗示它并非永久之物，为隧者最终选择留守并耗尽心脏做好铺垫。

:: 主线事件4：隧者的告别——「拉海洛」 ::

> 事件概况
- 战斗结束，隧者告知：阿列夫一并未被消灭，只是被放逐向了宇宙深处，它仍有能力再次回来。
- 隧者请求漂泊者允许它留守隧门这一侧看守门扉，阻挡阿列夫一再次进入索拉里斯——"请您允许我留在这里，守护他们到这颗心脏运转的最后一刻。"
- 漂泊者答应了隧者的请求。
- 隧者随后向共鸣者爱弥斯提出：将自己的部分力量分给她。隧者因最初在爱弥斯帮助下才能隔绝鸣式，对她如今的处境感到"愧疚"。这份力量也意味着曾经到来的死亡已与爱弥斯隔绝，她将与漫长的"存在"为伴。
- 爱弥斯笑着接受："我还要继续我救世主的职业道路呢，不厉害一点儿，可对付不了那些坏家伙吧？"
- 隧者讲述自己的来历：因使命远航至此，被阿列夫一追击坠落；借爱弥斯的共鸣驱逐了鸣式的一部分；本应封绝自身，却看到罗伊族在冰原上无助面对虚质灾厄的惊恐眼睛，遂取出炉芯嵌入冰层下，陷入休眠。
- 隧者注意到漂泊者携带了磁带，提出将一段频率刻录其中：在星海疾驰时曾与一架名为「眺望者」的古老探测器路线相交，记录下了它一直向索拉里斯发射的信号。隧者希望将此作为礼物送给拉海洛的人们。
- 最后，隧者请求更换编码，以「拉海洛」为名。漂泊者允许。
- 隧者——现在的「拉海洛」——完成告别："再见了，御者，请您替我向创造了这颗心脏的人们一并做暂时的道别。测算——我们会在无垠的星海中，再会。"

> 区块目标
- 目标1：完成隧者的人物弧光——从无言的机体到拥有"心脏"和"名字"的守护者，以改名「拉海洛」作为情感最高点。
- 目标2：通过分给爱弥斯力量，解决她的"存在"困境（不再面临死亡威胁），同时赋予她新的力量层级。
- 目标3：引出「眺望者」磁带作为最终章回归日常的情感纽带。

> 执行引导
- 描写重点：
  视觉镜头：隧者庞大的身躯缓缓关闭隧门，光芒从门缝中渐渐收窄直至消失；爱弥斯接受力量时周身环绕的光芒与隧者体内磨损心脏的脉动同步跳动。
  互动细节：漂泊者将磁带递出时，手指在磁带表面停留了片刻，这张磁带承载了他在拉海洛的全部身份。
- 必然触发台词/梗：
  - 隧者："请求：御者，我因人类的希望而生，请您允许我留在这里，守护他们到这颗心脏运转的最后一刻。"
  - 爱弥斯："我还要继续我救世主的职业道路呢，不厉害一点儿，可对付不了那些坏家伙吧？"
  - 隧者："尖锐嘶哑的磨损，这是我第一次听到'自己'的声音，这是'他们'赋予我的心跳声。"
  - 「拉海洛」："送别：再见了，御者，请您替我向创造了这颗心脏的人们一并做暂时的道别。"
  - 「拉海洛」："测算——我们会在无垠的星海中，再会。"
- 伏笔/暗线：
  隧者提到拟制炉芯的运转在"激烈地磨损着自身"，暗示它并非永恒，隧者的守护是有时间限制的。「眺望者」探测器来自天空海尚未覆盖索拉里斯的时代，为后续揭示索拉里斯古文明的探索精神做铺垫。

:: 主线事件5：归来与星海的回响 ::

> 事件概况
- 漂泊者与爱弥斯一同返回索拉里斯。爱弥斯感叹好久没看到索拉里斯的景色。
- 两人总结：不是英雄拯救世界，而是"世界拯救英雄"的故事——先行者们在虚质中测绘信息，拉海洛的人们解析磁带完成拟制炉芯，最终让隧者击退鸣式。
- 绯雪在黯原等候，漂泊者简短地回应"嗯，我回来了"。
- 深空联合公用频段发布战后报告：阿列夫一的影响未波及黯原以外区域；虚质浓度显著降低；鸣式浓度无限趋近于零；隧者反应已彻底消失但研究仍将继续；学院机兵清剿了异常造物；罗伊族将获得更广阔的活动区域。
- 数日后，漂泊者来到研究院，与爱弥斯、琳奈、西格莉卡、洛瑟菈、莫宁众人一起，聆听磁带中隧者留下的礼物的解析结果。
- 洛瑟菈揭晓：「眺望者」是天空海尚未覆盖索拉里斯时，被发射向天穹的九架系外探测器之一。深空联合标志上的九颗星星就象征着它们。天空海闭合后，索拉里斯再也没有收到过它们发回的讯息——直到今天。
- 磁带中传出宇宙白噪声——那是「眺望者」在不同星系中记录下的声音，是前人留下的星海之声。

> 区块目标
- 目标1：以"世界拯救英雄"的主题收束全章叙事，将群体的力量凝聚为核心表达。
- 目标2：通过「眺望者」磁带的播放，完成从危机到日常的情感转换，以古老探测器跨越时空的讯息为全章画上诗意的句点。
- 目标3：群像聚集的温馨场面，呼应此前各角色的送别与承诺，展现"凯旋"的兑现。
