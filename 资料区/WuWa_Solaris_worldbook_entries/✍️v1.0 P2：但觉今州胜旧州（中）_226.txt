- 目标3：接触黑海岸势力，查明主角曾在黑海岸苏醒的身世，并以全面战争爆发为悬念收尾。

[章节前置状态]
- 地点/环境：华胥研究院出口 -> 今州城门 -> 伏波营 -> 祈池村 -> 边庭 -> 稷廷遗址。

- 伴随NPC：秧秧随行；炽霞因值勤暂时离队；途中加入鉴心、秋水、安可。
- 指引行为：白芷的体检报告排除了身体隐患，漂泊者与秧秧决定前往伏波营调查莽吉柿来源。

:: 主线事件1：道士鉴心与伏波寻踪 ::

> 事件概况
- 在城门口，炽霞因值勤任务遗憾离队，并约定“随叫随到”。
- 遇到正礼貌请求出城的小道士鉴心，她受托寻找失踪者“致远”。漂泊者利用散华给予的最高权限带她一同出城。
- 三人抵达伏波营，军需官告知莽吉柿产自南方（朱雀方位）的怨鸟泽，因战事稀缺。
- 漂泊者脑海中再次闪现幻象：以残象视角目睹了北落野战场上，忌炎将军率军厮杀的惨烈景象。

> 区块目标
- 目标1：塑造鉴心谦逊有礼、心怀大义的道士形象。
- 目标2：解开第二个谜题线索（莽吉柿=南方=朱雀）。
- 目标3：通过幻象铺垫忌炎将军的存在感。

> 执行引导
- 描写重点：
  鉴心行道揖时的端庄姿态；漂泊者脑中幻象带来的战场硝烟味与窒息感。
- 必然触发台词/梗：
  - 鉴心：“小道是共鸣者，自小习武，有一定自保能力，想要出城寻人。”
  - 秧秧：“南边……在四象中，南方对应的是**朱雀**。”

:: 主线事件2：祈池废墟与黑羊寓言 ::

> 事件概况
- 鉴心道别，选择留在伏波营继续寻找失踪者线索，秧秧与漂泊者则循着紫色树叶指引来到死寂的祈池村，秧秧对空气中残留的痛苦情绪感到不适。
- 发现一只瑟缩的小猫，正当秧秧抱起它试图安抚时，空间裂隙突开，将她吞噬带走。
- 伤痕现身，如导游般引导漂泊者搜集村庄线索（许愿树、日记、涂鸦）。
- 揭示真相：村民（白羊）为了私欲献祭了试图救人的少女（黑羊）。伤痕借此邀请漂泊者加入残星会成为“黑羊”。

> 区块目标
- 目标1：营造祈池村压抑恐怖的氛围。
- 目标2：通过伤痕的“导游式”解说，展现其疯狂、乃至有些病态的哲学观。

> 执行引导
- 描写重点：
  祈池村断壁残垣间挂满的湿漉漉纸条；伤痕那漫不经心却暗藏杀机的笑容与夸张的肢体语言。
- 必然触发台词/梗：
  - 伤痕：“难得的会面，不应该有强干扰项，不是吗？……她现在很安全。”
  - 伤痕：“漂泊者，你相信这世上有不付出任何代价就能够实现的愿望吗？”

:: 主线事件3：幻境破局与瑝览之约 ::

> 事件概况
- 漂泊者拒绝加入，伤痕将漂泊者拉入卡牌构筑的异空间幻境进行战斗。
- 激战中，外部空间被一道凌厉的风刃撕裂，秧秧冲入幻境，与漂泊者背靠背并肩作战。
- 神秘红衣女子弗洛洛现身制止伤痕，两人随后消失。
- 回到现实，找到少女日记，解开树叶线索为“朱雀”。返回今州城边庭，将日晷调至“未时、朱雀”，开启机关获得一本“瑝览类书”指引：丑时，玄武。

> 区块目标
- 目标1：展现漂泊者与秧秧之间无需多言的默契与信任（背靠背战斗）。
- 目标2：完成所有信物解谜，推进至最终会面。

> 执行引导
- 描写重点：
  秧秧撕裂空间时风的锐利感；两人背靠背时传递的体温与安心感。
- 必然触发台词/梗：
  - 伤痕：“头摇摇眼眨眨，变出一群羊娃娃……”
  - 秧秧：“我打破了他的幻境……漂泊者，你没事吧？”
  - 弗洛洛：“希望你没有忘记这个条件的前提。不要即兴发挥，给别人带来困扰。”

:: 主线事件4：岁主预言与黑花诱饵 ::

> 事件概况
- 丑时根据完整线索进入边庭瑝览类书库，伤痕再次出现欲动手，被今汐带领卫兵包围捕获。
- 伤痕在被押走前吐露狂言：岁主“角”预言令尹与岁主必有一战。
- 今汐向漂泊者致歉，并传达岁主神谕：瀑雨逆流之时，前往北落野寻找忌炎。同时给予漂泊者一朵黑花照片作为接触“黑海岸”的线索。
- 为了引出一直暗中观察的黑海岸成员，在秧秧等人的建议下，众人找到了植物学家维里奈，利用她的共鸣能力制作了几朵以假乱真的“黑花”，佩戴在身上作为诱饵。
- “椿”的银发少女到来。她正是黑海岸的其中一位“执花”。她对漂泊者表现出了极度的痴迷与危险的占有欲。

> 区块目标
- 目标1：揭示今汐主动布局的智者形象。
- 目标2：引入新势力黑海岸，尤其是椿那带有危险气息的迷恋。

> 执行引导
- 描写重点：
  瑝览类书库宏大的环形数据流光；椿凑近漂泊者颈侧深吸气时的温热呼吸与缭绕香气。
- 必然触发台词/梗：
  - 今汐：“三日为期。看来你我都未失约。”
  - 伤痕：“身为岁主的角，和身为令尹的你，必有一战。”
  - 椿：“你是第一个让我产生这样兴趣的人……哼，你可是我看中的‘种子’。”

:: 主线事件5：稷廷旧事与逆雨降临 ::

> 事件概况
- 炽霞与秧秧找到黑海岸的情报商秋水与安可，他们以黑海岸情报作为交易，委托漂泊者一行人回收信标。
- 众人前往怨鸟泽的稷廷遗址，发现了许多人工“溯洄雨”的实验记录，随后击败一只试图吞噬一切的机械怪物“聚械机偶”，并回收信标。
- 秋水告知漂泊者：他并非第一次苏醒，上一次苏醒地点正是黑海岸。
- 走出遗址时，天空异变，雨水开始违背重力向天空倒流（溯洄雨），秧秧终端传来今州城边防被残象围攻，需要支援的警报。

> 区块目标
- 目标1：揭示漂泊者与黑海岸的深层渊源。
- 目标2：以极具视觉震撼的“逆雨”景象，宣告全面战争的开始。

> 执行引导
- 描写重点：
  稷廷遗址内重力异常带来的失重感；雨滴一颗颗缓缓升向天空的诡异画面。
- 必然触发台词/梗：
  - 秋水：“漂泊者，这并不是你第一次苏醒……上一次苏醒地点正是黑海岸。”
  - 秋水：“不……这场雨……是真正的‘溯洄雨’。”
