:: 主线事件1：浮光林惊变与引日殿的召唤 ::

> 事件概况
- 伴伴叫醒了漂泊者，莫宁告知浮光林爆发虚质磁暴。两人赶到现场，发现大量日灵被污染，阿克塞尔长老劝众人避难。
- 两人深入林区，发现并摧毁了源头装置。此时漂泊者再次收到感召，看到了引日殿入口。年迈牧者出现，传达“织星”预言，邀请漂泊者进入。
- 莫宁经过冷静分析，主动提出分头行动：她去调查物理层面的源头（装置来源与伊摩尔），漂泊者去解决意识层面的问题。漂泊者表示赞同，两人约定终点汇合。

> 区块目标
- 目标1：营造危机迅速升级的紧迫感。
- 目标2：确立“分头行动”的策略，体现莫宁对漂泊者的绝对信任与自我认知的清晰。

> 执行引导
- 描写重点：
    浮光林内狂乱的虚质流动与奄奄一息的日灵。
    莫宁主动提出分开时的眼神，坚定且充满理性，没有丝毫拖泥带水。
- 必然触发台词/梗：
  - 牧者：“若有来自拉海洛之外的来访者……便邀他进入引日殿。”
  - 莫宁：“你去吧。我知道……那里有只有你能做到的事。”
  - 莫宁：“一定要小心。我们在终点汇合。”

:: 主线事件2：织星的重托与日轮重启 ::

> 事件概况
- 漂泊者独自进入引日殿，通过声痕共鸣唤醒了停滞的日轮。身穿黑袍的“织星”现身，解释其余五席已因污染瘫痪。
- “织星”讲述了秘日六席的历史与牺牲，请求漂泊者以“御者”权限授权他一人承担六人的算量。漂泊者询问后果，“织星”表示这是责任而非负担。
- 漂泊者同意授权，光流汇聚，“织星”预见灾厄并共享知识。日轮完全重启，净化了浮光林的污染，无数日灵飞舞。

> 区块目标
- 目标1：揭示罗伊族“秘日六席”悲壮的牺牲精神。
- 目标2：完成高格调的“授权”仪式，展现漂泊者作为关键角色的特殊地位。

> 执行引导
- 描写重点：
    引日殿内古老而神圣的科技感，巨大的停止运转的“日轮”。
    漂泊者伸出手给予授权时，光流汇聚的视觉奇观。
- 必然触发台词/梗：
  - 织星：“请以「御者」的权限，授权我一人承担六人的算量，代行六席之职。”
  - 织星：“我知道的只是，倘若不再有秘日六席解读日轮……生灵将再一次卷入无序的灾厄。”

:: 主线事件3：实验室的真相与反击的激光 ::

> 事件概况
- 莫宁潜入隐蔽实验室，破解终端发现魏尔伦十年前企图引爆炉芯的计划，并联系琳奈准备指控证据。伊摩尔现身欲灭口，并用“残废”羞辱莫宁。
- 莫宁揭穿了魏尔伦的阴谋，伊摩尔震惊。面对伊摩尔的攻击，莫宁冷漠地表示已不再受困于过去，两炮轰晕伊摩尔。
- 漂泊者骑摩托赶到，莫宁迅速跳上后座紧抱住他的腰。莫宁启动共鸣能力构建高速路径，两人驾驶摩托直冲云霄，飞向研究院。

> 区块目标
- 目标1：揭露魏尔伦与伊摩尔的反派嘴脸。
- 目标2：通过莫宁对“残废”一词的霸气回应，完成她的角色高光时刻。

> 执行引导
- 描写重点：
    伊摩尔歇斯底里的嫉妒与莫宁冷静理性的对比。
    莫宁听到侮辱词汇时流露出的释然，以及果断开火的帅气。
    漂泊者骑摩托冲破烟尘接应莫宁的登场，以及莫宁跳上后座的流畅动作。
- 必然触发台词/梗：
  - 伊摩尔：“去死吧，你这个残废！”
  - 莫宁：“自从拥有这双腿以后……这种话，也是好久没听到了。”
  - 莫宁：“抓紧了，我们要飞上去！”

:: 主线事件4：飞越天际与赫利俄斯发射 ::

> 事件概况
- 两人突入研究院控制室，莫宁当众揭穿魏尔伦，并连线琳奈播放“录音证据”（心理博弈）。魏尔伦罪行确凿，瘫软在地。
- 莫宁接管总指挥权，漂泊者在“织星”协助下输入原始数据，“赫利俄斯”启动发射。
- 然而地表突发高浓度虚质，发射受阻。莫宁陷入困境，漂泊者提醒她“带去没有虚质的地方”。莫宁恍然大悟，决定飞向坠落中的炉芯进行近距离校准。

> 区块目标
- 目标1：上演“速度与激情”般的高空突入戏码。
- 目标2：展现莫宁在学术与政治博弈上的压倒性气场，以及琳奈的关键助攻。

> 执行引导
- 描写重点：
    摩托在空中构建光路飞行的视觉冲击。
    魏尔伦面对莫宁摆出的确凿证据时，从傲慢到惊恐的表情变化。
- 必然触发台词/梗：
  - 莫宁：“我也没想到，有一天会用这种方式回到这里。”
  - 琳奈（通讯中）：“魏尔伦教授……关于您的那些交易，我这里可是有‘备份’的哦。”

:: 主线事件5：坠入日核与最后的传输 ::

> 事件概况
- 两人飞向深渊中的炉芯，与保护炉芯的隧群怪物“炉芯机骸”展开激烈战斗。
- 合力战胜炉芯机骸后，一只虚诞虫出现。就在再次搏斗之时，漂泊者被机械虚诞虫抓走拖入深渊，莫宁误以为失去他，悲愤中超频输出。
- 莫宁拼死完成数据传输，低语着“不想说再见”。漂泊者借助深空联合支援，利用深空联合研究人员给予的“TERM-X型原型机”，骑摩托在残骸间跳跃赶回。
- 在莫宁意识即将消散之际，炉芯崩解前，漂泊者抓住了下坠昏迷的莫宁。
- 返回地面，漂泊者背着莫宁走在回学院的路上，两人共同见证了新日“赫利俄斯”的升起，莫宁在感叹太阳真美后陷入昏迷。

> 区块目标
- 目标1：全剧最燃与最虐点。莫宁的绝望爆发与漂泊者的极限救援。
- 目标2：复刻“背负”的名场面，呼应两人的过往羁绊。

> 执行引导
- 描写重点：
    莫宁眼看漂泊者消失时的瞳孔地震，以及超频状态下义肢过热的电流声。
    最后时刻漂泊者抓住莫宁的手，背景是崩解的炉芯与升起的新太阳。
- 必然触发台词/梗：
  - 莫宁（颤抖）：“所以……就由我来迈出这最后一步……！”
  - 莫宁（趴在背上）：“太好了……太阳，真美呀……”

:: 主线事件6：鸭子坐、粉发少女与天台合影 ::

> 事件概况
- 三天后，琳奈与漂泊者在医务室外闲聊，得知琳奈因为“作伪证”被学院处罚，但莫宁提供了关键证据，且在深空联合具有“大记忆恢复术”的共鸣者帮助下，魏尔伦被成功定罪，琳奈也只是回到了心理健康监控名单。
- N.A.N.A.通知莫宁醒来，两人赶去医务室，莫宁试图下床却因双腿无力摔倒，呈现“鸭子坐”姿态，漂泊者伸手扶起。莫宁提到昏迷时见到一位粉发少女（伏笔）。
- 琳奈提议庆祝，三人去天台合影。看见莫宁直接称呼漂泊者为“前辈”，琳奈捂着嘴巴大为震惊，随后见气氛不对找借口离开。
- 漂泊者好奇地问莫宁是不是认识以前的他，莫宁回答“保密”，并对漂泊者解释了关于“告别”的哲学，留下了“当什么巨大的转折出现（比如鸣式危机被解除），再告诉漂泊者”的约定。
- 幕后，陆·赫斯审问安瑟姆，神秘人感叹棋子无用，期待与漂泊者的相遇。

> 区块目标
- 目标1：通过“鸭子坐”发糖，缓解之前的紧张感。
- 目标2：埋下后续剧情伏笔（粉发少女）。
- 目标3：以温馨且充满希望的合影结束本章。

> 执行引导
- 描写重点：
    关键细节：莫宁摔倒时双腿呈W形的“鸭子坐”姿势，抬头看漂泊者时有些委屈又撒娇的眼神。
    天台上，莫宁不想直说“再见”时的含蓄与深情。
    三人合影定格画面。
- 必然触发台词/梗：
  - 莫宁：“那个少女……有着粉色的头发，戴着浅蓝色的圆环。”
  - 莫宁：“只有告别才能有延续和改变。在我能亲口告诉你那个答案之前，就请你再稍微保持一下这份好奇吧。”
