:: 主线事件1：棋局传说与谜语之约 ::

> 事件概况
- 你在虹镇街头听一位老人说书，他讲述着樵夫入乘霄山观仙人对弈，转瞬百年物是人非的传说。
- 当听众质疑故事的真实性时，老人则神秘地表示，这与山中真正的“长生秘法”有关。
- 此时，你察觉到一道目光，人群边缘，一位气质清雅的女子（长离）正静静注视着你。
- 她对你比出“嘘”的手势，留下一句低语“……再会”，身影便悄然隐入阴影，只留下一张信笺飘落至你脚边。
- 阿布被惊醒，念出信笺上的谜语：“纵览虹景，环迹望飞檐处，雪影成溪时，静候君归。”

> 区块目标
- 引入贯穿剧情的核心意象：“棋局”与“时间扭曲”。
- 塑造长离神秘、难以捉摸的形象，并主动与漂泊者建立联系。
- 给予漂泊者一个明确的短期目标：解开信笺上的谜题。

> 执行引导
- 描写重点：
	视觉镜头：镜头从热闹的说书场缓缓扫过，最终定格在与喧嚣格格不入的长离身上。她消失时如雾气般散去的特效。
	互动细节：长离落在你身上的精准目光，以及信笺从她指尖轻盈滑落的动态。
- 必然触发台词/梗：
	- 老人：“——他面前只是一个破败的棋盘，和一副腐朽的骸骨罢了。”
	- 老人：“这就不得不提到另一个乘霄山的传说了……隐藏着真正的「长生秘法」！”
	- 长离：“嘘。”
	- 长离：“……再会。”
	- 阿布：“完全看不懂！”
- 伏笔/暗线：
	说书人提到的“烂柯百年”传说，是整个故事的核心引子，暗示乘霄山内存在真实的时间异常区域，为后续事件提供背景。

:: 主线事件2：寻访虹镇的众说纷纭 ::

> 事件概况
- 为解开谜题，你开始在虹镇四处询问。
- 从居民启弘处，你了解到“雪影成溪”是本地传说，指代未时（下午1-3点）。
- 紧接着，一个神色紧张的青年警告你远离长离，称她为“心思深不可测”、“玩弄人心的佞臣”，并提及她曾“将今州贵客骗进乘霄山”的传闻。当你揭示自己就是那个“贵客”后，他立刻惊慌失措地逃走了。
- 最后，你遇到长离的崇拜者、先行公约的攸宁。她将长离描绘成“为天下太平自请下放”的无私英雄，并为你指明谜题的地点线索：虹镇东北边最高的建筑。

> 区块目标
- 解决谜题中的时间与地点信息。
- 通过截然相反的评价，立体地塑造长离充满争议的公众形象。
- 制造一个戏剧性的反转（揭示“贵客”身份），增加剧情的趣味性。

> 执行引导
- 描写重点：
	视觉镜头：可疑青年惊弓之鸟般的神态，与攸宁谈及长离时眼中闪烁的崇拜光芒。
	互动细节：青年逃跑时慌乱的脚步和尴尬恐惧的表情。
- 必然触发台词/梗：
	- 可疑的年轻人：“奉劝你离她远点，那人心思深不可测，是个不折不扣的佞臣。”
	- 可疑的年轻人：“我看啊，她就是怕那人与令尹太熟络坏了她的事……真是其心可诛。”
	- 攸宁：“她是玄渺真人的亲传弟子……一人之下万人之上！”
	- 攸宁：“这种无私之举，如何不让人敬仰？”
- 伏笔/暗线：
	对长离的两极评价，暗示她的行事风格不为常人所理解，其布局深远，往往会招致误解。

:: 主线事件3：凉亭相会与共同之约 ::

> 事件概况
- 在前往约定地点的路上，你偶遇了辛夷。她评价长离“一贯九曲心肠，布局晦涩难辨”，但相信长离的行为必有深意。
- 你来到虹镇最高的建筑，根据指引确定了最终的目的地是远处山石上的一座凉亭。
- 未时抵达凉亭，长离已在此等候。她为这种“弯弯绕”的方式致歉，坦言是出于“落子能得知己相应”的私心。
- 她讲述了另一个版本的传说：百年前，她的师父与一位异人（过去的你）在山中对弈。并透露师父的残谱中留有“此局或解时流之乱”的关键信息。
- 长离推测这或许与治愈岁主损伤有关，并正式请求你与她一同进山，寻找这个传说中的棋局。

> 区块目标
- 借由辛夷的口，提供一个中立且深刻的视角来解读长离。
- 让漂泊者与长离正式会面，得知更接近真相的传说版本。
- 正式发布主线任务：寻找与“时流之乱”和岁主恢复相关的棋局。

> 执行引导
- 描写重点：
	视觉镜头：从高处俯瞰，环形遗迹与远处凉亭构成的风景。凉亭中，长离在午后微风中拂起发丝的恬静模样。
	互动细节：长离说出“私心”时微微放缓的语速和移开的目光。
- 必然触发台词/梗：
	- 辛夷：“她若是想做什么，绝对不会像表面上看上去那么简单。”
	- 长离：“落子能得知己相应，实在是一件欣喜之事。”
	- 长离：“此局或解时流之乱。”
	- 长离：“所以……这只是我私人的请求。”
- 伏笔/暗线：
	长离的版本中，异人（你）与修行者（她师父）是旧识。这为你与这个故事的深层联系埋下了第一个明确的伏笔。

:: 主线事件4：山中遇险与救援梧黎 ::

> 事件概况
- 你答应请求，与长离一同深入乘霄山。途中她作为“引路人”，对山中捷径十分熟悉，并感慨有你同行令人心安。
- 突然，阿布察觉到前方山洞里混杂着残象、人类以及一种混乱的频率气息。
- 你们立刻前往查看，在洞中发现探险家梧黎正被残象围攻，已是筋疲力尽。
- 你们即刻投入战斗，长离上前护住梧黎，为你创造攻击机会，两人默契配合消灭了所有残象。

> 区块目标
- 通过并肩同行，初步拉近漂泊者与长离的关系。
- 引入新的悬念：本应安全的乘霄山为何会出现残象和遇险者。
- 展现长离在战斗中的冷静可靠，以及与漂泊者形成的初步默契。

> 执行引导
- 描写重点：
	视觉镜头：乘霄山中白雪皑皑的风景。山洞内昏暗光线和激烈的战斗场面。
	互动细节：长离将探险家护在身后的动作，以及她投向你那默契而笃定的眼神。
- 必然触发台词/梗：
	- 长离：“或许也是因为……同行的旅伴令人心安。”
	- 阿布：“不对……还有更奇怪的，有残象，人类，还有……这种频率混乱的味道……”
	- 长离：“请退后，交给我们吧。”
- 伏笔/暗线：
	山中反常出现的残象与“混乱的频率”，暗示此地存在着非自然的能量扰动，很可能就是你们要找的“时流之乱”。

:: 主线事件5：归镇寻亲与时流之谜 ::

> 事件概况
- 获救的梧黎在昏迷前念叨着“终于找到了”和“伏翎在等我”。你们决定先将他带回虹镇。
- 经人指引，你们在旅馆见到了梧黎的恋人，病体沉重的伏翎。
- 在与伏翎的交谈中，核心谜底揭晓：为了寻找传说中“时间永恒停滞”的世外桃源来拯救伏翎，梧黎进入了乘霄山。他在山中仅度过了半日，而伏翎却已在虹镇苦等了七天。

> 区块目标
- 揭示“山中半日，世上七天”的核心悬案，将“时间乱流”这一概念具象化。
- 阐明梧黎的动机，使其形象从鲁莽的探险家转变为为爱痴狂的悲情角色。
- 引出“长生秘法”的第三个版本：一个能让时间停滞的世外桃源。

> 执行引导
- 描写重点：
	视觉镜头：伏翎因病弱而苍白的脸色，与梧黎短暂转醒后兴奋的神情形成对比。
	互动细节：梧黎虚弱地伸出手想触碰伏翎，与得到回应后满足地昏睡的表情。
- 必然触发台词/梗：
	- 梧黎：“我找到了！我找到了能真正延缓你病的办法，我们以后……可以一直在一起！”
	- 伏翎：“我已经……在这里等了他快七日了。”
	- 伏翎：“梧黎说……只要在那里生活，我的病就永远也不会恶化。”
- 伏笔/暗线：
	梧黎与伏翎的故事，实际上是长离自身命运的一种预演，为后续长离对生死的选择和感悟埋下情感基调。

:: 主线事件6：夜归军帐下的疑案分析 ::

> 事件概况
- 伏翎得知真相后，并未沉溺于虚假的希望，反而清醒地请求你们务必解决那个危险的时间乱流。
- 你们立刻向辛夷汇报了情况。在夜归营帐中，三人一同分析案情。
- 长离指出，梧黎对传说的了解和找到乱流的速度都极不正常，断定他是在残星会的刻意引导下，找到了一个被隐藏的乱流。
- 结论浮现：残星会的目标是棋局传说所指向的“时序之能”，梧黎只是他们引你们入局的棋子。
- 辛夷随即下令封锁乘霄山，并将驻军调度权限交予长离。长离决定返回山洞查探，你选择与她同行，主动破局。

> 区块目标
- 正式确认幕后黑手为残星会，并揭示其阴谋。
- 展现伏翎的坚强与清醒，深化其角色形象。
- 巩固你与长离、辛夷的“铁三角”合作关系。

> 执行引导
- 描写重点：
	视觉镜头：伏翎说出“梦总有苏醒的时候”时坚定的眼神。三人围在地图前严肃分析案情的紧张氛围。
	互动细节：长离分析时逻辑清晰、一针见血的锐利感。
- 必然触发台词/梗：
	- 伏翎：“梦总有苏醒的时候，重要的是醒来之后的人应该继续前进。”
	- 长离：“最难以辨别的‘谎言’往往是隐瞒部分事实的真相。”
	- 长离：“看来我们的‘老朋友’还是没有死心。”
	- 长离：“费尽心思引我们至此，若是不应，倒是显得我们有些不知趣了。”
