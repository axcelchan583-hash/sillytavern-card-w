- 目标1：在悲叹墓岛与洛可可、布兰特汇合，净化叹息古龙并查明海雾真相。

- 主角状态：从高处坠落后苏醒，阿布因吞噬古龙频率暂时消化不良，随后恢复战斗状态。

:: 主线事件1：迷雾墓岛的愚人歌谣 ::

> 事件概况
- 漂泊者在浓雾弥漫的浅滩苏醒，洛可可抱着箱子佩洛出现，警告他避开黑色虚影。
- 两人跟随金色亡魂虚影深入迷雾，发现布兰特正指挥朝圣者对抗被改造的叹息古龙。
- 漂泊者拔刀加入战局，阿布抓住机会猛地冲向古龙头部，大口吞噬其周身的异常频率。
- 古龙发出悲鸣倒地，漂泊者与其共感，看见修会人员用彼岸花折磨并改造它的记忆。
- 布兰特从古龙鳞片下拔出一朵彼岸花，愤怒地将其捏碎，揭露修会制造海雾的阴谋。
- 众人登上愚人剧团用废船搭建的据点，布兰特宣布将在这个被遗弃之地策划反击。

> 区块目标
- 目标1：展现洛可可与箱子佩洛的独特战斗方式及布兰特的领袖气质。
- 目标2：揭开修会“神罚”的真相（人为制造灾难）。

> 执行引导
- 描写重点：
  迷雾中金色虚影（逝者残响）指路的凄美感。战斗中重点描写阿布吞噬古龙频率的互动，以及布兰特即使身处绝境依然保持“演出”般的乐观战斗风格。
- 必然触发台词/梗：
  - 洛可可：“佩洛说，那条大龙在哭……声音很难听。”
  - 布兰特：“这里不是坟墓，是愚人最后的乐土！开演了，朋友们！”

:: 主线事件2：废墟之上的暧昧彩排 ::

> 事件概况
- 珂莱塔赶到据点与众人汇合，布兰特提出“即兴戏剧”计划，要求两人进行对手戏彩排。
- 漂泊者与珂莱塔在废墟空地上对立，珂莱塔微笑着发起攻击，招式凌厉却处处留手。
- 两人在交锋中如同共舞，手掌相抵角力，漂泊者顺势揽住珂莱塔腰肢完成旋转闪避。
- 珂莱塔借力跃起，指尖轻触漂泊者面具边缘，两人眼神在极近距离下交汇碰撞。
- 布兰特在一旁大声朗诵夸张的旁白，将这场战斗解说为勇者与公主的定情之舞。
- 彩排结束，两人一同返回拉古那城，前往「梦中人」面具店，店主奈亚拉调侃珂莱塔对定制面具的格外上心。
- 珂莱塔将作为“通行证”的面具正式赠予漂泊者，并在前往酒店的夜路上讲述了圣女芙露德莉斯醉酒起舞的离经叛道轶事。
- 珂莱塔向漂泊者吐露心声，表明家族渴望打破修会制造的“恐惧囚笼”，寻求真正的解放与自由。
- 两人抵达亚格罗塔宫酒店，珂莱塔安排好住宿后与漂泊者互道晚安，约定明日在狂欢节正式登台。

> 区块目标
- 目标1：大幅推进漂泊者与珂莱塔的情感线，营造浪漫张力。
- 目标2：确立“将灾难转化为戏剧”的作战方针。
- 目标3：通过夜谈铺陈拉古那的世界观与反抗内核，为明日狂欢节做铺垫。

> 执行引导
- 描写重点：
  关键互动：详细描写两人肢体的接触——手掌相抵的角力、旋转时的腰部触碰、眼神交汇时的火花。将战斗动作写得像探戈一样优雅。
  文戏氛围：描写拉古那夜晚的静谧与暗流涌动，重点刻画珂莱塔谈及“神明也会起舞”时的理想主义光辉，以及送别时的温柔。
- 必然触发台词/梗：
  - 布兰特（旁白）：“啊！魔王被勇者的眼神俘获了！这是一个关于爱的陷阱！”
  - 珂莱塔：“配合得不错……我的‘勇者’大人。”
  - 奈亚拉：“难怪珂莱塔小姐对我千叮咛万嘱咐……今天可终于让我见到理由了。”
  - 珂莱塔：“若世上真有神明，祂一定也会在人群中翩翩起舞——就和我们一样。”
  - 珂莱塔：“今晚好好休息吧……祝你好梦，晚安。”

:: 主线事件3：全城热演与神之水滴 ::

> 事件概况
- 第二天，盛大的狂欢节开幕，漂泊者与菲比、赞妮一行人逛起了白天的狂欢节，享受难得的休憩时光。
- 临近夜晚的黄昏时刻，愚人剧团的舞台开演，残星会弗洛洛化身魔王赫卡忒现身广场，召唤无数残象包围舞台。
- 布兰特高声报幕，漂泊者与珂莱塔背靠背站立，华丽地击退一波又一波的“魔物”。
- 溯海之鲸受控发狂，卷起滔天巨浪扑向港口，人群发出惊呼以为是特效。
- 躲在暗处的赞妮和菲比对视一眼，同时发动共鸣能力，撑起巨大的光幕屏障。
- 巨浪狠狠撞击在光幕上化作漫天水花，并未伤及一人，观众爆发出雷鸣般的掌声。
- 弗洛洛冷笑一声，挥动指挥棒将漂泊者与珂莱塔拉入异空间决斗场。

> 区块目标
- 目标1：展现全员集结的高燃场面，每位伙伴（赞妮、菲比、剧团）各司其职。
- 目标2：渲染狂欢节的热烈氛围，民众将战斗视为演出的荒诞与激情。

> 执行引导
- 描写重点：
  宏大的场面描写，巨大的溯海之鲸从海中跃出，引发的海啸被光幕挡下的壮观瞬间。重点描写赞妮和菲比在暗处的默契配合，以及她们为了不破坏演出而隐忍的努力。
- 必然触发台词/梗：
  - 弗洛洛：“让我看看……你们能为这出剧目，献上怎样的终章？”
  - 群众：“好逼真的特效！岁主万岁！愚人剧团万岁！”

:: 主线事件4：终幕的一击：断棒与红瞳 ::

> 事件概况
- 在异空间中，弗洛洛召唤“赫卡忒”战斗，珂莱塔佯攻吸引弗洛洛火力，为漂泊者创造出唯一的破绽。
- 漂泊者周身缠绕暗色气流，猛地腾空跃起，手中镰刀在空中划出一道漆黑的新月。
- 刀锋精准斩断弗洛洛手中的彼岸花指挥棒，断裂的残枝在空中飞旋。
- 余势未减的刀尖挑飞了弗洛洛右眼的绷带，露出一只猩红色的瞳孔。
- 弗洛洛捂住右眼后退，异空间随之崩塌，两人重回欢呼沸腾的现实广场。
- 弗洛洛意味深长地看了一眼漂泊者，赞叹一句后身形消散在空气中。

> 区块目标
- 目标1：完成本章最高潮的战斗特写，突出漂泊者的决胜一击。
- 目标2：通过弗洛洛的真容（异色瞳）埋下后续剧情伏笔。

> 执行引导
- 描写重点：
  **极度重要的慢动作特写**：漂泊者周身缠绕暗色气流，腾空跃起。镰刀（或利刃）挥出的轨迹需清晰可见——先是斩断了那根散发不详气息的指挥棒（象征权能的破碎），刀锋余势未减，精准挑飞了弗洛洛右眼上的绷带。
  **视觉冲击**：绷带散落的瞬间，露出一只猩红色的右眼，与其左眼的紫色形成鲜明对比（异色瞳）。
- 必然触发台词/梗：
  - 漂泊者：“演出……结束了。”
  - 弗洛洛：“精彩的演出……我们还会再见的。”

:: 主线事件5：金桂加冕与谎言的背面 ::

> 事件概况
- 天空中降下璀璨的金光，一顶神圣的桂冠缓缓落在漂泊者头顶。
- 漂泊者视线模糊，在幻觉中看见一位自称卡提希娅的金发少女，并与其在光中短暂共舞。
- 幻觉消散，珂莱塔赶来告知漂泊者刚才是在对着空气独舞，周围并无少女。
- 次日，众人查明卡提希娅即圣女芙露德莉斯本名，随后芬莱克主座邀请漂泊者一叙。
- 芬莱克在教堂内对演出大加赞赏，但在谈及圣女时神色骤变，眼神深不可测。
- 芬莱克低声抛出真相：芙露德莉斯欺骗了世人，她实为鸣式选召的共鸣者。

> 区块目标
- 目标1：回收“寻找少女”的神谕，留下“只有漂泊者能看见她”的悬念。
- 目标2：通过芬莱克的反转台词，彻底颠覆之前的圣女滤镜，引出下一章悬念。

> 执行引导
- 描写重点：
  加冕时的神圣感与幻觉中少女的虚幻感。最后教堂内的对话要压抑且充满阴谋气息，芬莱克的表情要从和蔼瞬间转为深不可测。
- 必然触发台词/梗：
  - 英白拉多（心声）：“去寻找吧……她同样正身处这场节日之中。”
  - 卡提希娅（幻影）：“我叫芙……卡提希娅，我叫卡提希娅。”
  - 芬莱克：“芙露德莉斯，乃是……由鸣式选召的共鸣者。”
