:: 主线事件1：虚弱的课堂与迟到的幽灵 ::

> 事件概况
- 漂泊者强忍着“存在被世界剥离”的剧烈眩晕感步入教室，洛瑟菈校长正优雅地讲述罗伊人神话中“炉芯纪”太阳化身巴德尔封印赫龙格的故事。
- 助教娜波摩被点名讲述隧者对抗鸣式的历史，她结结巴巴地提到“隧者一度力竭，但炉芯忽然发出炽盛的光”。
- 爱弥斯（幽灵形态）充满活力地打断叙述，一边说着“世界得到拯救，可喜可贺”，一边径直穿过实体课桌，坐在了漂泊者身旁。
- 爱弥斯向漂泊者搭话，调侃他是“同桌”，并嘲笑学院效率低还没发校服，漂泊者惊讶地认出她是之前在虚质空间见过的少女。
- 当漂泊者感谢她在“鸣式巨眼”前救了自己时，爱弥斯露出了真实且困惑的表情，表示完全不记得此事（关键伏笔：此时她尚未经历未来的时间闭环）。
- 爱弥斯感应到异样（辛吉勒姆的靠近），神色突变，假装向老师请假后匆匆离去，只留下漂泊者对着空气说话。
- 课后，洛瑟菈将漂泊者留下，意味深长地指出爱弥斯是十几年前在“虚境作训场”失踪的适格者，暗示漂泊者看见了“幽灵”，并怂恿他去探查真相。

> 区块目标
- 目标1：通过漂泊者的生理不适（重影、耳鸣）强调频率流失的紧迫感。
- 目标2：确立爱弥斯“看不见、摸不着”的幽灵特性，以及她开朗跳脱的性格。
- 目标3：埋下时间循环伏笔——确认此时的爱弥斯确实不知道自己救过漂泊者。

> 执行引导
- 描写重点：
    视觉镜头：全息教案上古老的图腾；爱弥斯半透明的身影与实体桌椅重叠的违和感；洛瑟菈仿佛看穿一切的眼神。
    互动细节：漂泊者按住胸口极力维持清醒的动作；爱弥斯凑近时，漂泊者下意识闪避却发现对方没有实体的细节。

- 必然触发台词/梗：
    - 娜波摩：“但千钧一发时，不知为何，它的炉芯忽然又发出了炽盛的光——”
    - 爱弥斯：“——而后隧者举起剑，封印鸣式。未来的拉海洛幸存于世，世界得到拯救，可喜可贺！”
    - 爱弥斯：“是吗？抱歉哦，我不记得了。”
    - 爱弥斯：“从现在起修满四个学部的学分可不容易——欢迎来到学、业、泥、沼。”
    - 洛瑟菈：“看见一个失踪多年，基本被认定死亡的人，原因不明地停驻在学院中……这是你的癔症，还是背后另有隐情？”

:: 主线事件2：作训场的核心与被监听的通讯 ::

> 事件概况
- 漂泊者跟随发光的日灵来到尘封已久的虚境作训场，这里只有老旧设备闪烁微光。
- 靠近那台爱弥斯曾失踪的驾驶舱时，漂泊者频率流失骤然加剧，世界出现严重的重影与撕裂感。
- 一个机甲形态的怪物前来攻击漂泊者，爱弥斯现身，用自己的刀剑阻挡怪物袭击，随后怪物在不敌爱弥斯后飞行离开。
- 爱弥斯指出怪物的名字叫“辛吉勒姆”，且告知漂泊者身上有“隧者炉芯的核心”，这是他在与莫宁共同完成的换日仪式中得到的，这枚核心现在相当于她的心脏，也是漂泊者能看见她、触碰到她的媒介。
- 爱弥斯解释自己十多年前为封印鸣式超频而死，现在作为“电子幽灵”依附核心存在，但力量正被名为“辛吉勒姆”的机兵夺取。
- 爱弥斯展示手掌边缘如数据般崩解的特效，请求漂泊者携带核心走过她生前留痕之地找回“存在”，以击败辛吉勒姆。
- 从爱弥斯复杂的表情和无法自圆其说的部分信息，敏锐察觉到爱弥斯有所隐瞒，但还是选择相信她并与她同行。
- 漂泊者联系莫宁询问并汇报隧者核心的事宜，通讯却被一阵杂音打断，洛瑟菈接入频道。在漂泊者的质问下，她承认自己在监听并代表“深空联合”介入。
- 洛瑟菈以“麻烦的大人”自居，虽行为冒犯但给予了漂泊者最高行动权限（联运椎骨、兵装库），要求他归还核心以唤醒隧者对抗阿列夫一。

> 区块目标
- 目标1：解释“炉芯核心”作为连接生死的媒介设定。
- 目标2：展示爱弥斯身体“数据化崩解”的异常状态，引出危机感。
- 目标3：刻画洛瑟菈作为深空联合高层的压迫感与掌控欲。

> 执行引导
- 描写重点：
    视觉镜头：作训场昏暗灯光下爱弥斯发光的手掌边缘；漂泊者视角的眩晕特效（色块分离）；通讯屏幕上洛瑟菈戏谑的笑容。
    互动细节：爱弥斯指尖轻触漂泊者胸口核心位置的动作；漂泊者对监听行为表现出的冷硬态度。

- 必然触发台词/梗：
    - 爱弥斯：“我是它的共鸣者。只不过……早就死掉，变成电子幽灵了~”
    - 爱弥斯：“拯救什么这种事，本来就很酷，很美好啊。”
    - 洛瑟菈：“呵，真没想到，会以这种方式再次听到这个名字……”
    - 洛瑟菈：“‘我们’，深空联合。你就当作是在这所青春洋溢的学院里，那些‘麻烦的大人’吧。”
    - 洛瑟菈：“就算只是想和我谈谈心——我也随时欢迎哦。”

:: 主线事件3：未完成的歌与冰原公路片 ::

> 事件概况
- 前往礼堂，门口学生热情塞来一张海报，兴奋地讨论着即将举办的传说歌手“飞行雪绒”的歌友会。
- 漂泊者看着海报上梦幻的雪花飞鸟，身旁的爱弥斯作为本尊却无人能见，只能看着自己的海报感慨。
- 进入礼堂，爱弥斯习惯性地向路过却看不见她的学生挥手，解释这是为了不忘却自己人类身份的“自我提醒”。
- 爱弥斯承认自己就是“飞行雪绒”，听着礼堂播放的那首未完成的歌，遗憾“小小奇迹”变成了遗作。
- 两人搭乘巨大的联运椎骨电梯上升至冰原地表，爱弥斯回忆小时候也曾这样偷偷溜去冰原玩耍。
- 刚出舱门，遇到校医陆·赫斯。陆医生虽看不见爱弥斯，却敏锐地察觉到漂泊者在与人同行，并帮忙改装了摩托以适应冰原环境。
- 陆·赫斯看着冰原，讲述这里曾是密林的历史，以及天空海异变后人类被“宇宙拒绝”的现状。
- 漂泊者骑上摩托，载着爱弥斯在风雪交加的冰原上疾驰，爱弥斯兴奋地张开双臂，称这为“公路片”。

> 区块目标
- 目标1：揭示爱弥斯“飞行雪绒”的身份，展现她对生活的热爱与遗憾。
- 目标2：通过陆·赫斯的台词铺垫“天空海”与“被封锁的星空”这一宏大背景。
- 目标3：营造“只有你能看见我”的独特浪漫感与公路旅途的羁绊。

> 执行引导
- 描写重点：
    视觉镜头：礼堂喧闹人群与独自挥手的透明少女；冰原上矗立的巨型古代机械残骸；摩托在雪原留下的长长辙印。
    互动细节：爱弥斯在摩托后座（或飘浮旁侧）虚抱住漂泊者的动作；陆医生递给漂泊者工具时的默契。

- 必然触发台词/梗：
    - 学生：“她不露脸就是不想和粉丝牵扯太深吧，喜欢歌就好咯。”
    - 爱弥斯：“如果不这样做，就容易忘掉身为人类的立场。这算是一种……自我提醒。”
    - 爱弥斯：“早知道就在那次训练前先写完发出去了，一不小心就变成遗作了呢，哈哈。”
    - 陆·赫斯：“自那之后，宇宙就拒绝了我们。”
    - 爱弥斯：“偷偷告诉你，幽灵是感觉不到温度的。”

:: 主线事件4：天槎空间站与像素大冒险 ::

> 事件概况
- 在冰壁下偶遇正在挖掘“坠落物”的娜波摩，她对太空照片表现出极度的狂热。
- 地面突然震动，冰壁崩塌。漂泊者迅速拉开娜波摩，巨大的落石穿透了爱弥斯的灵体，再次强调了她“已死”的事实。
- 退路被封，陆·赫斯提议进入附近的“天槎空间站”遗迹等待救援，漂泊者使用权限开启了“模块架构器”稳定频率入口。
- 进入空间站，在陆·赫斯操纵设备后，全息导航员文露介绍“天空海液滴”是通往星空的介质。俊熹与未楹随后登场，可当未楹画面出现后，娜波摩因为误触意外关掉了画面（伏笔：娜波摩刻意隐瞒了未楹的研究内容）。
- 漂泊者询问陆·赫斯为什么认识稷庭文字还能操作其面板，赫斯表明这里人多眼杂，想听的话可以之后来医务室单独聊。
- 前路被虚质方块阻挡，爱弥斯骇入系统，在漂泊者的AR视野中生成了像素风格的界面，播放起游戏风格音乐。
- 触发“拉海洛方块”小游戏，爱弥斯化身像素小人与漂泊者一起消除方块，解决了阻挡在他们路途中的虚质方块。
- 爱弥斯发现系统曾被宇航员克莱门特在坠落后短暂激活，指令却是绝望的“封锁舱门”。

> 区块目标
- 目标1：通过娜波摩的狂热侧面描写人类对星空的渴望。
- 目标2：利用“像素游戏”这一视觉奇观，展现爱弥斯作为电子幽灵的黑客能力与童心。
- 目标3：通过克莱门特的记录渲染空间站内绝望的氛围。

> 执行引导
- 描写重点：
    视觉镜头：空间站内漂浮的尘埃与虚质颗粒；漂泊者眼中叠加的复古游戏UI、爱弥斯的像素Q版头像；娜波摩惊魂未定的表情。
    互动细节：漂泊者对着虚空认真比划“石头剪刀布”；爱弥斯在虚拟屏幕上做出的胜利动作。

- 必然触发台词/梗：
    - 爱弥斯：“哎呀，我没事~毕竟是幽灵嘛，不会被砸到的。”
    - 爱弥斯：“瞧，‘拉海洛方块’拼好了！《太空战士卡佳》系列里每次都会附赠这个小游戏。”
    - 爱弥斯：“你会玩石头剪刀布吗？现在出拳吧！哇赢了！你的操作真棒！”
    - 陆·赫斯：“这支笔上的字符也是稷庭文字……说起来也算和你有些关联。”

:: 主线事件5：无铭探索者与被截断的影像 ::

> 事件概况
- 娜波摩因高浓度虚质，手臂皮肤出现类似烧伤的痕迹，爱弥斯为了保护她化作流光钻入控制台稳定系统。
- 穿着宇航服的人形怪物“无铭探索者”（克莱门特异化）从阴影中走出，发出“离开这里”的警告，并发起攻击，击飞了正在沉迷于研究的娜波摩。
- 漂泊者挥剑迎击，在交锋中震惊地发现，这具机兵的招式频率竟与自己产生共鸣（伏笔：同源力量）。
- 击败机兵后，漂泊者操作控制台，屏幕上自动播放了一段留存的影像。
- 影像内容：一个黑发身影（过去的漂泊者）正穿透倒悬的“天空海”液体层，从“另一侧”向索拉里斯坠落。
- 漂泊者对自己“从天而降”的身世感到震惊。刚刚退出系统的爱弥斯看着影像，表情变得晦暗不明（伏笔：她知道漂泊者被故乡拒绝的真相），拳头捏紧充满了愤怒情绪，却假装什么都没发生，转过头对漂泊者笑了笑搪塞了过去。
- 陆·赫斯迅速切断影像播放，警告漂泊者不要让这段记录流出，并称呼漂泊者为“黑海岸的领导者”，表明自己知道漂泊者身份的真相。
- 陆·赫斯决定留下照顾受伤的娜波摩，让漂泊者先行离开，前往下一个核心地点。

> 区块目标
- 目标1：通过战斗展示漂泊者的战力以及与“无铭者”的共鸣。
- 目标2：揭示核心悬念：漂泊者是“天外来客”，且爱弥斯对此知情并感到难过。
- 目标3：借陆·赫斯之口点破漂泊者身份，结束Part 1，为后续情感爆发做铺垫。

> 执行引导
- 描写重点：
    视觉镜头：无铭探索者破败躯壳下闪烁的核心；全息影像中倒悬的黑色海洋与坠落的光点；爱弥斯瞬间黯淡又强颜欢笑的眼神。
    互动细节：战斗中剑刃相抵时的频率震颤；陆·赫斯切断画面时的果决动作。

- 必然触发台词/梗：
    - 无铭探索者：“……我们……自星海坠落……危险……离……开……这里……”
    - 漂泊者：（那条指令也是为了不让已经异化、逐渐丧失理智的自己到达上层舱段，伤害到其他人吧。）
    - 陆·赫斯：“这段录像，你最好还是以自己的记忆保存——就别留给星炬学院之后来的人了。”
    - 陆·赫斯：“你说呢？黑海岸的……领导者。”
    - 爱弥斯：“没事的，漂泊者。”

:: 主线事件6：废墟中的幻影与被遗忘的存在 ::

> 事件概况
- 两人骑行至一处被风雪严重掩埋的废墟。爱弥斯收起路上的玩笑，严肃指出这里是她曾工作的科考站，因遭遇突发高浓度虚质磁暴而全员覆没。
- 爱弥斯向漂泊者解释残酷的“存在”消逝设定：被磁暴吞噬不仅是死亡，连“曾经存在过”的信息和“被遗忘”这件事本身都会消失。
- 应爱弥斯请求，漂泊者激活身上的炉芯核心。空间泛起金色的涟漪，原本死寂的废墟中逐渐浮现出旧日研究员与罗伊人的半透明身影。
- 幻影们栩栩如生，谈论着未完成的学术论文、哪家的炖锅好吃，并在对话中夸赞爱弥斯的才华，鼓励她不要因为磁暴的困难而难过。
- 重点描写爱弥斯的表情：她看着这些只有她还记得的人，露出了一种“释然却令人心碎的笑”，眼底带着湿润。
- 爱弥斯解释：变成电子幽灵的好处就是不再受虚质影响，就算整个世界都遗忘了他们，她也能永远记得大家。
- 漂泊者看着她的笑容感到心疼，说道“一直笑会很累”，爱弥斯却反驳这是她决定的快乐生活方式。
- 经过这次交心，两人距离拉得更近了。在爱弥斯的指引下，她再次坐上了摩托的后座，两人出发前往渐湖。

> 区块目标
- 目标1：通过幻影的温馨与现实的虚无，渲染“存在被吞噬”的残酷。
