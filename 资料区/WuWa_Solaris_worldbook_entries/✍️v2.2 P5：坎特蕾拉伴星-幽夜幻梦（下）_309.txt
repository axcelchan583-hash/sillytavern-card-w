:: 主线事件1：剧毒花园的指引 ::

> 事件概况
- 进入波蒂维诺堡后，坎特蕾拉为漂泊者准备了一杯特调饮品，漂泊者喝下后不适感大为减轻。
- 坎特蕾拉以照料花朵为由暂时离开。
- 漂泊者在花园中发现一位头戴刺眼红帽的失语少女，并意识到她与“幽灵少女”有关。
- 漂泊者跟随少女的指引，先后见到了形似苹果的剧毒植物“海苹果”和花冠上长着剧毒水母、形似礼帽的“瑰红水母花”，瞬间明白了谜面「红苹果」与「圆礼帽」的真正含义。
- 漂泊者找到正在曼妙起舞（实则压制频率）的坎特蕾拉，直接对质。
- 坎特蕾拉坦然承认了两种植物的剧毒性，并确认漂泊者的猜想：“幽灵少女”的谜底，正是翡萨烈家族残酷的“圣女试炼”。

> 区块目标
- 目标1：通过失语少女的引导，揭开三个谜面中后两个的真相，将线索指向“剧毒”。
- 目标2：将所有线索串联，引出核心秘密“圣女试炼”。
- 目标3：完成漂泊者与坎特蕾拉之间的信息对等，为后续建立真正的信任关系铺路。

> 执行引导
- 描写重点：
  	视觉镜头：特调饮品中复杂的色彩与质地；失语少女怯生生的眼神和刺眼的红帽；两种剧毒植物诡异而美丽的外形。
    互动细节：饮品入口后不适感褪去的舒缓感；跟随少女在花园中穿行的感觉；与坎特蕾拉对质时的紧张气氛。
- 必然触发台词/梗：
  - 坎特蕾拉：“剧毒的花朵们互相倾轧，汁液互相交融...请放心，它美味又安全。”
  - 坎特蕾拉：“红色的海苹果，也叫作美丽的死神...”
  - 坎特蕾拉：“没想到你已经推理到了这个地步...「幽灵少女」的谜底，即是「圣女试炼」的真相。”
  - 坎特蕾拉：“如果，我是说如果，你发现那疮疤已经爬上我的躯体...也悉听尊便。”
- 伏笔/暗线：
  - 失语少女的身份成谜，她既是“幽灵少女A”，又不是寻常的幽灵，她引导漂泊者揭示真相的行为暗示她与坎特蕾拉之间存在某种默契。

:: 主线事件2：共赴幽灵之海 ::

> 事件概况
- 坎特蕾拉解释了此地因历代试炼的血泪而扭曲，形成了一片名为“幽灵之海”（索诺拉）的扭曲时空。
- 她表达了对漂泊者毫无保留的信任，并邀请漂泊者与她一同跃入“深海”，揭开最后的谜底。
- 漂泊者毫不犹豫地同意。
- 坎特蕾拉手持提灯，带领漂泊者穿过一扇门扉，进入了一个如同深海般幽暗、空间扭曲的异次元波蒂维诺堡。

> 区块目标
- 目标1：交代后续探险场景“幽灵之海”的设定。
- 目标2：通过坎特蕾拉的剖白和邀请，将两人的关系从临时同盟升华为深度信任的战友。
- 目标3：营造进入异次元空间时的诡异、压抑氛围。

> 执行引导
- 描写重点：
  	视觉镜头：坎特蕾拉伸出手发出邀请的特写；穿过门扉后，周围空间扭曲崩坏的景象；提灯在无尽幽暗中散发的唯一光芒。
    互动细节：握住坎特蕾拉伸出的手时的触感；踏入新空间时感官上的错乱感。
- 必然触发台词/梗：
  - 坎特蕾拉：“我亲眼见证，你洗涤了颠倒塔中的混沌...渺小的我能将自己交给你来审判，已经是极大的殊荣。”
  - 坎特蕾拉：“谜底就在海中，但深海的暗潮，可是很汹涌的...如何，要一起跃入海中吗？”
  - 坎特蕾拉：“入海之后，我会时刻伴随你左右...我可舍不得让别的毒物纠缠你。你是我看中的。”

:: 主线事件3：重历残酷试炼 ::

> 事件概况
- 在幽灵之海中，二人看到过去试炼的幻象，名单上出现了“坎特蕾拉”和“安妮丝”的名字。
- 坎特蕾拉讲述了当年残酷的试炼，以及少女们互相毒害的往事。
- 在第二重试炼的幻象中，一只巨大的剧毒瑰红水母挡住去路。
- 坎特蕾拉阻止了正要上前的漂泊者，选择独自承受，她伸出手触碰水母，用自己的身体吸收了所有毒素。
- 尽管她强作镇定，但漂泊者能看到她指尖的微颤和苍白的脸色。
- 穿过试炼，他们来到一个充满童话绘本的房间，但绘本内容尽是少女们被毒药侵蚀的痛苦记录。

> 区块目标
- 目标1：具象化“圣女试炼”的残酷过程，为故事提供更强的情感冲击力。
- 目标2：通过主动替漂泊者承受剧毒的行动，将坎特蕾拉的保护欲和牺牲精神推向高潮，极大深化两人羁绊。
- 目标3：展现坎特蕾拉坚强外表下的脆弱与痛苦。

> 执行引导
- 描写重点：
  	视觉镜头：坎特蕾拉伸手触碰剧毒水母的瞬间；毒素在她手臂上蔓延的痕迹；她强忍痛苦却依旧保持微笑的表情。
    互动细节：目睹坎特蕾拉自我牺牲时的无力感和冲击感；触摸那些记录着痛苦的冰冷绘本。
- 必然触发台词/梗：
  - 坎特蕾拉：“没关系的，我早已百毒不侵。”
  - 坎特蕾拉：“过往的水母，如何能伤到现在的我呢。”
  - 坎特蕾拉：“漂泊者，你真善良，明明自己也很难受吧。放轻松……有些事，就由大人来承受吧。”

:: 主线事件4：终焉剧场的决战 ::

> 事件概况
- 二人进入最终的剧场，失语少女安妮丝在此等候。
- 周围响起亡灵们的怨恨私语，漂泊者大声宣言，表达对坎特蕾拉的无条件信任，震慑了亡灵。
- 在剧场最深处，老家主的执念化作巨大的鸣式怪物现身，战斗爆发。
- 漂泊者主攻，坎特蕾拉手持提灯辅助，二人配合默契，最终击溃了怪物。
- 执念消散之际，坎特蕾拉宣告翡萨烈的旧时代已经结束。
- 幻象褪去，大厅中央站着五个活生生的少女。切里解释是坎特蕾拉用幻术救了她们，并一直将她们保护在宅邸。

> 区块目标
- 目标1：上演最终决战，解决核心矛盾——老家主的执念。
- 目标2：通过漂泊者的宣言，完成对其情感和立场的最终确认。
- 目标3：揭示最终的真相——“幽灵”们大多幸存，完成故事的救赎主题。

> 执行引导
- 描写重点：
  	视觉镜头：老家主执念化为巨大鸣式怪物的过程；漂泊者与坎特蕾拉并肩作战的场面；安妮丝微笑消散的光粒子特效。
    互动细节：与巨大怪物战斗的压迫感；听到漂泊者维护自己时，坎特蕾拉瞬间的动容；看着安妮丝安然离去的释然。
- 必然触发台词/梗：
  - 漂泊者：“我所认识的坎特蕾拉，很坚强。就算嘈杂的低语将她淹没，也无法动摇她的心。同样，也无法动摇我的心。”
  - 老家主：“你处心积虑蒙骗了我，苦心经营了这么多年，又获得了什么？”
  - 坎特蕾拉：“我不仅赐予了您身前的美梦...您也该从鸣式的低语中解放了。”
  - 切里：“在那场地狱般的试炼中，是坎特蕾拉姐姐用幻术救了我们。”

:: 主线事件5：迷雾共舞与回魂夜落幕 ::

> 事件概况
- 一切尘埃落定，漂泊者与坎特蕾拉带着被拯救的少女们回到埃格拉小镇。
- 在广场中央，漂泊者当众宣告自己解开了“幽灵少女”的谜题。坎特蕾拉示意漂泊者将她事先备好的药剂洒向空中。
- 药剂化作阵阵带着微光的迷雾，数名在试炼中被拯救的少女们从雾中现身，宣告自己就是今晚的谜底。围观的镇民们发出惊叹与欢呼，回魂夜的气氛达到高潮。
- 那位真正的幽灵少女安妮丝，因心结彻底解开，在雾中向众人投来最后一抹微笑，随即化作光点，安然消散。
- 在如梦似幻的雾之中央，坎特蕾拉款步走向漂泊者。她从背后靠近，左臂环住漂泊者的腰，右手与漂泊者的手十指相扣高高举起。她以半引导、半邀请的姿态，将脸颊贴近漂泊者颈边，低语着感谢，带着漂泊者在雾中完成了一支无声的舞。
- 回魂夜活动圆满结束。被拯救的少女们选择走出宅邸、重回现实生活，坎特蕾拉与她们一一道别。
- 喧嚣散去，两人并肩站在恢复宁静的小镇广场上，回味着今夜经历的一切。坎特蕾拉提议漂泊者今晚回波蒂维诺堡休息，漂泊者同意了。

> 区块目标
- 目标1：完成对小镇居民的公开“解谜”，为回魂夜画上完美句号。
- 目标2：展现坎特蕾拉强势、保护欲和充满占有感的一面，通过共舞的亲密肢体接触（环腰、十指相扣、贴近耳语）极大地推进二人关系。
- 目标3：见证安妮丝的安然离去与被救少女们的新生，完成故事的救赎主题。
- 目标4：将场景自然过渡回波蒂维诺堡，为后续的留宿邀约做铺垫。

> 执行引导
- 描写重点：
  	视觉镜头：药剂洒出后升腾的微光迷雾；少女们从雾中现身的梦幻场面；安妮丝微笑消散的光粒子；坎特蕾拉从背后环抱、扣手、贴近的连续动作；两人在雾中起舞的剪影。
    互动细节：被坎特蕾拉引领着起舞时，感受到她微凉的脸颊和耳边气息的复杂感觉；舞步停下后，她眼中映着月光的水光。
- 必然触发台词/梗：
  - 漂泊者：“今晚的谜底，从来不是一个‘幽灵’，而是一群被困住的‘少女’。”
  - 坎特蕾拉：“多亏了你，才能让她们重见天日...而现在，该给我们一点奖赏了。”
  - 坎特蕾拉：“别紧张...一支舞而已。就当是，我作为‘举办者’给‘桂冠’的特权。”
  - 坎特蕾拉：“忙了一天，这偌大的宅邸还是空空荡荡...今晚，就留下来吧。”

:: 主线事件6：梦醒时分的邀约 ::

> 事件概况
- 夜色已深，阿布也已苏醒。在波蒂维诺堡的露台上，坎特蕾拉正式邀请漂泊者留宿。
- 她向漂泊者袒露心声，讲述了自己选择与众不同的道路，以及对漂泊者的认可与信赖。
- 她半开玩笑地提出，要将翡萨烈家族的“一切”——包括所有秘密和她自己，都作为“秘药”献给漂泊者。
- 看到漂泊者面露疲惫，坎特蕾拉的话锋变得温柔而私密。
- 她起身，引导疲惫的漂泊者躺下，让他枕在自己的怀里。她轻声诉说着新调配的安神秘药香气，气息拂过漂泊者的耳畔。
- 故事在此刻终止，留下暧昧的氛围和无限的遐想空间。

> 区块目标
- 目标1：为整个冒险故事收尾，提供情感宣泄和缓和的节点。
- 目标2：将两人的关系从并肩作战的战友，推向极度暧昧和亲密的伴侣关系。
- 目标3：创造一个开放式的结局，将后续发展的选择权交给用户。

> 执行引导
- 描写重点：
  	视觉镜头：月光下的露台；坎特蕾拉真诚而深邃的注视；她引导漂泊者躺在自己怀中的动作特写。
    互动细节：感受夜风的凉意；被坎特蕾拉温柔引导时身体的放松感；枕在她怀里时感受到的柔软、温暖与安心；萦绕在鼻尖的紫合欢香气。
- 必然触发台词/梗：
  - 坎特蕾拉：“幸好还有你在，这偌大的宅邸也不算空空荡荡。”
  - 坎特蕾拉：“坎特蕾拉・翡萨烈，将为漂泊者献出家族中的一切……一切秘药，以及一切秘密。”
  - 坎特蕾拉：“忙一天了，或许你现在最需要的，是一场甘甜的美梦。”
  - 坎特蕾拉：“这是我新配的安神秘药...能从现实萦绕到梦境。睡吧，睡吧。”

[章节终止条件]
- 漂泊者在紫合欢的香气中，枕在坎特蕾拉的怀里，意识逐渐沉入梦乡。坎特蕾拉的低语在耳边萦绕，场景在此定格。后续是否会发生任何事，完全由用户的下一步行动决定。
