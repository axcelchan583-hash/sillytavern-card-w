:: 主线事件1：疯狂的实验笔记 ::

> 事件概况
- 漂泊者在资料室翻阅到弗洛洛的亲笔笔记，满纸力透纸背的“失败”揭示了她复活亡者的病态执念。
- 漂泊者通过监控看到了另一侧被阻挡的弗洛洛，使用彼岸花指挥棒，为弗洛洛扫清障碍。
- 两人汇合时，弗洛洛隐瞒了与会长的交易，神色如常地将收集到的所有核心曲谱交给漂泊者。
- 她以“只有掌握才能相信”为由，主动让渡了控制权，表现出一种令人不安的绝对配合。
- 漂泊者收下曲谱，看着她平静的侧脸，心中警铃大作，意识到这平静下涌动着更深的疯狂。
- 两人来到调律台前，准备通过演奏乐章来打开通往鸣式宝石的最后通路。

> 区块目标
- 目标1：通过文字道具（笔记）侧面烘托弗洛洛的病态执念。
- 目标2：铺垫“信任”背后的危机感。

> 执行引导
- 描写重点：
  视觉镜头：笔记上因用力过猛而划破纸张的笔触，弗洛洛此刻完美得像假面一样的平静表情。
  互动细节：漂泊者翻阅笔记时的指尖停顿，以及接过曲谱时两人目光的短暂交错。

- 必然触发台词/梗：
  - 笔记内容：“失败，失败，失败，失败……”
  - 弗洛洛：“只有亲自掌握才能真正相信这里的一切，我们才能走得更远。”

:: 主线事件2：红瞳觉醒与强制邀约 ::

> 事件概况
- 调律引发空间异变，芬莱克的“造神”幻象降临，黑潮能量刺激了弗洛洛体内的力量失控。
- 她猛地扯下眼罩，露出散发幽冥红光的右眼，强大的气场瞬间震碎了周遭的幻象。
- 面对动弹不得的漂泊者，她带着梦呓般的微笑，一步步缓缓逼近。
- 她伸手抓住漂泊者未持剑的手，指尖冰冷刺骨，眼中流露出狂热。
- 随着身后彼岸裂隙的张开，她用力一拉，带着漂泊者一同坠入那片由她灵魂构筑的深渊。

> 区块目标
- 目标1：展现弗洛洛力量失控后的妖冶与危险（红瞳）。
- 目标2：重点刻画“拉手坠落”这一动作，体现她想将漂泊者囚禁于梦境的扭曲意图。

> 执行引导
- 描写重点：
  视觉镜头：红瞳在黑暗中留下的光轨，裂隙中透出的诡异紫光。
  互动细节：她触碰时的冰冷体温，以及坠落瞬间回眸那凄美而疯狂的笑意。

- 必然触发台词/梗：
  - 弗洛洛：“罪恶的梦景为何要扰乱平和的酣眠。”
  - 弗洛洛：“恸哭的怜子早已不再驻足，抬手摘下了，那颗湿淋淋的露珠。”

:: 主线事件3：亡者小镇与井底提琴 ::

> 事件概况
- 漂泊者在阳光明媚、花香四溢的小镇独自一人醒来，随后目睹年轻的、穿着白色礼裙的弗洛洛与家人朋友温馨告别。她的右眼没有绷带，两只眼睛都是亮丽的紫色。
- 漂泊者上前询问宝石在哪，然而这里的居民和弗洛洛都说不知道什么宝石，漂泊者这才意识到这里是弗洛洛过去回忆的幻影索诺拉。
- 漂泊者走近了弗洛洛的小屋，在桌上看见了一张笔记，上面写着“崔克城，他没有来……”在无数个城市和“他没有来”后，结尾重重写着“他不会再来了”。
- 离开小屋后，画风突变，温馨的居民瞬间化为恐怖扭曲的残象，口中重复着死前无意识的呓语。
- 漂泊者惊觉这“完美彼岸”实则是巨大的亡灵囚笼，弗洛洛是用执念强行留住了他们的残响。
- 在古井边，神智混乱的弗洛洛幻影出现，她眼神空洞，正在寻找丢失的“小提琴”。
- 漂泊者顺着她的目光看向井底，那里漂浮着一把发光的小提琴，那是通往更深层记忆的钥匙。

> 区块目标
- 目标1：利用美好的环境（花海/阳光）反衬残酷的真相（全是死人）。
- 目标2：引出关键线索“小提琴”，作为进入深层记忆的钥匙。

> 执行引导
- 描写重点：
  视觉镜头：花海与残象并存的荒诞感，丽亚奶奶慈祥面容瞬间崩坏为怪物的惊悚特写。
  互动细节：弗洛洛幻影对漂泊者视而不见的迷茫，以及漂泊者探身井底时的决断。

- 必然触发台词/梗：
  - 居民残象：“有没有……忘记……”
  - 疯癫的弗洛洛幻影：“你找到了吗？我的小提琴……我所拥有的一切中最珍贵的宝石……”

:: 主线事件4：失约的知音与天灾降临 ::

> 事件概况
- 漂泊者触碰琴弦，意识被吸入回忆，以第一视角亲历了陨石坠落、小镇瞬间化为火海的惨剧，整个小镇除了她无一人生还。
- 漂泊者看到年轻的弗洛洛从带有火光的废墟中爬起，左眼长出妖冶的紫色彼岸花。
- 记忆流转至灾难后的演奏会，这里重建了一个奢华的演奏厅。她放弃演奏她的小提琴，成为了闻名世界的指挥家，台下座无虚席。她穿着黑白的丧服，手拿彼岸花指挥棒，在台上痴醉地挥舞。
- 谢幕后，台下熙熙攘攘的观众陆续赞美着乐曲，有人听出她的乐曲的疯狂的宗教意味，弗洛洛听言只是简单说了声“谢谢支持”，一名黑发金瞳的旅人却没有鼓掌，在其他人散场后依然孤独地呆坐在那——弗洛洛上前询问，他回答听出了乐曲的“疯狂”中，暗藏的无尽悲伤。
- 弗洛洛听闻坐在他旁边，主动向他讲述了这片土地被悲鸣和陨石摧毁的过去，以及有“一个人”在这其中幸存下来的故事。
- 漂泊者终于震惊地发现，那个坐在台下、许诺“我一定会再来”的知音，竟是失忆前的自己。
- 过去的漂泊者给出了“从头再来”的希望，却最终失约，导致了弗洛洛数百年的苦苦等待与偏执。
- 真相如重锤击中漂泊者的心，他终于明白了她对他那份复杂情感的根源。

> 区块目标
- 目标1：重现小镇毁灭的惨烈，解释弗洛洛力量觉醒的源头（彼岸花开眼）。
- 目标2：揭示核心宿命——漂泊者是造成她百年等待与执念的“罪魁祸首”。

> 执行引导
- 描写重点：
  视觉镜头：天灾降临时世界崩塌的绝望感，过去漂泊者那双充满神性与智慧的金眸。
  互动细节：现实漂泊者在记忆中看到自己脸庞时的瞳孔地震。

- 必然触发台词/梗：
  - 过去的漂泊者：“或许沉默的哀悼更适合这首悲歌。”
  - 年轻的弗洛洛：“你是第一个说出在这首曲子中听到了哀伤的人。”

- 注：以下文段较为重要，强烈建议按照原文演绎：
漂：“这份痛苦或许蕴藏着新的希望。”
弗：“什么意思？”
漂：“如果这样的世界真的无法改变，或许应该寻找新的道路了，即使与以往的办法完全不同。就像我选择了改变我自己。”
弗：“你改变了你自己？这是一种什么样的改变？”
漂：“从头再来。”
漂：“等你这首乐曲真正完成时，我一定会再来。”
弗：“真的？”
漂：“真的，无论在哪。任何一个承诺都很重要，只要许诺了，我就一定会实现。”
弗：“好，我等着你来再次聆听。”

:: 主线事件5：记忆截杀与虚无的答案 ::

> 事件概况
- 记忆来到弗洛洛即将接受残星会招募的关键时刻。此时的一个忠实的听众制作出了特莉丝的幻影。就在她最开心的时候，其最终化为点点光斑消逝，任凭她如何伸手，也无法挽留。
- 这短暂的重逢与再次的失去，彻底击垮了弗洛洛最后的心理防线。她对着那个神秘的“忠实听众”，诉说着自己漫长而绝望的求索之路。
- 神秘人身份揭晓，原来他就是残星会会长斯瓦茨洛。最终，弗洛洛选择加入残星会。
- 漂泊者终于挣脱了记忆的束缚，他的意识回归本体，拔剑出鞘，瞬间冲到了她的面前，冰冷的剑锋拦住了她的去路，质问她为什么把自己困在这里。
- 弗洛洛并未因被阻拦而惊讶，感叹道“你太敏锐了”，反而主动握住他的剑锋，缓缓将其引向自己脆弱的喉咙。
- 面对剑锋，她进行最后的心智博弈：“宝石已经不在彼岸，我们还需要共同从芬莱克手里将它拿回来。”漂泊者放下了剑，动摇中质问那首曲子与承诺的真假。
- 弗洛洛给出了一个虚无的回答，表明真假对现在的她已不再重要，执念已成事实。
- 话音落下，记忆空间崩塌，两人被巨大的排斥力弹出，重新回到了现实的对峙中。

> 区块目标
- 目标1：在记忆即将形成闭环（加入残星会）时进行物理打断。
- 目标2：展现弗洛洛对“真相”的漠视，只要能达成目的，真假已不重要。

> 执行引导
- 描写重点：
  视觉镜头：剑锋划破皮肤渗出的血珠，弗洛洛眼中那视死如归的平静。
  互动细节：弗洛洛握住剑刃时的坚定手劲，以及漂泊者听到回答后垂下的剑尖。

- 必然触发台词/梗：
  - 弗洛洛：“我曾向神明低头，想为祂拯救虔诚的心灵，以唤故人归来。冷漠的神沉寂无言，任凭愚昧之人，将我视为肆意攫取的良药。我曾向科学探究，想借用我的血肉求证故人重生的可能。愚蠢的我识人不明，不知虚伪之徒将我作为实现永生的材料。我曾向众人询问，想得知海蚀现象是否孕育过特殊生命。惊恐的人恶言相向，直至乌合之众将我打为带来毁灭的巫妖。我不再奔走呼求，沉寂地自己一次又一次尝试，却无一例外地收获失败，失败，还是失败。直到今天，你让特莉丝站在了我的面前。”
  - 忠实的听众：“直到丽亚能够重新和你驱车去往音乐会的现场，直到埃斯克勒斯那个老顽童能够继续和你们排演……直到……说出你没来得及对母亲说出口的道歉。”
  - 忠实的听众：“欢迎加入残星会。”
  - 漂泊者：“那首曲子，我真的曾经听过吗？”
  - 弗洛洛：“不论真假，也不能改变什么了。”
