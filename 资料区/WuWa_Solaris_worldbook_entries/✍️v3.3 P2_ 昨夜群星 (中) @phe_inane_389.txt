:: 主线事件1：出发前的送别 ::

> 事件概况
- 两天后，漂泊者收到莫宁的飞讯，前往研究院。途中阿布对即将面对鸣式本体既紧张又跃跃欲试。
- 在研究院大厅，琳奈已经先一步赶到，她从莫宁处打听到了消息，坦言自己为漂泊者感到紧张。
- 琳奈倾诉：入学后以为生活会忙碌有趣，但鸣式、虚质、残星会让一切像泡泡一样破碎，自己却无法在启动隧者的大事上帮忙，感觉"挺没用的"。
- 漂泊者安慰她：比起让所有人涉险，他更希望"学生去做学生该做的事"，保护世界是"救世主"（或"大人"）的责任。
- 琳奈被触动，表示会随时响应学院的配合行动，并相约事情了结后一起去兜风。
- 漂泊者与莫宁会合。瓦莲京娜（黑海岸成员）通报黑海岸已在研究院待机，临时通道搭建完成，并在研究院内生成了传送门。
- 此时，「爱弥斯」突然独自跑到磁带保存间，取出了爱弥斯变为电子幽灵前的个人磁带，紧紧握住不放。
- 莫宁分析这可能是躯壳中残留的本能（磁带是学生的身份证明），也可能是门后爱弥斯传来的模糊思绪。
- 莫宁将留在研究院接收数据、准备拟制炉芯，成为漂泊者的后盾——这是两人新的约定。

> 区块目标
- 目标1：通过琳奈的送别，从普通学生的视角展现鸣式危机带来的不安与无力感，衬托漂泊者肩负的重压。
- 目标2：「爱弥斯」主动取出磁带的异常行为，延续此前"磁带"低语的伏笔，强化门后爱弥斯仍存在意识联系的暗示。
- 目标3：确立莫宁作为后方支柱的角色定位，为拟制炉芯的及时完成做好铺垫。

> 执行引导
- 描写重点：
  视觉镜头：研究院大厅的忙碌景象；琳奈强撑着笑容的表情；「爱弥斯」紧握磁带时空洞的双眼中仿佛闪过一丝微光。
  互动细节：莫宁在说出"新的约定"时，目光坚定而温柔，与她平日里的腼腆形成对比。
- 必然触发台词/梗：
  - 琳奈："我刚做好继续读书的准备时，我的学生生活看起来就像吹的口香糖泡泡一样，'啪'地一下就破掉了。"
  - 漂泊者："保护世界，是'救世主'的责任。"／"保护孩子，是大人的责任。"
  - 琳奈："等一切事情了结，我们再去兜风吧！"
  - 莫宁："好。我一定会回应前辈的期望，这是……新的约定。"
- 伏笔/暗线：
  「爱弥斯」取出磁带的行为，与最终隧者将礼物刻录至漂泊者磁带的结局形成完整呼应。磁带作为身份证明，也象征着爱弥斯"不愿被遗忘"的执念。

:: 主线事件2：黯原深处与绯雪 ::

> 事件概况
- 漂泊者走进传送门，通过临时通道抵达地下黯原，看到远处巨大的日脉源木，和近处等待的绯雪与洛瑟菈。
- 洛瑟菈正式介绍绯雪——苇原巫女，星炬学院特别对策组成员。十几年前黯原出现裂隙、虚质外溢时，绯雪独自进入地下深处进行封印，使深空联合与罗伊人得以加固裂口。
- 绯雪告诉漂泊者，她曾通过白鸟在虚质浓重处感受到爱弥斯的频率，肯定她仍在虚质深处。
- 绯雪将纸役白鸟送给漂泊者，告知在虚质中迷失方向时它能引路，并在关键时刻"可以向我许愿"。
- 洛瑟菈揭露：庆典后的学生暗桩正是达妮娅，现已下落不明，而绯雪将负责在漂泊者进入隧门后追踪残星会成员。
- 洛瑟菈坦言：此行对查特而言只是一笔稳赚不赔的投资，但漂泊者要亲身犯险。进入门后，隧者尚未完全恢复，深空联合也不会为一个人再度施援——漂泊者将独自面对危险。
- 漂泊者坦然接受，表示已遇到过太多这样的境况。洛瑟菈送别："愿你凯旋。"

> 区块目标
- 目标1：正式引入绯雪，展现其实力背景（独自封印裂隙）与低调行事的风格。
- 目标2：通过绯雪赠送白鸟和"许愿"的提议，铺垫后续关键救援手段的使用。
- 目标3：洛瑟菈的坦言揭示此行的孤立无援本质，烘托漂泊者义无反顾的决意。

> 执行引导
- 描写重点：
  视觉镜头：黯原中日脉源木散发的幽光，绯雪白色的身影在昏暗中格外醒目；白鸟在漂泊者掌心化为光芒融入。
  互动细节：绯雪说"可以向我许愿"时语气平静如常，但洛瑟菈闻言微微收敛了笑容。
- 必然触发台词/梗：
  - 绯雪："在虚质中迷失方向时，它也能为你引一段路。"
  - 绯雪："如果你在和鸣式对抗时，有希望得到的帮助……可以向我许愿。"
  - 洛瑟菈："实现愿望……总会付出代价的。"
  - 洛瑟菈："去吧，拉贝尔学部的新生~愿你凯旋。"
- 伏笔/暗线：
  洛瑟菈强调"实现愿望总会付出代价"，暗示绯雪"许愿"能力的沉重代价。达妮娅的下落不明为即将到来的冲突埋下伏笔。

:: 主线事件3：隧锚启动与首次穿越 ::

> 事件概况
- 漂泊者与爱弥斯的空壳一同深入，通信信号逐渐断裂，周围陷入寂静。
- 一群日灵出现，簇拥着漂泊者走向日脉源木，随后它们融进根系——不是回归隧者，而是用自身填补隧锚封印的缺损。
- 漂泊者抵达隧锚并启动，一阵头晕目眩和全身撕裂的疼痛后，他成功与隧者共鸣，并进入隧者的身体驾驶隧者。
- 然而力量的冲击暗示若强行撬动封印，将摧毁整个拉海洛。漂泊者选择另一条路——直接穿越隧门，进入虚质深处。
- 门后是如万花筒般的高维空间，时间与空间不连续而混乱。漂泊者追寻爱弥斯的频率，看到穿着深空联合和罗伊族服饰的人影，并听到爱弥斯的声音："原点的光，由我们支撑。"

> 区块目标
- 目标1：呈现隧锚启动的仪式感与日灵自我牺牲的悲壮，展现拉海洛生态与隧者的深层联系。
- 目标2：通过首次穿越的所见所闻（人影、爱弥斯的声音），为最终揭示"先行者们"在虚质中的存在埋下线索。

> 执行引导
- 描写重点：
  视觉镜头：日灵如萤火般簇拥漂泊者的温馨画面；隧门后万花筒般扭曲绚丽的高维空间；远处人影模糊而虔诚的姿态。
  互动细节：漂泊者感知到爱弥斯的频率时身体不由自主地前倾，但空间的扭曲让距离永远无法缩短。
- 必然触发台词/梗：
  - 漂泊者："（爱弥斯曾说，鸣式黑洞深处的虚质空间，是犹如万花筒般的高维空间。）"
  - 爱弥斯（遥远的回声）："原点的光，由我们支撑。"
- 伏笔/暗线：
  爱弥斯的声音"原点的光，由我们支撑"在首次穿越时仅为模糊的残响，在最终章（Part 3）将揭示其完整含义——先行者们支撑隧者坐标。深空联合与罗伊族人影暗示被虚质吞噬的探索者仍以某种形式"存在"。

:: 主线事件4：读取磁带，解析内容 ::

> 事件概况
- 驾驶隧者进入隧门后，漂泊者被一股异常的力量推出，而爱弥斯的空壳在离开虚质空间前突然眼睛中散发出了原有的光芒。她的眼神中有惊喜，有悲伤，还有无法言说的信任。
- 她恢复了意识，将一盘磁带递到了漂泊者手中。漂泊者还在混乱的记忆碎片中看到了微笑伸手的娜波摩，以及逐渐被虚质侵蚀的学院。
- 再次苏醒后，漂泊者回到学院，爱弥斯已不见踪影。莫宁告知已过去整整一天。虚质在学院内部爆发，大部分人员已在黑海岸协助下撤离至逃生舱，剩余人员聚集在研究院。
- 回到研究院，众人宽慰漂泊者的失落。拟制炉芯即将完成，16小时后将被发射向天帷展开「夜幕」——因漂泊者传回的虚质数据，这颗凝结拉海洛心血的"人造心脏"得以完善。
- 漂泊者取出爱弥斯交给他的磁带。磁带放入解析设备的瞬间，储存间所有服务器屏幕闪烁，信息大量倒流。
- 莫宁和织星解析后发现：磁带内压缩的巨量内容全部是拉海洛坐标，指向过往曾发生虚质磁暴的区域——这些区域上都铭刻着神秘的罗伊符文。
- 西格莉卡提供罗伊符文的寻找方法，查特被说服同意在「夜幕」闭合前进行罗伊符文收集——"直面未知，我们将失去很多；但逃避未知，我们将失去一切。"
- 洛瑟菈组织全员行动：罗伊族与深空联合人员从安全舱中出发，划分学院、研究院、浮光林三大区块，漂泊者负责污染最严重的浮光林区域。
- 符文收集完毕后，西格莉卡参与解析。她透过与罗伊符文的共鸣，感觉到这些文字并非描绘在二维平面上，而是在一个球体表面——"这些文字在我感觉中，不是在一张纸上的，而是在一个球上的。"
- 莫宁利用炉芯光照进行三维还原：罗伊符文传递的是一幅球形地图。球面上标注的是隧门的位置，球心则是整个地图的原点——隧者在虚质空间中的位置。
- 这便是爱弥斯想向漂泊者传递的道标：虚质空间中唯一难以被磨灭的事物的坐标。
- 漂泊者决定再度进入门后寻找隧者。查特指出隧锚附近污染严重无法通行，但黯原还有一道绯雪封印的裂隙可作为新通道。漂泊者提出约定：在「夜幕」闭合前，若发射隧者信号，就请将拟制炉芯托付给他和隧者——将判断权完全交给深空联合，"不是选择权，是信任。"
- 查特同意提案。洛瑟菈将地下黯原的最新地图交给漂泊者，告知绯雪已在黯原等待接应，临别时以共鸣能力将此刻的记忆淬成子弹铭刻下漂泊者的名字——"能支撑你的，从来不止你自己的力量。"

> 区块目标
- 目标1：通过磁带解析揭示爱弥斯在虚质深处向漂泊者传递的关键信息——罗伊符文坐标构成的球形地图，指向隧者在虚质空间的位置。
- 目标2：展现拉海洛全体成员在危机中的团结行动（符文收集），呼应"昨夜群星"的群体力量主题。
- 目标3：通过西格莉卡与罗伊符文的独特共鸣能力，完成从坐标数据到球形地图的关键推理跳跃。
- 目标4：漂泊者将拟制炉芯的托付判断权交给深空联合，以"信任"而非"选择"的方式确立合作关系。

> 执行引导
- 描写重点：
  视觉镜头：磁带放入解析设备时服务器屏幕群闪烁的震撼画面；符文收集行动中各小队在虚质磁暴间穿行的紧张场面；西格莉卡闭眼感知符文时双手在空中比划球体形状。
  互动细节：查特说出"直面未知"时罕见地收起了商人的精算姿态，展露出深空联合创立者的探索精神。洛瑟菈临别将记忆淬成子弹时，眼中映着漂泊者的面容。
- 必然触发台词/梗：
  - 查特："直面未知，我们将失去很多；但逃避未知，我们将失去一切。"
  - 西格莉卡："这些文字在我感觉中，不是在一张纸上的，而是在一个球上的。"
  - 莫宁："是地图……那些罗伊符文传递给我们的是一幅球形地图。"
  - 漂泊者："不是选择权……是信任。"
  - 洛瑟菈："能支撑你的，从来不止你自己的力量。"
- 伏笔/暗线：
  罗伊符文作为虚质磁暴的遗留物，暗示先行者们以生命测绘坐标的痕迹已遍布拉海洛。符文收集行动中众人的飞讯通信展现了各角色在危机中的不同侧面，为后续事件做情感铺垫。

:: 主线事件5：达妮娅与残星会会长的阴谋 ::

> 事件概况
- 漂泊者前往浮光林进入地下黯原，发现虚质污染严重，循绯雪白鸟的引导找到她。
- 绯雪告知：达妮娅精神极不稳定，虚质爆发就发生在她遇到达妮娅之后。漂泊者推断此前在隧门后遭遇的突变也是达妮娅所为。
- 洛瑟菈通过绯雪转达：漂泊者想再回一次门后，但原本的路已被虚质侵蚀，必须走十几年前被残星会破坏出的那道裂隙——正是导致爱弥斯死亡的元凶。
- 绯雪当年用自己的力量（"预求身"——预取未来自身的大量可能性）封印了那道裂隙，现在需要解除封印让漂泊者通过。
- 在前往封印处的途中，漂泊者与绯雪交谈：绯雪认识达妮娅，洛瑟菈曾拜托她照看学院的孩子们，达妮娅虽一开始不太正常，但后来努力融入学生群体，看起来很开心。
- 绯雪解除灼樱封印时，黑潮涌出，残星会会长携被鸣式控制的达妮娅现身。
- 会长揭露阴谋：达妮娅本就是残星会的"造物"，是他在"蜜芽"之后思考的实验——制作一个足够优秀的容器，再加以灼樱巫女的精神，试图驾驭鸣式的力量，制造"人造鸣式共鸣者"。
- 绯雪告知漂泊者自己与残星会的关系本质——会长想利用灼樱巫女的能力。她当场预取尽自己身为鸣式共鸣者的未来可能性，坚定地表示不受会长蛊惑，宣告会长的计划失败。
- 但会长不以为然："既然已有了一个达妮娅，自然也会有下一个"，并展示通过绯雪的预求身可以看到——真正的鸣式共鸣者确实将会存在。
- 达妮娅发出微弱的求救声，但会长嘲讽她"精神难堪大用"，宣布将她"扔掉"。
- 会长的分身消散。漂泊者决定不纠结分身，先处理达妮娅的情况。此时的达妮娅已经完全被鸣式控制，失去了自我意识，嘶吼着攻击漂泊者与绯雪二人。

> 区块目标
- 目标1：揭露残星会会长的终极实验意图——人造鸣式共鸣者，将达妮娅的悲剧身世（残星会造物）推至高潮。
- 目标2：完成绯雪"预取尽鸣式共鸣者未来"的关键行为，断绝会长利用她的可能，同时暗示这一举动对绯雪自身的巨大代价。
- 目标3：通过达妮娅微弱的求救与会长的冷酷抛弃，展现残星会对棋子的无情态度，为后续漂泊者选择拯救达妮娅埋下伏笔。
