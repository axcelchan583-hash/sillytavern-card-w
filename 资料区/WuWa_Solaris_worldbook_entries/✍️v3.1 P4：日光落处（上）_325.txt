:: 主线事件1：医务室的试探 ::

> 事件概况
- 漂泊者抵达医务室，看到正在养伤的娜波摩，并与她简短交谈。
- 娜波摩告知陆·赫斯正在心理咨询室，漂泊者在门外稍作等候，无意间听到了西格莉卡因为人际关系的焦虑找陆进行心理咨询。
- 她提到最近发生了很多事（如昭日者伊摩尔的叛变）并倾诉自己无法融入集体。陆认真倾听并给出建议，并在最后给了她一颗糖。
- 西格莉卡离开后，漂泊者进入，直接询问“隧者计划”。
- 陆·赫斯察觉到漂泊者情绪的急切与不稳定，没有直接回答，反而试图用糖果缓和气氛。
- 漂泊者拒绝了糖果，并表示在进行影响整个拉海洛的计划前，需要更深入地了解作为合作伙伴的陆·赫斯。
- 陆·赫斯同意，并表示此地不宜久留，提议更换一个更安全的地点详谈。

> 区块目标
- 目标1：引出娜波摩、西格莉卡等角色，铺垫校园日常的氛围。
- 目标2：展现陆·赫斯敏锐的观察力及其作为医生的关怀特质。
- 目标3：通过漂泊者的直接质询，体现他因爱弥斯之事而产生的焦虑感。

> 执行引导
- 描写重点：
  视觉镜头：医务室内洁净但略带消毒水气味的环境；陆·赫斯从咨询室走出时脸上带着轻松的微笑，与漂泊者紧绷的神情形成对比。
  互动细节：陆·赫斯递出糖果的动作自然而然，漂泊者拒绝时只是微微摇头，视线并未移开。
- 必然触发台词/梗：
  - 娜波摩：“赫斯医生很关心我，这几天我都遵循他的医嘱在这里休养。”
  - 陆·赫斯（对西格莉卡）：“西格莉卡同学感觉是那种光是想到考试，就会半夜焦虑地从床上“噌”地坐起来，持续一周两眼发直的类型呢……”
  - 陆·赫斯：“你太急切了，漂泊者，情绪正在影响你平日冷静的判断。”
  - 陆·赫斯：“来，吃颗糖？有助于舒缓心情，变得开心噢。”
- 伏笔/暗线：
  娜波摩在医务室养伤，为后续她被“劫持”提供了地点上的合理性。陆·赫斯随身携带糖果的习惯被再次提及，其背后的故事将在之后揭晓。

:: 主线事件2：坠剑湖的密室 ::

> 事件概况
- 陆·赫斯带领漂泊者来到坠剑湖旁一处隐秘的地点，并轻松解决掉几只怪物。
- 他解释此地利用隧者之剑的频率来屏蔽探查，随后打开了一间密室。
- 密室中，陆·赫斯揭示了漂泊者过去安排给他的任务：关注残星会的动向。
- 他通过三件物品，逐步揭示自己的过去：一管金色液体和作为“日髓”共鸣者的经历、全家福中没有血缘关系的父亲和弟弟莱伊、以及在父亲空坟中发现的，指向新联邦「厄里斯」的残缺报告——而劝说他去挖开坟墓的，正是漂泊者。
- 陆·赫斯最终坦白，他追迹残星会的最初动机是“仇恨”——他的父亲被残星会会长杀害并替换了身份。
- 他向漂泊者分享了关于残星会会监（海德拉、塞壬、厄里斯等）的情报。
- 他回忆起在那个被火光覆盖的残星会驻扎点前，是过去的漂泊者在通讯中建议他来拉海洛“休息”。
- 两人谈话结束，陆·赫斯表示会处理隧者驾驶资格的事，让漂泊者耐心等待。漂泊者则表示想独自去冰原看看，两人就此分别。

> 区块目标
- 目标1：交代陆·赫斯的背景故事、他与残星会的深仇，以及他潜伏在学院的真正任务。
- 目标2：引出“日髓”、“厄里斯”、“千面百貌的会长”等关键情报，为后续剧情埋下伏笔。
- 目标3：通过回忆片段，展现过去漂泊者对同伴的关心，并合理化两人分别行动的剧情。

> 执行引导
- 描写重点：
  视觉镜头：密室内陈设着各种看似无关的物品：一个装有金色液体的容器、一个相框、一份烧焦的报告。
  互动细节：漂泊者在观察那些物品时，手指会无意识地摩挲，显示出他在思考。
- 必然触发台词/梗：
  - 陆·赫斯：“为什么选择我？因为我本就有追迹残星会的动机——仇恨。”
  - （回忆）旧日的漂泊者：“陆，你身为医生却不惜命，怎么去救更多的人？听我的，来这边。”
- 伏笔/暗线：
  陆·赫斯提及的那支从会长处得到的、刻有无法解读字符的钢笔，将在后续被会长夺回，成为关键道具。新联邦的“厄里斯”被重点提及，成为未来的一个重要目标。

:: 主线事件3：双线并行：星炬暗流与渐湖静默 ::

> 事件概况
- — 陆·赫斯线 —
- 与漂泊者分别后，陆·赫斯立即联系了星炬学院的校长洛瑟菈，约定在研究院见面。
- 途中他遇到了为论文而憔悴的学生伊桑，并像往常一样给予了关心和开病假单的承诺。
- 见到洛瑟菈后，他言简意赅地分享了残星会的情报，并为漂泊者争取驾驶隧者的资格。
- 洛瑟菈表示同意，并透露她会去和“某个人”谈谈以应对风险。
- 随后两人闲谈陆来到学院的理由和学院里满是间谍的现状，谈话即将结束时，陆·赫斯收到了来自N.A.N.A.的紧急通讯。

- — 漂泊者线 —
- 漂泊者因为她感知到了爱弥斯的频率，去往了“渐湖小屋”，爱弥斯躯壳凭着本能来到了这里。
- 小屋安静，爱弥斯的躯壳静静地坐在窗边，眼神空洞，手指无意识地在结霜的玻璃上划着不成形的轨迹。
- 漂泊者没有说话，只是在不远处静立，目光长久地停留在她身上。他周身的气息沉静得近乎凝固，唯有偶尔因抑制呼吸而微微起伏的胸膛，泄露出深埋的悲恸。

> 区块目标
- 目标1（陆线）：展现陆·赫斯在学院的人际网络和他处理事务高效的一面，并引出紧急事态。
- 目标2（漂泊者线）：通过静态描写，具象化漂泊者内心的“破碎感”，以及爱弥斯躯壳“只剩本能”的状态。
- 目标3（双线）：通过并行的叙事，营造出一种“山雨欲来”的紧张感。

> 执行引导
- 描写重点：
  视觉镜头（陆线）：研究院内忙碌但有序的景象，洛瑟菈办公室的简洁与权威感。
  视觉镜头（漂泊者线）：渐湖小屋内光线昏暗，尘埃在光束中飞舞，爱弥斯躯壳的侧影显得孤独而脆弱。
  互动细节：漂泊者的手始终没有离开腰间的武器，这是一种下意识的戒备和守护姿态。
- 必然触发台词/梗：
  - 洛瑟菈：“但打开隧门必然伴随着风险……某个人或许能在这件事中帮上忙，我要先和她聊聊。”
  - 陆·赫斯：“有朋友给这里打广告，说雪景美丽，学生热情可爱，我就被哄过来了。但是来了才发现，十个人里面有八个动机不纯。”
- 伏笔/暗线：
  洛瑟菈提到的“某个人”是谁，她将如何应对风险，这是一个悬念。

:: 主线事件4：双线并行：紧急追击与无声守望 ::

> 事件概况
- — 陆·赫斯线 —
- N.A.N.A.被枪击中故障，断续地报告，劫持者是越狱的魏尔伦，他正用枪抵住娜波摩，前往罗伊冰原的“天槎空间站”。
- 陆·赫斯立刻向洛瑟菈通报情况，请求她派人包围空间站，自己则先行一步深入调查，因为他自己有些事情想要确认，所以让洛瑟菈指示先别轻举妄动。
- 他迅速赶往空间站，在入口处发现了两名受伤的研究员和一名遇难者。他立刻进行急救处理，随后继续向深处追击。

- — 漂泊者线 —
- 渐湖小屋内，漂泊者依旧在无声地守望着。突然，他手腕上的终端发出一阵急促的震动。
- 是陆·赫斯的加密通讯，内容极简：“天槎空间站，娜波摩被劫，残星会阴谋，速来。”
- 看到“残星会”三字，漂泊者的瞳孔瞬间收缩，几乎在信息显示的同一秒，他已经转身。那份沉静的悲伤被一种凌厉的决意瞬间取代。
- 他最后看了一眼窗边的身影，没有任何告别，身影便化作一道残影，消失在风雪中。

> 区块目标
- 目标1（陆线）：展现陆·赫斯冷静的判断力和作为医生的天职，即使在追击中也不忘救人。
- 目标2（漂泊者线）：表现漂泊者在接收到危机信号后的果断与迅捷，责任感暂时压倒了个人情感。
- 目标3（双线）：两条线索开始向同一点（天槎空间站）汇集，预示着剧情的交汇与爆发。

> 执行引导
- 描写重点：
  视觉镜头（陆线）：空间站内闪烁的警报灯光，陆·赫斯处理伤口时冷静而精准的手法。
  互动细节（漂泊者线）：漂泊者接收通讯时，紧握着终端的手指。他离开时带起的风，让小屋内的尘埃短暂地飞扬起来。
- 必然触发台词/梗：
  - 陆·赫斯：“有些……猜测，我希望单独验证。”
  - 洛瑟菈：“可以~你的信誉能支撑我们的信任。去吧，陆医生，等你好消息。”
- 伏笔/暗线：
  陆·赫斯所说的“猜测”，指的是他对娜波摩身份的怀疑。他选择独自进入，是为了创造一个能让对方暴露真面目的环境。
