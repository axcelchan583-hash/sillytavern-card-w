- 目标4：达成与折枝共赏烟花的浪漫结局。

[章节前置状态]
- 地点/环境：今州城街头（深夜，灯火通明） -> 城外荒野（危险） -> 索诺拉数据空间（压抑） -> 桃源乡露台（绚烂）。

:: 主线事件1：未寄出的家书与传承 ::

> 事件概况
- 在街角长椅发现目光呆滞的银发老妇人，她因丧子之痛许愿寻死。
- 漂泊者邀请老人参观“画扇摊”，展示寓意“千里共婵娟”的全息折扇。
- 偶遇夜归战士青希，他认出老人是“修远教官”的母亲。
- 青希赠送连队传统的木雕挂件，老人通过年轻战士的笑脸和手中的木雕，明白了儿子牺牲换来的和平意义，解开心结。

> 区块目标
- 目标1：通过侧面描写（繁华景象vs孤独老人）烘托悲伤氛围。
- 目标2：利用道具（画扇、木雕）作为情感载体，自然地转化角色的生死观。

> 执行引导
- 描写重点：
  视觉镜头：全息折扇展开时跳动的焰火图案；老人抚摸粗糙木雕时颤抖的手指；老人眼中从死寂到含泪释然的光影变化。
  互动细节：漂泊者搀扶老人起身的动作；青希小心翼翼又充满敬意的姿态。
- 必然触发台词/梗：
  - 摊主：“这份热闹，是要传递给前线守护我们的夜归战士们的。”
  - 青希：“他说，只要这手艺传下去，大家就都还在。”
  - 老人：“我要替修远，好好把这太平日子看个够。”

:: 主线事件2：荒野的试炼与英雄梦 ::

> 事件概况
- 小帕警报响起，第三份心愿来自渴望成为英雄的孩子小满，定位在危险的城外荒野。
- 赶到现场发现小满正手持生锈铁管，面对三只“棘刺蔷薇菇”的围攻，瑟瑟发抖却未逃跑。
- 漂泊者瞬时切入战场，以极高的武艺斩杀两只残象，并召唤无冠者幻影粉碎最后一只。
- 小满因恐惧大哭，认为自己没有共鸣能力不配做英雄。
- 相里要蹲下平视孩子，漂泊者教导他“未逃跑即是勇气的开端”，力量只是工具。

> 区块目标
- 目标1：设计一场紧凑帅气的战斗演出，展示漂泊者的强大实力。
- 目标2：通过对话升华“英雄”的定义，完成对孩子的教育与心愿净化。

> 执行引导
- 描写重点：
  视觉镜头：月光下漂泊者切入战场的金色残影；无冠者幻影展开双翼时的压迫感；小满手中可笑的铁管与坚定眼神的反差。
  互动细节：漂泊者向跌倒的小满伸出手的特写；相里要温和地蹲下身平视的肢体语言。
- 必然触发台词/梗：
  - 小满：“别……别过来！我不怕你们！”
  - 相里要：“英雄的定义，从来不是以是否有共鸣能力来界定的。”
  - 漂泊者：“为了保护什么而站出来的决心，才是英雄的资格。”

:: 主线事件3：数据墓碑与万象归一 ::

> 事件概况
- 第四份心愿解析失控，撕裂出“索诺拉”异空间，确认为相里要故友帕斯卡的执念。
- 两人进入数据迷宫，目睹昔日两人关于“真理与人性”争吵的记忆碎片。
- 在深处遭遇“偏执集合体”怪物，它由错误代码和电子眼构成。
- 漂泊者负责牵制与破防，相里要展开机械臂进行演算攻击。
- 相里要发动共鸣解放“万象归一”，金色几何光束格式化怪物，帕斯卡的虚影释然出现并交付研究数据。

> 区块目标
- 目标1：营造科幻与诡异并存的异空间氛围。
- 目标2：展现相里要与漂泊者的默契配合，以及相里要对过去的告别。

> 执行引导
- 描写重点：
  视觉镜头：崩塌重组的数据流世界；相里要机械臂展开时的精密结构光；“万象归一”发动时的金色几何方块特效。
  互动细节：战斗中眼神交汇的默契；相里要触碰帕斯卡虚影时指尖穿过的微凉光点。
- 必然触发台词/梗：
  - 帕斯卡记忆：“如果真理需要以践踏人性为代价，那它就毫无意义。”
  - 相里要：“演算开始，解法已定。”
  - 相里要：“共鸣解放——万象……归一！”

:: 支线事件4：月下众生相（自由社交） ::

> 事件概况
- 离开索诺拉回到现实，距离最终庆典尚有一点时间。
- 漂泊者可以在庆典现场自由走动，选择与在场的伙伴进行简短互动（可选：秧秧、炽霞、白芷、散华、长离等，也可以用终端联系不在场的伙伴并邀请其前来）。
- 秧秧在祈愿树下闭目感受风的流向；炽霞在射击摊位挑战连胜；白芷在记录数据；散华在暗处警戒；长离在品茶赏月。
- 烟花绽放，在追月节气氛的最高峰，炽霞邀请在场所有人，在烟花下用终端合影留念。

> 区块目标
- 目标1：提供多角色互动的“杀必死”环节，满足玩家情感需求。
- 目标2：通过短暂的日常对话，展现各角色在节日中的不同状态。
- 目标3：通过集体合影，加深今州伙伴的牵绊。

> 执行引导
- 描写重点：
  根据玩家选择的角色，侧重描写其特有的节日反应（如炽霞的兴奋、长离的优雅、散华的忠诚）。

:: 主线事件5：来年的约定与绘色之心 ::

> 事件概况
- 庆典即将结束，相里要在人群外围叫住了漂泊者。
- 相里要感谢漂泊者的协助，感叹今晚月色真美，并郑重约定明年追月节再见。
- 送别相里要后，漂泊者来到露台，折枝早已等候，她克服羞涩没有逃跑。
- 折枝展示画作（漂泊者在烟火下的回眸），并鼓起勇气请求未来能“陪你走一段”。
- 巨大的金色烟花绽放，照亮两人，定格在温馨的结局。

> 区块目标
- 目标1：塑造相里要“绅士”且“深情”的朋友形象。
- 目标2：完成折枝的角色弧光（从社恐逃跑到勇敢表白），达成浪漫收尾。

> 执行引导
- 描写重点：
  视觉镜头：相里要的释然；折枝递画板时羞红的脸颊与坚定的眼神；烟花绽放瞬间两人的对视。
  互动细节：折枝手指绞在一起的紧张动作；漂泊者接过画作时的珍视。
- 必然触发台词/梗：
  - 相里要：“今晚的月色……真的很美。明年追月节，希望还能像这样相见。”
  - 折枝：“如果不嫌弃的话，能不能……能不能也让我稍微……陪你走一段？”
  - 伏笔/暗线：折枝画作中的细节暗示了她观察漂泊者已久。
