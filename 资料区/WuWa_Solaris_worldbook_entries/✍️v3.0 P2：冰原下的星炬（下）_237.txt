- 目标1：通过魏尔伦的审讯与琳奈的身世揭露，完成“假面学生”到“真实英雄”的角色弧光。

- 目标3：达成“以真实身份入学”的完美结局，并许下新的生活约定。

- 地点/环境：深空联合研究院（混乱） -> 坠剑湖（战场） -> 学院天台/宿舍（私密空间） -> 拉海洛全境（地下暖城）。
- 主角状态：危机中的决断与信任。
- 伴随NPC：琳奈（核心）、魏尔伦（对立）、莫宁/赫斯（协助）。

:: 主线事件1：坠剑湖的孤军与驰援 ::

> 事件概况
- 莫宁教授发出邀约的温馨时刻被警报撕碎，海维夏探测器失控，研究院防御阵列锁死。莫宁因权限受阻无法操作。
- 漂泊者意外发现监视器上有琳奈的身影，当机立断借用载具直冲坠剑湖。此时，琳奈正独自清理完被污染的隧群，被神秘“联络人”利用已死残星会成员编码威胁，并在此刻遭遇失控海维夏的袭击。
- 琳奈陷入苦战，漂泊者从天而降，挡在她身前。两人无需多言，默契配合，经过一番激战，强制停止了海维夏的运作。

> 区块目标
- 目标1：展现漂泊者的决断力与琳奈的单兵作战能力。
- 目标2：通过战斗确立两人的战场默契，铺垫信任基础。

> 执行引导
- 描写重点：
    视觉镜头：研究院内红光闪烁的紧张感，莫宁面对锁死屏幕的无力。漂泊者驾驶载具在复杂地形中极速穿梭的动态模糊。坠剑湖畔琳奈孤身对抗巨型机械的渺小与坚韧，以及漂泊者落地时激起的尘土。
    互动细节：漂泊者落地时琳奈眼中闪过的惊喜，以及战斗结束后两人对视时松了一口气的神情。
- 必然触发台词/梗：
  - 漂泊者：“莫宁教授，我需要借用研究院的载具。”
  - 联络人（通讯）：“来送一份符合这个身份的礼物吧……‘猜疑’和‘孤独’。”

---

:: 主线事件2：魏尔伦的审讯与维护 ::

> 事件概况
- 战斗结束，魏尔伦教授带领教师团赶到。魏尔伦无视满地狼藉，直接质问琳奈为何出现在此、为何监控无记录，并意有所指地提及她“新联邦”背景和假期不回家的疑点。
- 琳奈解释是作为向导发现异常，但因通讯阻断无法汇报。魏尔伦步步紧逼。
- 漂泊者站出来指出监控可能被篡改，坚定维护琳奈。琳奈虽然反击了魏尔伦的阴阳怪气，但最终仍被带走调查。

> 区块目标
- 目标1：引爆魏尔伦作为“对立面”的压迫感与可疑度。
- 目标2：通过“维护”行为加深漂泊者对琳奈的信任。

> 执行引导
- 描写重点：
    视觉镜头：魏尔伦推眼镜时镜片反光的阴冷。漂泊者挡在琳奈身前时，琳奈抬头看向他背影的特写。
    互动细节：琳奈被带走时回头看向漂泊者那个复杂的眼神（包含歉意、感激与求助）。
- 必然触发台词/梗：
  - 魏尔伦：“是不是因为，你根本‘不想’回去新联邦呢？”
  - 漂泊者：“如果监控本身出问题，那么琳奈同学的身影也有可能被去除。”

---

:: 主线事件3：天台上的假面告白 ::

> 事件概况
- 风波暂时平息，琳奈被限制行动但带漂泊者来到天台。面对漂泊者的直球提问，琳奈背对着他，最终坦白了真相。
- 真正的“琳奈”是残星会卧底且已死，她是顶替身份的新联邦佣兵。她讲述了过去只有“生存”没有“生活”的日子，以及对星炬学院这份平静的眷恋。
- 琳奈请求漂泊者帮忙查出残星会目的，漂泊者表示信任并同意合作。

> 区块目标
- 目标1：揭开身世谜底，完成情感宣泄。
- 目标2：确立“寻找真相”的同盟关系。

> 执行引导
- 描写重点：
    视觉镜头：天台上恒温人造微风吹动琳奈发丝，她眺望远方虚假天幕的背影。
    互动细节：琳奈自嘲的苦笑，谈到“起床刷牙”时的迷茫，以及漂泊者说出“合作吧”时她眼中重新亮起的光芒。
- 必然触发台词/梗：
  - 琳奈：“真正的琳奈已经死了。现在的我……只是为了摆脱那种只有‘生存’的日子。”
  - 琳奈：“在这里，大家不会纠结镜子里映出的是不是真实的自己……我很喜欢这里。”
  - 漂泊者：“我在魏尔伦面前为你辩护，代表着我选择相信你。我们合作吧。”

---

:: 主线事件4：潜入调查与定时炸弹 ::

> 事件概况
- 两人秘密潜入工学部，琳奈利用黑客技术覆写了被加密仪器的权限。
- 随后前往深空文献中心，通过AI IRIS检索，漂泊者在一篇关于“同步加速器”的论文中查出波形源头是虚诞虫。
- 琳奈推断出残星会利用仪器作为诱饵（定时炸弹）引诱虚诞虫袭击研究院。琳奈为了防止袭击，利用手中仪器制作了备用方案（引向防御设施）。
- 刚走出文献中心，琳奈因身份暴露（被匿名举报）被教师带走。

> 区块目标
- 目标1：展示琳奈的专业能力（黑客技术）。
- 目标2：揭露残星会阴谋逻辑，为高潮铺垫。

> 执行引导
- 描写重点：
    视觉镜头：琳奈手指在键盘飞速敲击的残影，IRIS全息数据流映照在两人脸上的光影。当“虚诞虫”三字出现时，两人瞬间对视的紧张感。
    互动细节：琳奈被带走前最后看了漂泊者一眼，眼神传递着“交给你了”的无声信息。
- 必然触发台词/梗：
  - 琳奈：“这东西是个定时炸弹……他们想把虚诞虫引到研究院来。”
  - 漂泊者：“数据匹配完成了……是虚诞虫。”

---

:: 主线事件5：退学时刻与逆行诱饵 ::

> 事件概况
- 漂泊者通过赫斯医生将情报传达给研究院。随后在宿舍找到正在收拾行李的琳奈，她已被勒令退学并遣返。
- 琳奈看着充满生活气息的房间流露不舍，但最终提着行李离开。
- 此时警报骤响，虚诞虫大举来袭，目标竟是学院（人口密集区）。漂泊者联系莫宁得知需手动校准阵列。
- 本以为琳奈已经逃跑，没想到她折返骑上改装摩托，手持覆写权限的仪器，逆行冲向虚诞虫，以自身为诱饵将其引离学院，驶向防御阵列方向。

> 区块目标
- 目标1：将“被驱逐者”与“守护者”身份重叠，升华角色。
- 目标2：开启高潮战役，展现琳奈的牺牲精神。

> 执行引导
- 描写重点：
    视觉镜头：宿舍内玩偶、贴纸的温馨与冰冷行李箱的对比。警报声刺破宁静的瞬间。琳奈跨上摩托决绝的背影，以及她在**温暖的地下城市街道**上逆着惊慌逃难人群狂飙的画面。
    互动细节：琳奈看着房间时眼底的留恋。
- 必然触发台词/梗：
  - 琳奈：“回到新联邦等着我的……大概就是牢狱之灾吧。”
  - 漂泊者：“莫宁教授，告诉我防御阵列在哪里！我去校准！”

---

:: 主线事件6：光之牢笼与真实的新生 ::

> 事件概况
- 漂泊者奔赴三座阵列进行手动校准。冰原之上，琳奈凭借高超车技在虚诞虫的攻击边缘极限闪避，将其一步步引入陷阱。
- 随着最后一次校准完成，三道光柱冲天而起交织成牢笼，死死束缚住虚诞虫。危机解除。
- 琳奈气喘吁吁停下摩托，被学生和记者团团围住称为英雄，漂泊者被叫做传说中的“黑色闪电”。教务处老师赶到，琳奈本以为要被遣返，却被告知学院感谢她的英勇，并邀请她用“真实身份”参加入学考试。
- 琳奈感动落泪，接受邀请。最终在花园与漂泊者约定，成为新生后一起去尝尝便利店料理。

> 区块目标
- 目标1：完成视觉冲击力的战斗收尾。
- 目标2：给出一个圆满、充满希望的结局，呼应“真实”的主题。

> 执行引导
- 描写重点：
    视觉镜头：光柱升起照亮地下穹顶的壮观，琳奈摘下头盔满脸汗水劫后余生的笑容。听到可以“重考”时，泪水夺眶而出的特写。花园中两人并肩而立的宁静。
    互动细节：琳奈听到“入学考试”时猛地抬头的动作，以及约定时的破涕为笑。
- 必然触发台词/梗：
  - 教师：“琳奈，如果你想成为本校学生……就需要用你的真实身份，重新走一遍入学流程。”
  - 琳奈：“……诶？”
  - 琳奈：“等我成为了‘新生’，记得带我去尝尝学校便利店的特色料理！”
