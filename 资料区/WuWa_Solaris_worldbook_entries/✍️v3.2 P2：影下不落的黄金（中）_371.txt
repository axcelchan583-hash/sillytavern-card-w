:: 主线事件1：黑猫折纸与天台的飞鸟 ::

> 事件概况
- 漂泊者在会场入口见到了努力调整好状态的西格莉卡，她递上了一只亲手制作的黑猫折纸作为道歉与纪念。
- 西格莉卡解释这只猫代表着虽然找不到家、却依然坚定张开翅膀飞向光点的勇气，而漂泊者给她同样的安心感。
- 漂泊者收下礼物后，被西格莉卡拉着前往教学区顶端的天台，寻找一种名为“嘤嘤鹦”的飞鸟。
- 两人并肩站在天台边缘，西格莉卡兴奋地讲解鸟类的习性，感叹它们为了自由穿越风暴的勇敢。
- 漂泊者举起相机，西格莉卡凑近屏幕指导对焦，两人在微风中拍下了一张完美的飞鸟照片，西格莉卡开心地邀请漂泊者加入“飞羽同好会”。

> 区块目标
- 目标1：通过赠送手工作品的行为，体现西格莉卡对漂泊者日益增长的依赖与好感
- 目标2：借“飞鸟学飞”的意象，暗示西格莉卡内心渴望摆脱束缚、自由飞翔的祈愿
- 目标3：描写两人在天台并肩拍照时的肢体靠近与氛围拉扯

> 执行引导
- 描写重点：
  视觉镜头：天台高处吹拂而过的微风，西格莉卡发丝飞扬的侧脸；镜头中嘤嘤鹦互相赠送鲜花嫩叶的特写。
  互动细节：西格莉卡双手递上黑猫折纸时指尖的微微触碰；拍照时两人肩膀几乎靠在一起的亲昵距离，以及西格莉卡看向照片时亮闪闪的眼眸。
- 必然触发台词/梗：
  - 西格莉卡：“它们不是生来会飞，所以都要经历第一次……从高处跃下，任由风吹过它们的羽毛。”
  - 西格莉卡：“那种时候，一定很害怕吧。但它们最终会在空中自由自在飞翔。”
- 伏笔/暗线：黑猫折纸虽然长着翅膀，但本质上是一只“猫”，暗示西格莉卡潜意识里觉得自己并不属于那些生来就会飞的天才（鸟类），她的努力是一场痛苦的自我改造。

:: 主线事件2：跳跳糖狂欢与黄金期许 ::

> 事件概况
- 第二天上午，西格莉卡兴奋地拿来换日庆典特供的“彩虹豆豆跳跳糖”邀请漂泊者品尝。
- 吃下糖果后，漂泊者说话的音色变成了类似鹦鹉的奇怪声音，而西格莉卡则陷入了疯狂打嗝的状态。
- 漂泊者用鹦鹉音呼唤阿布，把一颗糖塞在阿布嘴里，阿布也发出了日灵的“啾噜”叫声；众人顶着滑稽的状态去找达妮娅，达妮娅因为之前的惨痛经历（吃了之后眼睛嘴巴散发彩虹光芒）慵懒而严正地拒绝吃糖。
- 两人在会场找到琳奈并骗她吃下糖果，琳奈触发了“说反话”的搞笑效果，说出来的所有词汇都变成了反义词，三人一起在游戏机前玩闹了十局。
- 药效过去后，西格莉卡与漂泊者来到二楼阳台休息。听着楼下的喧闹，西格莉卡向漂泊者吐露心声，坦言外界那些鼓励的话语就像滚烫的黄金。
- 漂泊者倾听着她的迷茫，告诉她有时候不需要纠结答案，遵从内心的声音去做就好。

> 区块目标
- 目标1：利用神奇糖果的变声机制，制造全篇最欢乐的喜剧反差效果
- 目标2：展现星炬学院学生们在换日庆典中的青春活力与友情羁绊
- 目标3：引出“期待犹如黄金”的核心比喻，深刻剖析西格莉卡不敢让期待落空的沉重压力

> 执行引导
- 描写重点：
  视觉镜头：五颜六色的跳跳糖在掌心跳跃；二楼阳台俯瞰下去张灯结彩、人声鼎沸的庆典全景。
  互动细节：漂泊者用滑稽的鹦鹉音说话时西格莉卡一边打嗝一边捂嘴偷笑的模样；二楼交谈时，西格莉卡双手搭在栏杆上，神情由欢快逐渐转为怅然的微表情。
- 必然触发台词/梗：
  - 西格莉卡：“这颗味道好怪……嗝……哦不对、我不……嗝！对不起！嗝！”
  - 琳奈：“好‘难吃’！‘对’，我‘想’这样说……？我说的话都变成‘相同’……嗯？”
  - 琳奈：“我们一定要‘坏坏’玩一下！”
  - 西格莉卡：“这些所有的话，都像滚烫的黄金一样，每一句话都那样耀眼，那样……温暖。我不敢让它们落在地上。”
- 伏笔/暗线：陆医生后来会揭晓这些奇奇怪怪的跳跳糖其实是他做的。而“滚烫的黄金”这一意象，将在最后的暗面决战中成为西格莉卡破除心魔的关键武器。

:: 主线事件3：抽屉里的秘密与微缩奇遇 ::

> 事件概况
- 洛瑟菈发来紧急通讯，告知有学生（包括千咲和艾莉亚）被困暗面。西格莉卡坚持要一同前往救援。
- 两人跟随符文指引来到西格莉卡的宿舍。漂泊者在房间内看到了西格莉卡的私密日记和与朋友们的合影，日记中写满了她对繁重学业的焦虑。
- 暗面的入口竟然藏在书桌抽屉里，两人只得硬挤进抽屉，伴随一阵晕眩再次坠入暗面空间。
- 在暗面中，两人率先解救了正在冷静研究电子琴与烧烤结合的可能性的千咲，并联手击退了一波因负面情绪而狂暴的残象。
- 漂泊者与西格莉卡分头追击引路的兔子，漂泊者的身体突然急剧缩小，视线中的一切变成了庞然大物。
- 漂泊者在微缩状态下，巧妙借助场地上巨大麦克风的力量下砸出音浪开路，随后恢复原状与西格莉卡重新汇合。

> 区块目标
- 目标1：通过翻阅日记与合影，进一步补充西格莉卡在“完美优等生”面具下的真实痛苦
- 目标2：描写钻入抽屉进入暗面的奇幻感，增加场景的趣味性
- 目标3：执行“身体变小”与“麦克风砸地”的特殊机制交互，丰富暗面的战斗表现

> 执行引导
- 描写重点：
  视觉镜头：摊开在桌面上的日记本上微微颤抖的字迹；漂泊者身体缩小后，仰视巨大化课桌椅和兔子的奇妙视角。
  互动细节：漂泊者不小心瞥见照片时，西格莉卡慌乱遮掩的举动；两人一前一后艰难挤入狭窄抽屉时的肢体摩擦感；漂泊者挥舞与自身等大麦克风砸向地面的震撼冲击。
- 必然触发台词/梗：
  - 洛瑟菈（通讯）：“事发时，她们应该都在宿舍大厅。学生们当中有人激活了谜题，暗面把他们都吸了进去。”
  - 千咲：你们看，如果把它和这个电烤系统绑在一起，让它的每一个黑白键都变成旋钮开关的话……
  - 千咲：“我把它们都剪断了……那只是一些旧影，困不住我。”
  - 西格莉卡：“居然把入口藏在抽屉里，怎么看这也不是装情绪的地方。只能从抽屉里挤进去了！”
- 伏笔/暗线：日记本中写着“大家希望你成为怎样怎样好的人”，这正是后来阴暗面化身（涡心）用来攻击西格莉卡的原话，表明暗面怪物就是她内心的投射。

:: 主线事件4：断裂的枷锁与医务室的眼泪 ::

> 事件概况
- 两人找到了失踪的艾莉亚。艾莉亚不堪重负，言辞激烈地质问西格莉卡为什么要一直迎合大家的期待，并揭露她在暗面看到了西格莉卡想逃避的真心。
- 西格莉卡被当面拆穿，陷入极度的无措与恐慌。漂泊者见状，主动伸出手，西格莉卡犹豫片刻后紧紧回握住了漂泊者的手，两人十指相扣面对艾莉亚。
- 艾莉亚情绪失控，化身为被红布缠绕的残星会怪物。漂泊者将其击败，但西格莉卡却被拖入了深层的精神幻觉。
- 在幻觉中，老师和同学的窃窃私语化作利刃。代表阴暗面的“另一个西格莉卡”现身，残忍地揭露所有暗面异象都是因为她害怕失败而滋生的。
- 西格莉卡在绝望中失去意识，随后在现实的医务室病床上惊醒。漂泊者一直守在床边。
- 面对漂泊者，西格莉卡彻底卸下防备，泪珠大颗大颗砸在腿上，哭泣着承认是自己搞砸了一切。
- 漂泊者坐到床沿，伸手温柔地抚摸她的头顶，默默陪伴她平复情绪，随后用爱弥斯（那位明知命运仍走向牺牲的救世主）的故事鼓励她无需在乎失败，尽力无悔即可。最后漂泊者悄然离开，留她独自思考。

> 区块目标
- 目标1：借由艾莉亚的指责，彻底扯下西格莉卡的伪装，引出暗面怪物本体就是西格莉卡心魔的真相
- 目标2：描绘危机时刻漂泊者主动牵手的动作，给予角色强烈的安全感支撑
- 目标3：刻画全篇最高潮的情感破防戏码，用摸头、陪伴和讲故事完成深度的灵魂共鸣

> 执行引导
- 描写重点：
  视觉镜头：艾莉亚异化为红布怪物的惊悚瞬间；幻境中阴暗面西格莉卡那捧着脸、充满恶意的讥讽笑容；医务室苍白的光线下，少女低垂着头、泪水连成线的脆弱模样。
  互动细节：漂泊者伸出手时掌心的温度，西格莉卡颤抖着紧紧攥住的力度；在病床边，漂泊者的手掌覆上她发顶轻轻顺毛的怜惜动作，以及抽走纸巾为她擦拭泪水的克制。
- 必然触发台词/梗：
  - 艾莉亚：“你要如何一直去满足大家的期待呢，西格莉卡？”
  - 西格莉卡？（涡心）：“因为我就是你……而你，一定会回到这里。”
  - 西格莉卡：“我搞砸了……对不起，真的对不起，我真的很抱歉……”
  - 西格莉卡：“那些期待犹如黄金，我不敢让它们落在地上。”
- 伏笔/暗线：漂泊者讲述的那个“想成为救世主的孩子”，正是前文洛瑟菈提到过的爱弥斯。这个故事中的勇气与决心，将化为下一章节西格莉卡重新站起来、直面心魔的最强驱动力。
