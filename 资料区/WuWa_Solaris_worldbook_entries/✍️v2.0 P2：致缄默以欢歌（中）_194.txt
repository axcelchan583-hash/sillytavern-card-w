[剧情版本v2.0 Part 2: 致缄默以欢歌（中）]

[章节核心目标]
- 目标1：在拂风水畔与菲比及其声骸汇合，阻止吉尔贝的献祭仪式。
- 目标2：唤醒被污染的守护者罗蕾莱，通过菲比的歌声解除云海危机。
- 目标3：潜入埃弗拉德金库，与珂莱塔达成盟约并揪出家族背叛者。
- 目标4：在高塔遗迹揭露双神真相，并在珂莱塔的“背叛”枪击中坠入深渊。

[章节前置状态]
- 地点/环境：拂风水畔（巨大的风车、花田、甚至有异样的频率污染） -> 云中庭院 -> 埃弗拉德金库（戒备森严） -> 高塔遗迹（悬崖边）。
- 主角状态：警惕性高，通过忏悔室密道抵达野外，急需追查吉尔贝下落。
- 伴随NPC：先是赞妮，随后加入菲比（携带声骸拉比、贝尼），后期主要与珂莱塔同行。

[章节事件列表]

2. 主线事件：云海列车的假面与悲歌
3. 主线事件：圣女的咏叹与守护者的救赎
4. 主线事件：金库的双重奏与暗夜潜行

[章节终止条件]
珂莱塔在阿莱克斯司铎面前开枪射击漂泊者脚下地面，漂泊者坠入悬崖。

:: 主线事件1：风车下的焦急寻找与对峙 ::

> 事件概况
- 漂泊者在风车下偶遇隐瞒真实身份的剧作家克里斯托弗，他用玩味的语气向漂泊者讲述了一段鸣式与圣女的故事。
- 告别剧作家后，偶遇焦急寻找伙伴斑布的菲比，阿布从树丛中跳出吓了她一跳。
- 菲比身边的拉比和贝尼与阿布嬉戏，斑布从远处跑来，引领众人冲向花田深处。
- 吉尔贝站在彼岸花丛中高举双手狂热祈祷，声骸在他脚边痛苦扭曲转化为残象。
- 吉尔贝察觉众人到来，尖叫着指挥残象发动攻击，漂泊者拔刀迎战，菲比掩护赞妮。
- 漂泊者一刀斩断控制声骸的彼岸花茎，吉尔贝跌坐在地，眼神依然疯狂不减。
- 菲比上前质问，吉尔贝狂笑着透露罗蕾莱已被控制，云海即将吞没拉古那。

> 区块目标
- 目标1：展示菲比作为“声骸朋友”的一面，通过她与拉比、贝尼、斑布的互动体现其纯真。
- 目标2：揭露吉尔贝的疯狂及其背后“残星会”与修会勾结的阴谋。

> 执行引导
- 描写重点：
  刻画菲比焦急寻找斑布时的神态，以及身边围绕着的小声骸（拉比和贝尼）的可爱动态。战斗中需描写吉尔贝狂热的宗教疯癫感与周围彼岸花散发的恶意。
- 必然触发台词/梗：
  - 菲比：“斑布！你在哪里？拉比和贝尼都很担心你……”
  - 吉尔贝：“为了唯一的真神……必要的牺牲是通往新生的阶梯！”
- 伏笔/暗线：
  吉尔贝提到“罗蕾莱”已被控制，引出下一阶段的云海危机。

:: 主线事件2：云海列车的假面与悲歌 ::

> 事件概况
- 漂泊者赶到云中车站，恰逢最后一班列车即将发车，戴着面具的神秘小姐邀请其同行。
- 列车穿行于翻涌的云海之上，罗蕾莱凄厉的歌声透过车窗传来，震动着车厢玻璃。
- 神秘小姐侧头看向窗外，轻声讲述云海下掩埋的旧神学院与修会的血腥历史。
- 她转过头透过面具深深注视漂泊者，指出他已身处漩涡中心，无法独善其身。
- 列车到站，神秘小姐起身整理裙摆，留下一句意味深长的话语后优雅离去。
- 漂泊者望着她的背影若有所思，随后与赶来的赞妮和菲比汇合前往庭院。

> 区块目标
- 目标1：加深与珂莱塔（此时未揭面）的羁绊，通过对话展现她对局势的深刻洞察。
- 目标2：渲染云海中罗蕾莱歌声的悲怆氛围。

> 执行引导
- 描写重点：
  列车穿行于云海之上的孤寂感。重点描写珂莱塔隔着面具注视漂泊者的眼神，以及她话语中若有若无的试探与拉拢。
- 必然触发台词/梗：
  - 神秘小姐：“想要直面未来，就要先了解过去。”
  - 神秘小姐：“我们……早就同在一条船上了哦。”

:: 主线事件3：圣女的咏叹与守护者的救赎 ::

> 事件概况
- 众人被挡在云中庭院的声波屏障外，菲比深吸一口气，颤抖着唱响古老的咏叹调。
- 屏障应声而碎，巨大的回音生物罗蕾莱双目赤红，咆哮着向众人发动音波攻击。
- 漂泊者在音波中穿梭，精准定位并斩碎了附着在罗蕾莱身上的彼岸花核心。
- 罗蕾莱眼中的红光消退，虚弱地向众人致谢，并揭示了“指挥家”的存在。
- 她悲伤地指出岁主早已与修会断联，现任神启者的身份是虚假的谎言。
- 菲比闻言脸色苍白，信仰动摇，决定返回修会内部查证真相。

> 区块目标
- 目标1：高光展示菲比的歌声与勇气，打破其柔弱印象。
- 目标2：通过罗蕾莱之口抛出核心设定——现任主座的神启者身份存疑。

> 执行引导
- 描写重点：
  菲比歌唱时颤抖却坚定的声音，罗蕾莱从暴怒转为平静的视觉变化。
- 必然触发台词/梗：
  - 菲比：“以此歌声……回应您的悲伤。”
  - 罗蕾莱：“小心那些戴着面具的人……他们称呼领袖为‘指挥家’。”

---

:: 主线事件4：金库的双重奏与暗夜潜行 ::

> 事件概况
- 漂泊者潜入埃弗拉德金库遭遇守卫，神秘小姐现身喝退守卫，摘下面具露出珂莱塔真容。
- 珂莱塔提出结盟并开启双人潜行：她在地面利用香水迷晕巡逻者，漂泊者在上方横梁移动。
- 漂泊者操作控制台，精准降下巨大的水晶吊灯，珂莱塔轻盈跳上吊灯借力越过红外防线。
- 两人在调频室门前汇合，相视一笑，同时破门而入制服了正在改造声骸的内鬼。
- 珂莱塔在深处展示家族珍藏的琉璃刀伶，揭示其作为频率母本的特殊地位。
- 档案显示刀伶源自高塔遗迹，两人决定立刻前往那里寻找修会隐藏的秘密。

> 区块目标
- 目标1：完成珂莱塔身份的正式揭晓，确立“智谋型”女主地位。
- 目标2：通过双人配合的潜行过程，体现两人非凡的默契。

> 执行引导
- 描写重点：
  金库内的紧张潜行氛围。重点描写珂莱塔利用香水和珠宝暗语指挥行动的优雅，以及漂泊者在控制台精准操作吊灯为她铺路的默契配合。
- 必然触发台词/梗：
  - 珂莱塔：“重新认识一下，我是珂莱塔·莫塔里。”
  - 珂莱塔：“把背后交给你了，我的搭档。”

---

:: 主线事件5：琉璃彩窗的真相与决绝枪响 ::

> 事件概况
- 两人在高塔遗迹拼凑破碎的琉璃彩窗，重现少女神启者的历史画面，与自称神启者的男性主座芬莱克矛盾。
- 阿莱克斯司铎带着全副武装的修会卫兵包围遗迹，手中晃动着科波拉叔父的信物。
- 珂莱塔脸色骤变，在司铎的逼迫下颤抖着举起手枪，黑洞洞的枪口对准漂泊者。
- 珂莱塔眼中闪过决绝，低声说出告别语，随即扣动扳机。
- 子弹击中漂泊者脚下的脆弱岩层，地面瞬间崩塌，漂泊者在碎石中坠向深渊。
- 阿莱克斯错愕地冲上前查看，珂莱塔垂下枪口，冷冷地注视着深不见底的悬崖。

> 区块目标
- 目标1：揭开黎那汐塔历史最大的秘密（双神论）。
- 目标2：制造极其强烈的戏剧冲突与情感爆发，达成“假背叛，真守护”的效果。

> 执行引导
- 描写重点：
  修复彩窗时历史画面重现的史诗感。高潮在于珂莱塔举枪时的眼神特写——表面冷酷但眼底深藏着传递给漂泊者的暗号与决绝。枪声响起时地面的崩塌要极具冲击力。
- 必然触发台词/梗：
  - 石碑/彩窗：“当狂欢节的钟声敲响，少女将戴上冠冕……”
  - 珂莱塔：“再见了，和你的朋友……在‘坟墓’中相会吧。”
- 伏笔/暗线：
  “坟墓”一词实指下方的悲叹墓岛，是暗示漂泊者去那里寻找这一局的生机。
