:: 主线事件1：伊什麦尔的推销与空气垂钓者 ::

> 事件概况
狂欢节后的拉古那，热情的伊什麦尔叫住漂泊者，极力推销海钓的魅力，并指引其前往黎乔利群岛寻找亚哈。抵达群岛后，漂泊者目睹亚哈船长正与空气进行激烈的无实物表演。在确认漂泊者身份后，亚哈豪爽地将视为家人的贡多拉拉结号交付给他。

> 区块目标
- 目标1：通过伊什麦尔的连珠炮式对话，确立轻松、充满烟火气的开场基调。
- 目标2：刻画亚哈疯癫但专业的形象，并让玩家获得专属载具拉结号。

> 执行引导
- 描写重点：
  伊什麦尔推销时眼中闪烁的期待光芒，以及亚哈对着虚空拉扯时紧绷的肌肉线条和滑稽的专注感。
- 必然触发台词/梗：
  - 伊什麦尔：“钓鱼嘛，钩要弯，鱼饵不能太咸……去黎乔利群岛找亚哈船长吧！”
  - 亚哈：“小伙子！别愣着呀，来搭把手！……肯定是条大鱼……”

:: 主线事件2：集市解围与“帽子”的誓言 ::

> 事件概况
漂泊者在集市发现菲比被排外的村民围攻。漂泊者出面解围，亚哈随后赶到，用大海包容万物的理论喝退众人，并鼓励菲比戴着自己的帽子去寻找溯海之鲸。菲比深受触动，正式加入队伍。

> 区块目标
- 目标1：通过冲突展现菲比柔弱的一面，激发漂泊者的保护欲。
- 目标2：通过帽子的隐喻（是修会的教士，还是她自己），确立菲比的人物成长主线。

> 执行引导
- 描写重点：
  菲比在人群中紧握手杖的无助细节，以及在得到漂泊者和亚哈肯定后，眼神中重新燃起的希望之火。
- 必然触发台词/梗：
  - 亚哈：“不拥抱大海，又怎能期望大海能回应你呢？只要你戴着‘自己’的帽子就好啦。”
  - 漂泊者：“做你自己最重要。”

:: 支线事件3：海上的黄金假期（捕鱼、升级与约会） ::

> 事件概况
为了筹备出海寻找斯塔巴克斯的资金与装备，漂泊者带着新手菲比开始了漫长的捕鱼修行。
阶段一（教学）：手把手教菲比甩杆，她因笨拙而脸红，第一次钓上鱼时的惊喜。
阶段二（经营）：将渔获卖给章鱼商人普朗克，赚钱升级拉结号的引擎、购买更高级的鱼饵和图鉴。
阶段三（约会）：捕捉到稀有的大型观赏鱼卖出高价后，两人用这笔钱在夕阳下的群岛享用了一顿丰盛的海鲜大餐，享受片刻的二人世界。

> 区块目标
- 目标1：核心互动区。大幅拉长篇幅，描写两人在狭窄船只上的肢体接触（调整姿势、庆祝击掌）。
- 目标2：建立努力工作 -> 升级变强 -> 享受生活的愉快循环，营造像度假一样的轻松氛围，作为后续沉重剧情的缓冲。

> 执行引导
- 描写重点：
  海面波光粼粼的视觉美感；菲比钓到鱼时兴奋地抱住漂泊者的下意识动作；升级船只后的成就感；约会时暧昧流动的空气与菲比在夕阳下的侧脸。
- 必然触发台词/梗：
  - 菲比：“哇！钓上来了！漂泊者你看，是金色的！”
  - 菲比：“用我们赚的钱买的晚餐……好像比修会的饭菜好吃一百倍呢。”

:: 主线事件4：深海的荒诞婚礼与物理驱魔 ::

> 事件概况
线索指向斯塔巴克斯，众人在舵盘村找到他。他正因海妖魅惑，抱着一条滑腻的咸鱼举行婚礼。菲比误判形式，试图用金属手杖对斯塔巴克斯进行物理净化（敲晕），被漂泊者紧急拦下。随后两人配合，通过狸猫换太子战术，强行换走了那条鱼，解除了魅惑，并从鱼腹中找到了通往遗迹的钥匙。

> 区块目标
- 目标1：利用巨大的画风突变（从浪漫约会到恶心婚礼）制造喜剧效果。
- 目标2：展现菲比天然黑/暴力牧师的反差萌属性，增加肢体互动。

> 执行引导
- 描写重点：
  斯塔巴克斯对咸鱼深情款款的恶心神态；菲比一脸圣洁地举起凶器（手杖）时的反差；漂泊者从背后抱住或抓住菲比手腕阻止她的紧急动作。
- 必然触发台词/梗：
  - 斯塔巴克斯：“嘿嘿……宝贝儿……你是证婚人？来得正好！”
  - 菲比：“英白拉多的赐福（物理），注视于此——哎？漂泊者你为什么拦着我？”

---

:: 主线事件5：肉麻的情诗与大海的三重馈赠 ::

> 事件概况
众人利用钥匙来到莱耶斯遗迹，却被风暴阻挡。石碑要求献上爱意，亚哈粗鲁的尝试失败，刚清醒的斯塔巴克斯用肉麻的情诗成功驱散了风暴。进入遗迹深处后，菲比通过解读古老石碑，发现了大海的三重馈赠谜题。漂泊者驾驶小船在周边海域精准捕获了三种特定的鱼类，分别满足了三个执念幽灵的愿望，从而点亮了最后三座灯塔，彻底解开了封印。

> 区块目标
- 目标1：通过斯塔巴克斯与亚哈的对比制造笑料，同时推进剧情。
- 目标2：详细描写菲比解读古文、分析线索的高光时刻，展现她作为教士的专业能力。
- 目标3：完成捕鱼解密环节，通过满足幽灵的执念，引出亡者残响的概念铺垫。

> 执行引导
- 描写重点：
  斯塔巴克斯朗诵情诗时陶醉、夸张的肢体语言；菲比抚摸石碑文字时专注认真的神情；幽灵在吃到鱼后满足消散的温情光影。
- 必然触发台词/梗：
  - 亚哈：“意思是说，我们还得爱点儿什么？哼。”
  - 斯塔巴克斯：“哦，我的爱人，我的生命之光。风暴不能将我们阻隔……”
  - 菲比：“展示大海的三重馈赠，灯火将永燃不熄……漂泊者，我们需要找到这三种鱼。”

---

:: 主线事件6：裴魁德号的冲锋与最后的港口 ::

> 事件概况
亚哈留书决死，斯塔巴克斯驾驶小船冲入漩涡殉情。关键时刻，布兰特船长驾驶修好的裴魁德号赶来支援。漂泊者登上大船，指挥布兰特冲入风暴中心。溯海之鲸平息了海妖的暴走并救起众人。菲比完成与鲸的和解。回到港口后，一行人在哪也找不到亚哈等人，伊什麦尔揭示真相：亚哈等人早已是亡魂（残响）。如今心愿已了，他们微笑着消散。

> 区块目标
- 目标1：布兰特登场，展现裴魁德号的压迫感与决战气势。
- 目标2：利用叙述性诡计的揭露，升华主题，完成最后的告别。

> 执行引导
- 描写重点：
  布兰特稳健掌舵的身影与风暴的对比；巨鲸破水而出的神性瞬间；最后空荡荡的码头，伊什麦尔告别时身体逐渐透明或化为泡沫的唯美画面。
- 必然触发台词/梗：
  - 布兰特（通讯）：“这艘船还真是不错，就是有点旧了。怎么样，准备好出发了吗？”
  - 漂泊者：“准备好了！布兰特船长！”
  - 伊什麦尔：“其实，亚哈船长他们……几十年前就已经葬身海底了。这里，就是他们最后的港口。”
