- 达妮娅惊醒后，医疗AI N.A.N.A.检测到她的生命体征恢复但频率未谐振，准备启动第二方案。
- 同时，漂泊者先后收到两条飞讯：西格莉卡请求关照住院的达妮娅，说N.A.N.A.回复千篇一律令她担心；研究员阿里曼正式告知达妮娅已苏醒，希望就收容事宜进行合作。漂泊者应下双方请求，前往医务室。

> 区块目标
- 目标1：通过梦境闪回建立达妮娅的创伤源头——她作为"第一个接受阿列夫一馈赠的个体"的身份，以及娜波摩冷酷的"慈悲"哲学。
- 目标2：通过双线飞讯呈现不同立场——西格莉卡作为朋友的纯粹关心，深空联合作为机构的理性审慎。

> 执行引导
- 描写重点：
  视觉镜头：梦境中娜波摩扭断猫头鹰脖子时手指的精确与漠然；达妮娅在病床上猛地睁眼时瞳孔的收缩；终端屏幕上两条飞讯的并列弹出。
  互动细节：漂泊者阅读西格莉卡消息时看到她反复追问N.A.N.A.的记录，感受她的担忧。
- 必然触发台词/梗：

  - 娜波摩："你必须学会接受它，融入祂。这是世界赠与你的最初和最后的礼物。"

- 伏笔/暗线：

- 问题的具体原因不明——无法判断是生理性衰退还是鸣式所致。雪上加霜的是，新联邦正以隧门事件和黯原残留频率为由，要求立即引渡达妮娅。深空联合拒绝了这个要求，不愿将她作为"样本"交出。
- 阿里曼提出需要漂泊者的协助：达妮娅抵触与深空联合沟通，但因漂泊者与她有过接触且具备鸣式对策经验，希望他作为中间人打开突破口。漂泊者答应尝试，但要求此次会面内容不留档案——包括IRIS的数据库。阿里曼同意。

> 区块目标
- 目标1：建立达妮娅性命垂危的紧迫感，同时铺垫深空联合与新联邦之间的政治角力。
- 目标2：确立漂泊者作为唯一"突破口"的独特地位，以及他对信息管控的谨慎。

> 执行引导
- 描写重点：
  视觉镜头：阿里曼揉着眉心时的疲惫，他说话时不停看向监控屏幕的戒备。漂泊者提出"不留档"时两人短暂的眼神对峙。
  互动细节：阿里曼提到洛瑟菈校长时下意识压低声音；漂泊者接过阿里曼递来的安定类药物时手指的微顿——暗示他对药物作用的疑虑。
- 必然触发台词/梗：

  - 阿里曼："新联邦要求我们立刻将达妮娅送过去——当然，我们没有同意。"

:: 主线事件3：病房中的交易——生日与情报 ::

> 事件概况
- 漂泊者推开病房门，达妮娅早已察觉他的到来。她以轻松到近乎轻浮的口吻调侃漂泊者空手而来，并坦然承认自己"快死了"——残星会会长在她体内留下了类似"项圈"的保险措施，时候一到她便可以去"睡个好觉"。
- 她直接戳破漂泊者的来意：他想从她这里获取情报。漂泊者没有否认，但坦言是"想试着相信她"。达妮娅却回以"我无法相信你"，随即话锋一转，提议做一个交易——她愿意告知会长相关的情报并做个了断，条件是漂泊者陪她过本周末的"生日"。
- 漂泊者警觉她口中的"了断"，达妮娅直言如果有他在仍不放心，可以给她装限制器或炸弹。漂泊者沉默后答应了协商。他补充说深空联合没有公开她的身份，"至少那天，你仍然可以是名为达妮娅的学生"。达妮娅怔住，低声嘟囔"净做些多余的事"，随即展露笑颜应下约定。

> 区块目标
- 目标1：展现达妮娅独特的性格——表面轻浮潇洒、内心已接受死亡，却仍保留着对"人的身份"的隐秘渴望。
- 目标2：建立两人之间基于"坦诚的算计"的微妙信任——彼此不否认各有所图，但也不掩饰真实想法。

> 执行引导
- 描写重点：
  视觉镜头：达妮娅坐在病床上望向窗外学生公寓的侧影；她笑说自己像"小白鼠"时手指轻敲床沿的小动作。漂泊者说出"你可以是名为达妮娅的学生"时她骤然收起笑容然后迅速恢复的微表情。
  互动细节：达妮娅提出"装限制器或炸弹"时语气像在讨论天气，漂泊者沉默的那几秒比任何回应都更有分量。
- 必然触发台词/梗：
  - 达妮娅："清楚呀，我快死了嘛。毕竟那个人很久以前就送过我一份礼物。"
  - 达妮娅："见识过我那副样子后还能说出这样的话……是因为你也有想从我这知道的事吧？"
  - 漂泊者："至少那天，你仍然可以是名为达妮娅的学生。"
  - 达妮娅："净做些多余的事。"
- 伏笔/暗线：
  达妮娅所说的"了断"——她计划在生日当天具体做什么？漂泊者将此疑虑传达给了阿里曼。

:: 主线事件4：车站交接与流程文件 ::

- 达妮娅将佩戴共鸣抑制设备，启动权限交由漂泊者——这是她本人主动要求的"合作保障"。阿里曼坦言以她的身体状态本不需要佩戴。此外，他递给漂泊者一盒安定类药物，承认深空联合尚未找到缓解频率衰竭的方法，目前只是临时方案。
- 达妮娅以焕然一新的姿态出现在站台，称自己今天是以"学生达妮娅"的身份出行。她注意到漂泊者面色严肃，伸手拉着自己的嘴角，演示了一个"微笑"的动作，对着他笑了笑说"笑起来才好看"。

> 区块目标
- 目标1：通过流程文件的细节展示深空联合的专业准备与对风险的全面评估。
- 目标2：展现达妮娅与前半日在病房中不同的面貌——从"认命的小白鼠"切换为享受当下的"学院学生"。

> 执行引导
- 描写重点：
  视觉镜头：终端上流程文件密级标注"内部受限"和详尽的六阶段流程图。达妮娅双手背在身后踱步走来的轻松姿态。她演示"表情管理"时指尖轻拉嘴角的动作。
  互动细节：漂泊者接过药物时内心确认了之前的猜测——达妮娅早就知道深空联合没有有效治疗方案。达妮娅说"笑起来才好看"时眼神有一瞬间的黯淡。
- 必然触发台词/梗：
  - 达妮娅："我今天可是学生达妮娅呀。"
  - 达妮娅："嘴角放轻松，眉眼自然点……对嘛，笑起来才好看。"
- 伏笔/暗线：
  流程文件中提及的"申报接触人员名单"——西格莉卡、排雪、娜斯塔霞等——暗示达妮娅今天要见的不仅是西格莉卡一人。

:: 主线事件5：花园中的逢场作戏 ::

> 事件概况
- 两人乘电梯下行前往花园区。达妮娅在电梯中问"坏了会被困住吗"，漂泊者答"可以把它切开"，被她笑骂"破坏学院公物"。下电梯时她忽然说有种"参加考试的感觉"，请求漂泊者"帮忙打掩护"。
- 在花园即将与西格莉卡见面时，达妮娅忽然拽着漂泊者想去游戏厅避开——她被西格莉卡远远喊住后瞬间切换成笑脸，亲切地打招呼。漂泊者以"受医生委托陪同病患"为由解释同行，西格莉卡不疑有他，只是担心达妮娅是否有后遗症。
- 三人散步时聊起达妮娅曾帮摄影部做模特拿了奖金。谈到那次她故意饿了一顿去吃"豪华自助"结果发现是食堂时，西格莉卡笑说她拍的宣传图上了《海豹也能懂的入学手册》。话题转向娜波摩，达妮娅沉默了一瞬，随即主动提出拍摄三人合影——她站在中间挽着西格莉卡的手臂，漂泊者在左后方。画面角落出现了一只白鸟纸役。

> 区块目标
- 目标1：通过三人自然的对话和互动呈现达妮娅与西格莉卡友情的真实质感——随意的吐槽、共同回忆、以及达妮娅刻意维持的表象。
- 目标2：在轻松氛围中埋入紧张——达妮娅试图逃避见面、提到娜波摩时的沉默、白鸟纸役（绯雪监视）的出现。

> 执行引导
- 描写重点：
  视觉镜头：三人合影时各异的姿态——西格莉卡闭眼微笑比"V"，达妮娅右臂向外伸展、睁眼微笑，漂泊者表情平静、右手食指伸出、左手比"V"。画面角落白鸟纸役一闪而过。
  互动细节：达妮娅看到西格莉卡时肩头一紧又迅速放松。提到娜波摩时达妮娅笑容出现极短暂的僵滞。拍摄合影前达妮娅主动挽住西格莉卡手臂的用力——像抓住某种即将滑落的东西。
- 必然触发台词/梗：
  - 达妮娅："我是模特啦。有人请我吃豪华自助午餐，条件是配合拍两张照……结果所谓的豪华自助居然指的是食堂。"
  - 西格莉卡："唯一一张合照还是娜波摩拉着你拍的。"
  - 达妮娅："……我们再来拍一次合影吧。"
- 伏笔/暗线：
  合影照片存入漂泊者终端，达妮娅说"有机会把它洗出来看看吧"——这是她留给漂泊者和西格莉卡的最后一张照片。

:: 主线事件6：通讯亭的谎言与真心 ::

> 事件概况
- 西格莉卡追逐白鸟远去后，达妮娅拉漂泊者到一旁。她问起那只白鸟是否是绯雪的能力，漂泊者承认，并说绯雪是个温柔的人。达妮娅说她知道——"她有时候和你挺像的"，随后低声补了一句"可惜我最讨厌这样的人"。漂泊者追问原因却被她轻巧地岔开——"这样打直球行不通的哦"。
- 与西格莉卡会合后，达妮娅提醒她去和家里通话。通讯亭前，达妮娅坦言自己并没有真要打电话——她只是随便点开一个通讯界面，"圆一个随口说出的谎"。漂泊者指出这是自欺欺人，达妮娅收起嬉笑：她做这一切都是为了西格莉卡能安心——"她那么小的个头，笑嘻嘻的，却背地里烦恼那么多事……我这个做朋友的还是让她安心点吧"。随后她又迅速切回轻浮语调："有这样一个天真可爱的朋友在，身份才不容易暴露，对吧？"漂泊者没有被她带偏，只说了四个字："自欺欺人。"

> 区块目标
- 目标1：通过谎言话题，揭示达妮娅对西格莉卡的保护方式——用一层又一层谎言建立安全的距离。
- 目标2：展现漂泊者不买账的态度——他不纵容达妮娅的"说谎舒适区"，用四字判断戳破她的防御。

> 执行引导
- 描写重点：
  视觉镜头：达妮娅随手点开通讯界面时空洞的屏幕反光。漂泊者说"自欺欺人"时她的笑容从嘴角褪去的慢镜头。
  互动细节：达妮娅说"让她安心点吧"时仍在微笑，但手指紧紧握着玩偶的边缘。漂泊者回答时没有看她——他看的是通讯亭内那个空无一人的界面。
- 必然触发台词/梗：
  - 漂泊者："你这样做只会让事情更麻烦。"
  - 达妮娅："圆一个随口说出的谎居然需要做这么多。"
  - 漂泊者："自欺欺人。"
  - 达妮娅："我还以为你会说得委婉点。"
- 伏笔/暗线：
  达妮娅"最讨厌温柔的人"——她在后续的长椅坦白中终于给出解释。

:: 主线事件7：橘子的约定与明年今日 ::

> 事件概况
- 一行人前往餐能中心购买蛋糕。达妮娅看着新出的甜品名字念得津津有味——"寒地星苔团""蕨团蒲藻饼""松针岩绒卷""鱼罐头松糕"——西格莉卡直呼恐怖。漂泊者考虑到流程中禁止食品摄入的规定，提议向一个"专业的甜品研究组织"（扎希拉）订蛋糕，并询问达妮娅想要什么口味。
- 达妮娅先开玩笑说彩虹豆豆跳跳糖、喀拉喀拉全都要加，被西格莉卡纠正后，她说出了真正的选择——订橘子的，"因为我答应过西格莉卡一起过生日，她喜欢橘子"。但西格莉卡即将协助莫宁去炉芯项目，无法在晚上参加。她愧疚地想请假，达妮娅伸手捏了捏她的脸颊说"不可以逃避"，随后又摸了摸她的头。
- 西格莉卡忽然停下脚步——她觉得今天的达妮娅不一样：主动约她、拉着拍照、坦白生日、鼓励她不要逃避……"这些全都不像是平时的你"。她直视达妮娅，问出了那个关键问题："明年今天，达妮娅还是会过生日的，对吧？"达妮娅攥紧了玩偶，答了一个"会"字。西格莉卡冲上前握住她的手，约定明年一起亲手做蛋糕。转身离开时，她偷偷抹着眼泪。达妮娅冲她的背影喊："加油啊，西格莉卡。"西格莉卡转过头笑着喊出"生日快乐"后跑远。
- 达妮娅长长地呼出一口气说"好累"。漂泊者说"西格莉卡真的很在乎你"，她抬起手弹了一下漂泊者的鼻子，说"这种时候又突然变得这么温柔，好狡猾"。然后她又恢复了嬉笑模样——"休息时间结束了，陪我玩会儿游戏吧"。
